Update seller-fulfilled return center settings

Last updated on Nov 6, 2024

Reading time: 2 min

* * *

Overview

You’re required to provide at least one return center for your account. Before adding or updating your return center settings, review [_Returns policy_](https://marketplacelearn.walmart.com/guides/Order%20management/Returns%20&%20refunds/returns-policy) to ensure your preferred settings comply with Walmart standards. Update your return center settings using the steps below.

## Add a new return center

### Step 1 – Get started

Navigate to the [**Returns**](https://seller.walmart.com/settings/shipping-profile/returns) page in your **Shipping Profile**.

Notes:

The return settings you specify will apply to all items in your catalog by default. If you have items that should not be included due to approved return exemptions (e.g., hazmat or freight shipments), you must [create item-level return rules](https://marketplacelearn.walmart.com/guides/Order%20management/Returns%20&%20refunds/Create-item-level-return-rules).

### Step 2 – Add a new return center

On the [**Returns**](https://seller.walmart.com/settings/shipping-profile/returns) page, navigate to **Return Centers (Required)** and select _Add Return Center_. Enter the required details and select _Add Return Center_.

![](https://azure-na-images.contentstack.com/v3/assets/blta7903c6b840b702d/blt2d021a62afcd79a8/66c4c8f4a6e1c3a460f54595/11193_GIF.gif?branch=us_mplearn)

Notes:

Walmart reserves the right to audit the return center addresses provided by Marketplace sellers for data quality and to standardize return addresses submitted by sellers to reflect the appropriate carrier formats. Failure to provide a valid U.S. return address could lead to suspension or termination. Walmart reserves the right to charge a Marketplace seller a failed delivery fee for items that are rerouted to Walmart’s Return Center as a result of a failed delivery attempt, due to an incomplete or invalid address provided by the seller in Seller Center.

## Frequently asked questions

* * *

How will the return center for an item be chosen?

Return centers are selected based on customer location. You may specify where items are returned by updating [item-level return rules](https://marketplacelearn.walmart.com/guides/Order%20management/Returns%20&%20refunds/Create-item-level-return-rules).

* * *

Can I edit an existing return center?

Yes. On the [**Returns**](https://seller.walmart.com/settings/shipping-profile/returns) page, navigate to the **Return Centers (Required)** section and find the return center you want. Select the pencil icon in the _Actions_ column to edit the information.

* * *

Do I need to have a return address in the United States?

Yes. Your return center address must be located in the U.S. and can’t be located in Hawaii, Alaska or U.S. territories including American Samoa, Guam, the Northern Mariana Islands, Puerto Rico and the U.S. Virgin Islands.

* * *

Can my PO Box be used as a return address?

No.
