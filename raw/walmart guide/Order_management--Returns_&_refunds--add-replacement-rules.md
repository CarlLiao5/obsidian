Add Replacement rules

Overview

There are some situations when a Walmart customer would rather receive a replacement item instead of a refund, such as when they receive an item that’s broken or the wrong color. In this guide, you’ll learn how to set up Replacement rules in Seller Center.

## Add a Replacement rule

### Step 1 — Get started

Navigate to your **Settings** inSeller Centerand select [_Returns_](https://seller.walmart.com/settings/shipping-profile/returns) under the **Shipping profile** section.Then, locate **Replacements by department**.

### Step 2 — Add a rule

Select _Add Replacement Rule_ and enter the **Department** you want the rule to be applied to **.** Under **Replacements enabled**, indicate _Yes_ or _No_, then select _Add Replacement rule._ The rule will be added to the **Replacements by Department** section on the Returns page in your Settings.

![](https://azure-na-images.contentstack.com/v3/assets/blta7903c6b840b702d/blt938b17f93c895a56/685afb8062bf7957636c2266/2506_Add-replacement-rule_screenshot.png?branch=us_mplearn)

Notes:

If you have [Keep it, Partial Keep it and](https://marketplacelearn.walmart.com/guides/Order%20management/Returns%20&%20refunds/Add-a-Keep-It-Rule) Replacement rules enabled for the same department across all three, Walmart will automatically suggest the most suitable option to customers.

## Frequently asked questions

* * *

Does the Replacement rule apply to in-store returns?

No. Replacements aren’t available for in-store returns.

* * *

Can I edit an existing Replacement rule?

Yes, to edit an existing Replacement rule, navigate to the Returns page in your Account settings, under Shipping Profile. Scroll to Replacements by Department, select the pencil under the **Actions** column for that **department**. Once you’ve made updates, select _Save._

* * *

What happens if I don’t add a replacement rule for a department?

The replacement rule will remain disabled by default. You must add a rule to offer the option of a replacement to the customer.
