Cancel fraudulent orders

Last updated on Dec 8, 2024

Reading time: 2 min

* * *

In this guide:

Overview

Walmart regularly screens Marketplace orders for fraud. If a fraudulent order is identified, Walmart will attempt to cancel the order automatically before you receive it. If the order has already been sent to your account, you’ll receive a "Fraud – Stop Shipment" email notification.

This guide shows you how to cancel a fraudulent order and how to request a review for an order you believe may be fraudulent.

## How to cancel fraudulent orders

#### Option 1 — Notification received from Walmart

If the fraudulent order hasn’t shipped yet and you’ve received an email notification from Walmart to cancel the order, you can update the order status in Seller Center. Select [_Orders_](https://seller.walmart.com/orders/manage-orders) . Orders identified as fraudulent will appear with an error icon. Next, select the purchase order number to find the order details.

Select _Cancel order_ in the upper right corner of the screen, then select the code _Cancel – Fraud, stop shipment_  from the dropdown menu, followed by selecting the _Cancel_ _Order_ button. If you’re able to cancel the order this way, no further action is required.

![](https://azure-na-images.contentstack.com/v3/assets/blta7903c6b840b702d/blt22c4b464ae3b8c0d/65fb292d812809000ab18c34/5996_Screenshot.png)

#### Option 2 – Fraudulent order shipped

If the fraudulent order has already been shipped, you need to initiate a **Stop Shipment** request with the shipping carrier you used. If you can successfully stop the shipment, have the item returned to you and refund the order.

Then, select the _Help_ button in the [Seller Center](https://seller.walmart.com/home) menu bar to contact Support with an invoice for the return shipping fee. Walmart will reimburse you for the charge. Include the Order Number and attach a copy of the "Fraud – Stop Shipment" email you received from us. Payment is typically disbursed within 60 days and is not part of the normal payment cycle. If you can’t stop the shipment, reply to the email notification you received from Walmart and let us know. A representative will follow up with you to reimburse the charge.

#### Option 3 – Request a fraud review

If you receive an order you think is fraudulent, and Walmart hasn't contacted you, then you may contact us to request a review. You can do this by selecting the _Cancel Order_ option, then select the code _Cancel – Fraud, stop shipment_  from the dropdown that appears in the popup window, and choose _Contact our Walmart Risk Prevention team_. In the email, include the order number you'd like reviewed. Walmart will review and notify you within four hours.

If Walmart doesn’t detect any fraud, you’re expected to fulfill the order normally. If fraud is identified, our team will cancel the order and notify you. Your performance will not be negatively impacted by orders cancelled for fraud.

Notes:

The _Cancel - Fraud, stop shipment_ cancel code is only available to use once Walmart confirms an order to be fraudulent. If you choose a different code, your performance could be negatively impacted, so be sure to select the proper code when canceling.

## Frequently asked questions

* * *

What happens if I’m unable to cancel a fraudulent order and it still gets delivered?

Do not refund the customer. You’ll be paid as part of the normal, regularly scheduled payment process if a fraudulent order is delivered.
