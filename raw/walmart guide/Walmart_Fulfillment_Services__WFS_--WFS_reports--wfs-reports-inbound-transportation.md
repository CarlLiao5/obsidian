WFS reports: Inbound transportation

Last updated on Feb 9, 2026

Reading time: 3 min

* * *

- [Dive into details](https://marketplacelearn.walmart.com/guides/Walmart%20Fulfillment%20Services%20(WFS)/WFS%20reports/wfs-reports-inbound-transportation#divein)

* * *

Overview

The Inbound transportationreport shows the details of your fees by shipment, including reasons for each charge and billable weight. This can be used as a comparison to the quoted information from purchase order creation.

## Access this report

1. Log in to Seller Center.
2. Navigate to the [**WFS reports**](https://seller.walmart.com/wfs/reports) page.
3. Under **Payments and Fees**, select _Download_ for the **Inbound transportation** report .
4. Set your desired time period.
5. Select _Download_ and open the report.

## Dive into details

### How to use this report effectively

- You can select time period by settlement date, to line up with your closed statements, or by a custom time period.
- Recent transactions will be available for review in your general payments statement once the statement period containing those transaction dates closes.
- The **Inbound transportation** report is refreshed in real time, although different transaction types post at different intervals (ranging from multiple times per day to one per week).
- See from and to information for each shipment, by shipment ID.

### Important insights

- See all inbound transportation fees and use the output to summarize by service type (parcel, LTL, FTL) or by reason code.
- Use the report to better understand quotes vs. actual charges for transportation.

## Understand report metrics

|     |     |
| --- | --- |
| **Column name** | **Description** |
| Shipment ID | Unique identifier for a shipment to a Walmart fulfillment center. |
| Delivery Date | Date the Walmart preferred carrier delivered to the destination. Items need to be checked in and received before they're available to sell. |
| Service Type | Shipping type, including small parcel, LTL (less-than truckload) or FTL (full truckload). |
| Reason Code | Charge type, including base and fuel surcharges that label your specific billing charges. |
| Actual Charge | Final amount billed to your account for using a Walmart preferred carrier. |
| Quote Amount | Amount that you confirmed to be preliminarily billed to your account to book the Walmart preferred carrier shipment. |
| Quote Weight | Weight you entered in the inbound order to calculate estimated price. |
| Quote Case or Pallet Count | Number of cases or pallets you entered in the inbound order to calculate estimated price. |
| Quote Freight Class | Freight class you entered, or we've calculated for you in the inbound order to calculate estimated price. |
| Actual Weight | Weight that the carrier measured. |
| Actual Billable Weight | Weight that the carrier used to charge for shipping. The carrier either uses dimensional weight (volume) or actual weight - whichever is greater. |
| Actual Case or Pallet Count | Number of cases or pallets that the carrier counted. |
| Actual Freight Class | Freight class that the carrier determined. |
| Ship From City | City where the shipment was picked up or dropped off. |
| Ship From State | State where the shipment was picked up or dropped off. |
| Ship To City | City where the Walmart facility is located. |
| Ship To State | State where the Walmart facility is located. |
| Comments | Details on why the quote and actual charge may differ. |
