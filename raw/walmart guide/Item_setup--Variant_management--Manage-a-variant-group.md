Manage a variant group

Last updated on Aug 20, 2025

Reading time: 5 min

* * *

In this guide:

- [Overview](https://marketplacelearn.walmart.com/guides/Item%20setup/Variant%20management/Manage-a-variant-group#overview)
- [Common scenarios](https://marketplacelearn.walmart.com/guides/Item%20setup/Variant%20management/Manage-a-variant-group#common)
- [Frequently asked questions](https://marketplacelearn.walmart.com/guides/Item%20setup/Variant%20management/Manage-a-variant-group#frequently-asked-questions)

* * *

Overview

Once you’ve created a variant group, you can manage it in several ways. You can edit the group, add or remove items, add new variant attributes and combine or separate groups. In this guide, you’ll learn how to manage a variant group in Seller Center.

Notes:

To make any changes to a variant group, you must own the brand or items in the variant group.

## Common Scenarios

### 1\. Edit a variant group.

#### What should I do?

To edit one item at a time, use the [**Catalog**](https://seller.walmart.com/catalog/list-items) page. To edit multiple items at once, use a template. You can edit a variant group’s product title, image and item attributes.

**Use the Catalog page**

Navigate to the **Catalog** page in Seller Center, select the _Filters_ button and select the _Variants_ drop-down menu. Choose _Variant item_ and select _Apply_. This filters your **Catalog** items so that only your variant groups are shown.

Locate the primary item for the variant group you want to edit, hoverover the item name and select the _Variant item_ button. Edit your information and select _Submit_.

Notes:

You can’t edit an item within a variant group that is controlled by Walmart. If you have questions about a variant item or a variant group, select the _Help_ button in the [Seller Center](https://seller.walmart.com/home) menu bar to contact Support.

**Use a template** Navigate to the [**Catalog**](https://seller.walmart.com/catalog/list-items) page in Seller Center, select the _Update items_ drop-down menu at the top of the page and choose _Update with GTINs_. Select the fulfillment method you want and select _Download file_ to download the template onto your computer.

To upload your completed template, return to the **Catalog** page and select _Add items_. Select the _Upload in bulk_ link located under **Add thousands of items using a spreadsheet**. You can drag and drop your file onto the browser window under **Upload**, or select the _browse_ link to locate the file on your computer.

Notes:

It can take up to four hours for your changes to appear on Walmart.com. If another seller or Walmart offers your item in a merged variant group and your changes aren’t reflected, select the _Help_ button in the [Seller Center](https://seller.walmart.com/home) menu bar to contact Support.

### 2\. Add or remove items in a variant group.

#### What should I do?

To add or remove one item at a time, use the **Catalog** page. To add or remove multiple items at once, use a template. To add an item that’s not currently in your catalog, you can create a new variant by copying the information from the primary SKU in the group.

**Use the Catalog page**

Navigate to the [**Catalog**](https://seller.walmart.com/catalog/list-items) page in Seller Center. Locate the primary item for the variant group you want to edit, hoverover the item name and select the _Variant item_ button. On the next page, select the checkbox next to the item you want to update.

- Add an item: Select _Add variants_ and then _Create new variants_. Add the GTIN of the new variant that you want to add to the group. Next, select _Add_ _variants_. Next, fill in the required fields and variant related information like product type. The attributes will be taken from the primary group. Finally, select _Submit_ and the newly created variant will be added to the existing variant group.

- Remove an item: Navigate to the **Actions** column for the item to be removed, select  _Remove Variant_ _s_. When you’re done, select _Update variants_.


Notes:

You can add existing items to a variant group only by using the template. To set up a new item and add it to the existing group, select _Add_ _variants_, then _Create new variants_.

**Use a template**

Navigate to the [**Catalog**](https://seller.walmart.com/catalog/list-items) page in Seller Center. Select the _Update items_ drop-down menu at the top of the page and choose _Update with GTINs_. Select the fulfillment method you want and select _Download file_ to download the template onto your computer.

Open the template and locate the variant group you want to edit. Locate the item you want to remove and navigate to the **Remove from Variant Group?** column. Select the drop-down menu and choose _Yes_.

To upload your completed template, return to the **Catalog** page and select _Add items_. Select the _Upload in bulk_ link located under **Add thousands of items using a spreadsheet**. You can drag and drop your file onto the browser window under **Upload**, or select the _browse_ link to locate the file on your computer.

Notes:

A variant group’s ID and attributes must match the existing group for an item to be added. Make sure that each variant item’s attributes are unique. Walmart may unpublish a variant item if duplicate variant values are found.

### 3\. Add new variant attributes.

#### What should I do?

To add one variant attribute at a time, use the **Catalog** page. To add or remove multiple variant attributes at once, use a template.

**Use the Catalog page**

Navigate to your [**Catalog**](https://seller.walmart.com/catalog/list-items) in Seller Center and locate any variant item that belongs to the group you want. Then navigate to the **Actions** column, select the column and choose _Manage variants_. Select _Add variants_ and then select _Create new variants_.

Enter the GTIN of an existing item you want to add to the variant group. You can add up to 500 items. Select _Add variants_ and complete the required information. The new item will be added to the existing variant group.

**Use a template**

Use the **Update all attributes** template. Navigate to the [**Catalog**](https://seller.walmart.com/catalog/list-items) page in Seller Center, select the _Update items_ drop-down menu at the top of the page and choose _Update with file_. Select the **Template** drop-down menu and choose _Update all attributes_. Choose your fulfillment type and then select the _Show Walmart templates_ link at the bottom of the window.

On the next page, select _Update all attributes_, choose your fulfillment type again and select _Next_. Choose the product types you want to include in your spreadsheet. When you’re done, select _Build spreadsheet_.

Open the template and add a new column for the new attribute. Complete the column so it corresponds with the new variant attribute. You can add variant attributes as needed.

### 4\. Combine or separate a variant group.

#### What should I do?

Use the **Update all attributes** template. Navigate to the [**Catalog**](https://seller.walmart.com/catalog/list-items) page, select the _Update items_ drop-down menu at the top of the page and choose _Update with file_. Select the **Template** drop-down menu and choose _Update all attributes_. Choose your fulfillment type and then select the _Show Walmart templates_ link at the bottom of the window.

On the next page, select _Update all attributes_, choose your fulfillment type again and select _Next_. Choose the product types you want to include in your spreadsheet. When you’re ready, select _Build spreadsheet_.

**Combine a variant group**

Open the template and locate the groups that you want to combine. Then set the same variant group ID for all items in all groups. Make sure that all of the attributes (such as **Variant Attribute Names** and **Swatch Images – Swatch Variant Attribute**) are identical.

To upload your completed template, return to the [**Catalog**](https://seller.walmart.com/catalog/list-items) page and select _Add items_. Select the _Upload in bulk_ link located under **Add thousands of items using a spreadsheet**. You can drag and drop your file onto the browser window under **Upload**, or select the _browse_ link to locate the file on your computer.

Notes:

If you’re combining variant groups, you must make sure that each group’s attributes match. For example, if one group is divided by color and another by size, the merged group must include values for both color and size. To avoid losing images, make sure to include **Swatch Images - Swatch Image URL** information for each item.

**Separate a variant group into two groups**

In the **Update all attributes** template, locate the variant group that you want to separate. Then assign a new variant group ID for the items you want to remove from your existing group and include in your new group. Make sure that all of the attributes (such as **Variant Attribute Names** and **Swatch Images – Swatch Variant Attribute**) are identical.

To upload your completed template, return to the [**Catalog**](https://seller.walmart.com/catalog/list-items) page and select _Add items_. Select the _Upload in bulk_ link located under **Add thousands of items using a spreadsheet**. You can drag and drop your file onto the browser window under **Upload**, or select the _browse_ link to locate the file on your computer.

## Frequently asked questions

* * *

How do I create a variant group?

You can create variant groups in Seller Center or with Walmart APIs. To learn more, visit [_Create a variant group in Seller Center_](https://marketplacelearn.walmart.com/guides/Item%20setup/Variant%20management/Create-a-variant-group:-Items-dashboard) or [_Create a variant group: Full setup_](https://marketplacelearn.walmart.com/guides/Item%20setup/Variant%20management/Create-a-variant-group:-Full-Setup-Template#frequently-asked-questions) to use APIs.

* * *

What happens when some of my variant group items match existing items in a variant group created by Walmart?

If you have 10 items varying in color and Walmart is selling five of those items in its own variant group, Walmart may merge the two groups to display on Walmart.com as a single group.

* * *

What happens when some of the items in my group match existing items in another seller's variant group?

Generally, your variant group will be merged with the other seller’s variant group. If you notice that a variant group should be combined but isn’t, select the _Help_ button in the [Seller Center](https://seller.walmart.com/home) menu bar to contact Support and request to merge the variant groups. For more information on merged groups, visit [_Product variant groups: Overview_](https://marketplacelearn.walmart.com/guides/Item%20setup/Variant%20management/Product-variant-groups:-Overview).

* * *

What happens when I send an item as standalone, but Walmart already has this item set up in a variant group?

Walmart will merge your item with the existing group.

* * *

What happens to my item when Walmart decides to make an item a standalone item?

The Walmart Marketplace item and listing become standalone even if you submitted the item as part of a group.

* * *

Do I need a unique GTIN?

Yes. Each product variant is a unique item with a unique identification number. Make sure to include a GTIN for each item in your variant group.

* * *

Can I copy items on a template?

Yes. You can continue using the item setup template to create new items. If you want the new item to be a part of the variant group, make sure that the appropriate variant group ID and variant attributes are added to the file.

* * *

Can I copy from an existing item that isn’t in a variant group yet?

No. Only the primary item in a variant group can be copied.

* * *

Can I provide more information for an item?

Yes. Locate the attribute that you want to update and enter your information into the text field.

* * *

Why can’t I submit my item?

Resolve all red-highlighted errors. You can also select missing fields to display columns that require attention.
