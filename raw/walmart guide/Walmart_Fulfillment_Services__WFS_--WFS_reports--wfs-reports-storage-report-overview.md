WFS reports: Storage

Last updated on Sep 24, 2025

Reading time: 2 min

* * *

* * *

Overview

A storage fee is the cost of storing your items at a Walmart fulfillment center. It's based on the volume of the product being stored and the length of time. The fees are based on a 30-day calendar month and are charged daily. The Storage report helps you better understand the fees your inventory has accrued in Walmart’s fulfillment centers.

## Access this report

To access the Storage report from Seller Center, log in and go to the [**Reports**](https://seller.walmart.com/wfsLite/reports) tab in the WFS section. This report will be located under the **Payments and fees** section. Select _Download_, then choose a report settlement date or custom time period to download the file.

## Dive into details

### How to use this report effectively

The top of the report will show you the total storage fees for the time period specified in the report start and end dates. This is further broken down by subtotals of standard, peak and long-term fees.

The body of the report will show you an item level breakdown of storage fees. Your Final storage fees are also reflected in your WFS Settlement report as a total amount.

For more information on how fees are calculated, please see the [_WFS Fees_](https://marketplacelearn.walmart.com/guides/Walmart%20Fulfillment%20Services%20(WFS)/WFS%20basics/WFS-fees) guide. You can also see real-time inventory age metrics in the [**WFS Overview page**](https://seller.walmart.com/wfs/wfs-overview).  For more details, see the [_WFS Overview page_](https://marketplacelearn.walmart.com/guides/Walmart%20Fulfillment%20Services%20(WFS)/WFS%20growth%20opportunities/wfs-overview-page) guide.

### Important insights

- Many times, items that are not selling can sit in our fulfillment centers for so long that they begin to accrue long-term storage fees. To learn about how you can move your slow sellers, see the [_Move aged inventory_](https://marketplacelearn.walmart.com/guides/move-aged-inventory) guide.


- This report helps you understand past fees charged. To learn about how to estimate future storage and fulfillment costs, use the [**WFS cost estimator**.](https://marketplace.walmart.com/walmart-fulfillment-services-pricing/)


## Understand report metrics

Top

|     |     |
| --- | --- |
| **Column name** | **Description** |
| Report start date | Fees will be included in the report if they occurred on or after the start date |
| Report end date | Fees will be included in the report if they occurred before or on the end date |
| Total storage fees | Total of all final storage fees calculated for the reporting period |
| Standard storage fees | Subtotal of storage fees across all units that were charged the standard storage fee (when item is stored fewer than 365 days during January to September) |
| Peak storage fees | Subtotal of storage fees across all units that were charged the peak storage fee (when item is stored during the peak months of October to December) |
| Long-term storage fees | Subtotal of storage fees across all units that were charged the long-term storage fee (when item is stored for more than 365 days) |

Bottom

|     |     |
| --- | --- |
| **Column name** | **Description** |
| GTIN | 14-digit Global Trade Item Number, including check digit, that's unique to the item. If it's less than 14 digits, add zeros at the beginning |
| SKU | Unique identifier created by you |
| Item ID | Unique identifier assigned by Walmart |
| Item name | Title of the item |
| Length (in) | Length of the packaged item, as measured by the fulfillment center |
| Width (in) | Width of the packaged item, as measured by the fulfillment center |
| Height (in) | Height of the packaged item, as measured by the fulfillment center |
| Weight (lb) | Weight of the packaged item, as measured by the fulfillment center |
| Standard: daily storage fee per unit | Daily fee when item is stored fewer than 365 days during January to September |
| Peak: daily storage fee per unit | Daily fee when item is stored during the peak months of October to December |
| Long-term: daily storage fee per unit | Daily fee when item is stored for more than 365 days |
| Standard: average units available | Average number of units on hand during this report period that were charged the standard storage fee |
| Peak: average units available | Average number of units on hand during this report period that were charged the peak storage fee |
| Long-term: average units available | Average number of units on hand during this report period that were charged the long-term storage fee |
| Days in report period | Number of days used to calculate the final storage fee |
| Final storage fee | The final storage fee is the sum of all your storage fees incurred for the reporting period for the item. <br>If you’ve qualified for any discounts, they will be reflected here. |
