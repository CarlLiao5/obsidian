Create a variant group in Seller Center

Last updated on Aug 20, 2025

Reading time: 6 min

* * *

In this guide:

- [Overview](https://marketplacelearn.walmart.com/guides/Item%20setup/Variant%20management/Create-a-variant-group:-Items-dashboard#overview)
- [Create a new variant group](https://marketplacelearn.walmart.com/guides/Item%20setup/Variant%20management/Create-a-variant-group:-Items-dashboard#items)
- [Frequently asked questions](https://marketplacelearn.walmart.com/guides/Item%20setup/Variant%20management/Create-a-variant-group:-Items-dashboard#frequently-asked-questions)

* * *

Overview

As a Marketplace seller, you can create variant groups in Seller Center or using [Walmart APIs](https://developer.walmart.com/doc/us/mp/us-mp-items/). In this guide, you’ll learn how to create a variant group for a group of up to 49 existing items using the **Catalog** page.

Notes:

Too many variations can be confusing for customers. We recommend no more than 50 items for a variant group with a single attribute (a T-shirt in two colors and three sizes) and no more than 500 items for a variant group with multiple attributes (a pair of pants in 25 sizes and 20 colors).

## Create a new variant group

### Step 1 — Get started

Navigate to the [**Catalog**](https://seller.walmart.com/catalog/list-items) page in Seller Center. Next, search for your items by SKU or Item Name using the search bar. Select the check box next to each item you want included in your variant group. Select the More actionsdropdown menu, then select _Create_ _v_ _ariant_ _g_ _roup_. Select _Create variant group_ again when the popup window appears.

![](https://azure-na-images.contentstack.com/v3/assets/blta7903c6b840b702d/blt10474cee1a42af42/6753266183ef6d4cab713166/2412_11477_Screenshot.png?branch=test2)

Notes:

If you try to include more than 49 items in your group using the **Catalog** page, you'll get an error message. Selected Items must be part of the same product type group.

### Step 2 — Add the group details

On the next screen, enter the details in each boxaccordingly:

- **Choose a** **v** **ariant** **g** **roup ID:** The variant group ID is a unique alphanumeric identifier you'll choose for this group, up to 20 characters long (for example, DryFit\_VG5). You’ll use it to track your variant group in the **Catalog** dashboard once it’s been created.


- **Select Primary Variant**: Select the primary variant to be displayed in search. Primary variants provide suggestions on what item should be highlighted in search for the variant group. The primary variant highlighted for the group and the in search may vary as Walmart determines what item is displayed to the customer. Select the _star_ icon next to the item to make your selection.


- **Select** **v** **ariant** **a** **ttribute** **n** **ames:** Select up to three variant attribute names for the group. These indicate which attributes vary within the group, such as size or color. For example, if an item comes in small, medium and large, the variant attribute name would be size.


Notes:

You can’t create multiple variants with identical values. Variant attributes that have the same values, such as size and color, on multiple items will be flagged as an error. Review your items for accuracy and make necessary changes to item details and variants before submitting.

### Step 3 — Remove items if needed (optional)

If there’s an item you don’t want to include in the group, select _X_ in the **Actions** column to remove it.

Notes:

There’s no undo function when you remove an item using the **Actions** column. Make sure you remove the correct item, or you’ll have to start the process from the beginning.

### Step 4 – Confirm details and submit

When all fields are complete, select _Submit._

## Frequently asked questions

* * *

How do I add or remove items from an existing variant group?

Check out  [_Manage a variant group_](https://marketplacelearn.walmart.com/guides/Item%20setup/Variant%20management/Manage-a-variant-group) to learn how to edit your groups.

* * *

What if I create a variant group that contains items sold by another seller?

Your groups will merge to appear on the same page on Walmart.com. While you can group, modify and delete your own variant groups, any items in other groups will continue to appear on the item page. Consequently, you may not always be able to define which content will be displayed on a product page if there are multiple sellers providing content for the same item. Walmart will determine the best available content for items unless provided by a brand owner or authorized reseller.

* * *

The variant attribute name that I want to use isn’t on the Supported Variant Attribute Names list. What should I do?

You can’t create custom variant attributes. Choosing the correct product type is important because variant attribute names are populated based on the product type of the items in the group.

Check to see if there’s another way you could set up your item using a supported variant attribute name. For example, if you’re selling vitamins in one-, two- and three-bottle packs, you could set them up to vary by count instead of by bottle. If you’re unable to find an alternative like this, you shouldn't set up your items in a variant group.

* * *

How does Walmart determine the ranking of variant attributes and how they show up on the item page?

Every subcategory or item type has a ranked list of variant attributes that defines how the attributes show up on the item page. For example, clothing items will always list the item’s size first followed by the color.

* * *

How many different variants can I add to an item?

That depends on the variant. For a single variant item, there's a limit of 50-100 variants—for example, a towel that comes in 100 different colors. For a multi-variant item, there’s a limit of 1,000 variants—for example, pants with 25 different sizes and 20 color options.

* * *

How can I change the primary variant if it’s out of stock?

If the primary variant goes out of stock, the grouping remains valid, with the primary variant showing as out of stock on the item page. If you decide to discontinue the  primary variant and never add inventory again, you have the option to set a different variant as the primary by updating the variant group list with a different primary item selected. If your item is in a merged variant group and the other seller's primary variant is out of stock, select the _Help_ button in the [Seller Center](https://seller.walmart.com/home) menu bar to contact Support.

* * *

How can I discontinue the primary variant in a variant group when it matches a variant group set up by Walmart?

The Walmart variant group takes precedence. Your items will be included under Walmart's group if all the items match the items in that group. Matching is done using the item's unique item ID (for example, the UPC or ISBN). If you decide to discontinue the primary variant (by never adding inventory again), you have the option to set a different variant as the primary by updating the variant group ID grouping with a different primary variant selected.
