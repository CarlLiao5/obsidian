Troubleshoot item setup errors

Last updated on Oct 2, 2025

Reading time: 5 min

* * *

Overview

Our system automatically scans every Walmart template for errors. It checks for missing required attributes, incorrect data types, invalid check digits, copy-paste errors and more. This guide shows you how to find and address errors if your template fails the pre- or post-validation check.

## Common upload errors

### 1\. Invalid product ID

The error message could identify invalid Product ID for the selected Product Identifier and will tell you which row has the error.

**What should I do?**

Correct the error in the file then save the file and upload it back in Seller Center. Check out [_Troubleshoot Product ID Errors_](https://marketplacelearn.walmart.com/guides/Item%20setup/Troubleshooting/Troubleshoot-product-ID-errors) to learn more.

### 2\. Missing attribute metadata

The error message could identify missing attribute metadata.

**What should I do?**

You should download a new spreadsheet if you are not sure what was deleted/altered. Make sure not to delete any data in Row 1-6. If you added multi-select attributes, make sure that all the information in rows 4-6 were copied and pasted into the new columns. Check out [this guide](https://marketplacelearn.walmart.com/guides/Item%20setup/Item%20setup/Add-items-in-bulk:-full-setup) to learn more about multi-select attributes.

### 3\. Error file

If you encounter an error while uploading an item setup spreadsheet, an **Upload error** message will automatically download the file. In this error report, you can easily identify the number of errors and their descriptions in columns A and B. All cells that contain errors which need to be fixed will be highlighted.

**What should I do?**

Correct the errors in the highlighted cells, then re-upload the template when finished. Once your upload completes, navigate to the [**Activity Feed**](https://seller.walmart.com/item/activity-feeds?feedsFor=uploads) dashboard to confirm the template processed successfully. Review [this guide](https://marketplacelearn.walmart.com/guides/Item%20setup/Troubleshooting/Troubleshoot-product-ID-errors?locale=en-US) if the error relates to an invalid product ID.

Pro tip

Before uploading your template, double-check that you’ve filled out all of the required attributes, that they are within the character limit and that attributes with closed-list options have been properly selected.

## Common post-upload errors

### 1\. Main Image URL does not meet our image URL requirements

**What should I do?**

Review [this guide](https://marketplacelearn.walmart.com/guides/Item%20setup/Item%20content,%20imagery,%20and%20media/Product-detail-page:-Image-guidelines-&-requirements) to view our image URL requirements and update your content accordingly **.** You can make updates within the report or within the **Full Item Setup** template. Then, re-upload the document into Seller Center and check the [**Activity Feed**](https://seller.walmart.com/item/activity-feeds?feedsFor=uploads) to make sure your errors are resolved.

### 2\. You are not authorized to set up ‘CUSTOM’ Product IDs for UPC exemptions

**What should I do?**

Find the correct product ID for your item. If you sell products that don’t have a product ID (e.g., private label or handmade jewelry), you can request a UPC or GTIN exemption to sell these items on Walmart.com. To request an exemption, select the _Help_ button in the [Seller Center](https://seller.walmart.com/home) menu bar to contact Support.

If the item you want to sell is something you’ve manufactured or produced, you may have to purchase a product ID. You may choose to purchase a GTIN, UPC or EAN from GS1 [here](https://www.gs1us.org/upcs-barcodes-prefixes/how-to-get-a-upc-barcode?utm_source=partner&utm_medium=referral&utm_campaign=idgtm-walmart-). If you choose to purchase an ISBN, consult the appropriate authority by country through using the [ISBN lookup tool](https://www.isbn-international.org/agencies).

### 3\. The item must have a Variant Group ID, Variant Attribute Name, and 'Is Primary Variant' value to be set up correctly on the website

**What should I do?**

Update each appropriate column by following the requirements listed in [this guide](https://marketplacelearn.walmart.com/guides/Item%20setup/Variant%20management/Create-a-variant-group:-Full-Setup-Template).

## Common system errors

If there is a technical issue in our system, you may receive a feed error when setting up your item. Wait a day as our system automatically attempts to reprocess your submission every few hours. If we’re unable to process your submission, select the _Help_ button in the [Seller Center](https://seller.walmart.com/home) menu bar to contact Support. Be sure to include the Feed ID and the spreadsheet.

## Walmart-fulfilled (WFS) errors

### 1\. Battery type is a required attribute

If your item is a battery or battery-containing item, we require follow-up information. Learn more about [WFS compliance and batteries](https://marketplacelearn.walmart.com/guides/Walmart%20Fulfillment%20Services%20(WFS)/WFS%20item%20setup/WFS-compliance-batteries-FAQ) _._

**What should I do?**

Update the item in your template. Open your template, navigate to the **Required for Walmart Fulfillment Services** section and locate the **Product is or Contains this Battery Type** column. Select the drop-down menu arrow and choose a battery type, such as _Lead Acid_ or _Lithium Ion_. You can find this information on the battery, on its packaging or in its [Safety Data Sheet](https://marketplacelearn.walmart.com/guides/Walmart%20Fulfillment%20Services%20(WFS)/WFS%20item%20setup/WFS-compliance-safety-data-sheets?locale=en-US). If your item doesn’t contain a battery, choose _Does Not Contain a Battery_.

### 2\. Hazmat (hazardous material) assessment is required

If your item is a [hazardous material](https://marketplacelearn.walmart.com/guides/Walmart%20Fulfillment%20Services%20(WFS)/WFS%20item%20setup/wfs-hazardous-materials-overview) or may contain a hazardous material, the item will be put on hold until the Walmart compliance team can review it. This can take up to three business days.

**What should I do?**

Wait for three business days and then check your item’s status on the [**Pending review**](https://seller.walmart.com/catalog/pending-review) page in Seller Center. Review the **Status** column for any items that are marked as _Action Needed_ and correct them. Learn more about [WFS hazardous materials and item setup](https://marketplacelearn.walmart.com/guides/Walmart%20Fulfillment%20Services%20(WFS)/WFS%20item%20setup/WFS-hazmat-item-setup) _._

### 3\. Item may contain a chemical, aerosol or pesticide

The Walmart compliance team has determined that your item may be a [hazardous material](https://marketplacelearn.walmart.com/guides/Walmart%20Fulfillment%20Services%20(WFS)/WFS%20policies%20&%20standards/wfs-hazardous-materials-overview) or may contain a hazardous material. You must specify whether the hazardous material is a chemical, aerosol or pesticide and answer any follow-up questions from the compliance team.

**What should I do?**

Update the item in your template. Open your template, navigate to the **Required for Walmart Fulfillment Services** section and locate the column title that matches your item (such as **Product is or contains a chemical**). Select the first blank cell in the column, select the drop-down menu arrow and choose _Yes_. Then re-upload your template to Seller Center.

## Frequently asked questions

* * *

Where can I learn more about adding items in bulk?

Check out [_Add multiple items: Full setup_](https://marketplacelearn.walmart.com/guides/Item%20setup/Item%20setup%20methods/Add-items-in-bulk:-full-setup) for details and guidance about adding multiple items.

* * *

How do I check for and resolve errors?

To check if your upload has errors, filter your results in the [**Activity Feed**](https://seller.walmart.com/item/activity-feeds?feedsFor=uploads) by selecting the _Filters_ button > _Feed Type_ menu > _Item Setup box_ \> _Apply_. View the **Error Report** column for items you recently uploaded, then select the download _icon_ to get the report.

You can also resolve errors for individual items directly from the [**Activity Feed**](https://seller.walmart.com/item/activity-feeds?feedsFor=uploads) dashboard in Seller Center. Select the eye icon under the **Errors** column and choose _Fix Error._ This opens a **Single Item Edit** view with your submitted item details. Navigate to the tab or section highlighted in red and fix the errors. When your updates are complete, select _Submit_.

* * *

What should I do if I still can't troubleshoot the errors?

Check for formatting errors in your template, as they may cause our system to read the file incorrectly.

- Some cells in the template contain dropdown lists. Make sure to choose from the list. Don’t paste or type into these cells, as this may cause an error in the formatting.

- If you need to copy data from another source, only paste the values and not the formatting. Right-click to open _Paste Special_ and select _Values._

- Avoid special characters, like the slash in "N/A." You can skip this field if it's not required.

- For Walmart-fulfilled items, make sure both the category and trade item configuration tabs are filled in. The information must be in the same order in both tabs of the sheet.


If you’re still having difficulty, select the _Help_ button in the [Seller Center](https://seller.walmart.com/home) menu bar to contact Support and include the Feed ID, impacted SKUs or product IDs and the Full Setup template.
