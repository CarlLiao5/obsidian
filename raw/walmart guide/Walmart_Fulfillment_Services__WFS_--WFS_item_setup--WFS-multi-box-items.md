WFS multi-box: Set up items

Last updated on Jul 14, 2025

Reading time: 9 min

* * *

Overview

_Multi-box_ means a single item that’s shipped in multiple boxes. Learn how to add, convert and edit these items in your Walmart-fulfilled catalog. You’ll use a template to do most tasks.

For guidelines and fees, check out [_WFS multi-box: Overview_](https://marketplacelearn.walmart.com/guides/Walmart%20Fulfillment%20Services%20(WFS)/WFS%20programs%20&%20services/wfs-multi-box-overview).

## Add new sellable sets

### Step 1 – Get started

Navigate to [**Catalog**](https://seller.walmart.com/catalog/list-items) in Seller Center and select _Add items_. Select the _Upload in bulk_ link located under **Add thousands of items using a spreadsheet**. On the next page, choose the **WFS multi-box** option under the **Download a spreadsheet** section.

### Step 2 – Add sellable GTINs

To build a multi-box template, add GTINs for up to 10 sellable sets. Then select _Add_ to move on.

![](https://azure-na-images.contentstack.com/v3/assets/blta7903c6b840b702d/blt107fb1eb439fdd10/64fa4acc103154000dae624a/Multi-Box_3.png?branch=us_mplearn)

Pro tip

For definitions and GTIN requirements, go to [_WFS multi-box: Overview_](https://marketplacelearn.walmart.com/guides/Walmart%20Fulfillment%20Services%20(WFS)/WFS%20programs%20&%20services/wfs-multi-box-overview).

### Step 3 – Add box GTINs

For each sellable GTIN, add the category, subcategory and product type. Then add the boxes that make up the set. You must have at least 2 boxes per sellable set, and the box GTINs must be unique. When you’re done adding all box GTINs, select _Save_. When you’re done, select _Build template_.

![](https://azure-na-images.contentstack.com/v3/assets/blta7903c6b840b702d/blt596a4d5f4eb243de/6717f216f94705096ab6f89f/Screenshot_2024-10-22_at_11.37.26_AM.png?branch=us_mplearn)

### Step 4 – Download template

Go to [**Activity Feed**](https://seller.walmart.com/item/activity-feeds?feedsFor=downloads) and download the template from the **Downloads** tab. Then fill in the remaining sections in the template. (For tips, go to [_Full setup template for WFS_](https://marketplacelearn.walmart.com/guides/Walmart%20Fulfillment%20Services%20(WFS)/WFS%20item%20setup/WFS-item-spec-sheet-setup).)

Notes:

Use descriptive names for component titles (“Chair Cushions” and “Chair Back and Legs”) rather than “Box A” and “Box B,” so they’re easy to tell apart.

### Step 5 – Upload template

Once you’ve filled out the template, go back to [**Catalog**](https://seller.walmart.com/catalog/list-items) and select _Add items_. Select the _Upload in_ bulk link, then upload your file on the next page.

Use the feed ID to track progress in the **Activity Feed**. If there are no issues, you’ll see all the sellable sets in the **Catalog** page. You can now start sending inventory to WFS.

## Hazardous materials

If your boxes contain regulated components such as [battery, chemical, aerosol or pesticide](https://sellerhelp.walmart.com/s/guide?language=en_US&article=000009156), they’ll be put on hold until they’re reviewed. A compliance review can take up to 3 business days. Boxes with complete, correct compliance information will be reviewed more quickly. Boxes with incomplete, inaccurate or conflicting compliance information will take more time.

Notes:

Make sure the compliance information for _each_ box is correct. Not disclosing that a box contains a regulated component — or incorrectly declaring that a box contains a regulated component when it doesn’t — may cause delays.

Look for your on-hold items in the **WFS Compliance** tab in the [**Pending Review**](https://seller.walmart.com/items-and-inventory/items-on-hold) page. If there are errors, you can view them from this page. Make sure to submit a single, corrected file for the whole sellable set, rather than updating just a box GTIN in Seller Center. After 30 days, error files will expire.

If more than 1 box is on hold, wait until they’ve all been reviewed so you only have to resubmit once. Once you’ve corrected any errors, the boxes will go through a compliance review again to confirm changes. For more on the compliance review process, go to [_WFS hazardous materials item setup_](https://marketplacelearn.walmart.com/guides/Walmart%20Fulfillment%20Services%20(WFS)/WFS%20item%20setup/WFS-hazmat-item-setup).

## Convert from seller-fulfilled

If you’ve been fulfilling a multi-box item on your own but also want to use WFS, follow the same steps as [adding a new multi-box item](https://marketplacelearn.walmart.com/guides/Walmart%20Fulfillment%20Services%20(WFS)/WFS%20item%20setup/WFS-multi-box-items#addnewsellablesets). The only difference is you’ll need to [prepare new GTINs](https://marketplacelearn.walmart.com/guides/Walmart%20Fulfillment%20Services%20(WFS)/WFS%20item%20setup/WFS-multi-box-items#frequently-asked-questions) for each box.

Notes:

Converting to WFS is permanent for multi-box items. Once you convert a sellable set, you cannot revert that GTIN to only seller-fulfilled. You’ll always need to manage inventory at the box level and not just the sellable level. The default fulfillment method will be Walmart-fulfilled, but it’ll automatically [switch to seller-fulfilled](https://marketplacelearn.walmart.com/guides/Walmart%20Fulfillment%20Services%20(WFS)/Troubleshooting/Troubleshoot-Walmart-fulfilled-orders-switching-to-seller-fulfilled?locale=en-US) if inventory for the complete sellable set runs out.

If you want to switch to only Walmart-fulfilled, you’ll need to back out the seller-fulfilled inventory to zero and then send inventory to WFS.

1. Go to the [**Catalog**](https://seller.walmart.com/item/list) page
2. Find your seller-fulfilled item. Select **View**.
3. Change the _On hand_ quantity for each fulfillment center to 0.

You can also [update items in bulk](https://marketplacelearn.walmart.com/guides/Catalog%20management/Item%20management/Update-item-content-in-bulk-in-Seller-Center) to change inventory for multiple items. Manage seller-fulfilled inventory on the **Catalog** page and WFS inventory on the **Inventory** page.

## Packing setup for multi-box items

Notes:

After you’ve set up a sellable set, you can’t change how it’s packed.

All boxes must be shipped as specified during item setup. For example, if your listing states that a sellable set will be shipped in 2 boxes, you cannot ship in 3 boxes.

In addition, you must ship the exact items described in your listing. You **cannot substitute different sizes, colors, quantities** or any other characteristics. For example, if you said that a box has a 12 oz container of liquid, you cannot replace it with a 9 oz bottle. You’d need to create a new sellable set with new GTINs.

A sellable set must always have the same box setup. For example: An item is listed with a sellable GTIN of 11111111111111. It’s made up of box GTINs 22222222222222 and 33333333333333. You cannot have another listing that’s sellable GTIN 11111111111111, but with different box GTINs of 33333333333333 and 44444444444444.

## Send multi-box inventory to WFS

Once your sellable sets are in your catalog, you can [send us inventory](https://marketplacelearn.walmart.com/guides/Walmart%20Fulfillment%20Services%20(WFS)/Shipping%20to%20WFS/WFS-inbound-orders:-Send-inventory) like you normally would. For the shipping plan, you’ll add units at the box level (e.g., number of armchairs and loveseats rather than the living room set).

Send enough inventory for complete sellable sets. Make sure to account for boxes that are shared between sellable sets, so your items don’t go out of stock. For example, if a loveseat is part of both a living room set and office set, send enough loveseats to complete both sets. Our associates will then pick, pack and ship each box to fulfill the sellable set.

## Frequently asked questions

* * *

How do I get a new Global Trade Item Number (GTIN) for my boxes?

You can create and license new GTINs from organizations such as [GS1](https://www.gs1us.org/upcs-barcodes-prefixes/how-to-get-a-upc-barcode?utm_source=partner&utm_medium=referral&utm_campaign=idgtm-walmart-). Each box needs its own GTIN. Do not reuse GTINs from existing, standalone items or sellable sets.

* * *

Can a sellable set have 2 different GTINs?

No, the sellable GTIN must be unique to the multi-box item. For example, you may have a sellable set (00111111111112) with 2 boxes (00111111111195 and 00111111111196). You cannot create another listing using the same combination of sellable GTIN and box GTINs. You'll either need to assign a new sellable GTIN or change at least one of the box GTINs.

* * *

How do I know the quantity of a box GTIN?

A box GTIN reflects the lowest level of how units are shipped and sold to the customer. For example:

- A patio set has 2 chairs. You’re sending each chair as 1 unique component. The box GTIN quantity is 2 per patio set.
- Let’s say you send the 2 chairs in one box, and the chairs are never to be separated. The box GTIN quantity is now 1 per patio set.

* * *

How do I handle variants for multi-box items?

There’s no change to the [current process](https://marketplacelearn.walmart.com/guides/Item%20setup/Variant%20management/Product-variant-groups:-Overview) for variants. Every variant has its own sellable GTIN. If you enter the correct GTIN, the template will pull the data and create variants for your multi-box items.

When you convert a variant group to multi-box, make sure the primary variant stays the same. If the “Is Primary Variant” column is inaccurate, update it before submitting the template. For example, this image shows that the item doesn’t have a primary variant, even though the original variant group had one.

![](https://azure-na-images.contentstack.com/v3/assets/blta7903c6b840b702d/blt27d02d23c6d6ee9a/6500fd5e6517c0000dfc9fdd/multi-box_FAQs.png?branch=us_mplearn)

* * *

Can I set up or edit multi-box items through API?

Unfortunately, APIs are not available for multi-box at this time.

* * *

How are multi-box items priced?

The sellable set’s price is the sum product of the box GTIN quantity and box price. Once you’ve set up the set, this ratio will stay the same whenever you update the sellable set’s price. For example:

- You have a patio set with a table at $100 and 2 chairs at $40 each. The total price will be set to $180.


- Let’s say you change the total price to $160. The box prices change based on percentage of box GTIN quantity at the time of setup. In this case, each chair was 22.22% of the total price and is now priced at $35.56.


The percentages can’t be changed, so make sure you choose box prices that reflect their relative values. In the event of loss or damage, this percentage will determine the amount reimbursed to you.

* * *

What happens to pricing once I convert an item?

Whether you add a new sellable set or convert an existing one, the pricing works the same way. The prices of the individual boxes will determine the ratio and overall price of the sellable set. If you convert a set, its price will become the total of the box prices.

If needed, you can update the sellable set price _after_ converting. Just go to the [**Catalog**](https://seller.walmart.com/items-and-inventory/manage-items) page and select the sellable set to [edit the offer details](https://marketplacelearn.walmart.com/guides/Catalog%20management/Item%20management/How-to-update-content:-Overview). Or if you need to update the prices of multiple items, use the [price template](https://marketplacelearn.walmart.com/guides/Catalog%20management/Price%20management/Update-price-in-bulk-in-Seller-Center).

Note that you _cannot_ change the prices for boxes. Those prices are set in our system, so you can only manage prices at the sellable GTIN level.

* * *

How does the referral fee work for multi-box items?

We charge the referral fee by the box. You can check your statement on the **Payments** page for fees.

* * *

How do I update attributes of my multi-box item?

Use the same multi-box setup template you used to [add the item](https://marketplacelearn.walmart.com/guides/Walmart%20Fulfillment%20Services%20(WFS)/WFS%20item%20setup/WFS-multi-box-items#addnewsellablesets). If you change any attributes using a regular item maintenance template or any other processes in Seller Center, your item won’t be updated.

* * *

How do I track sales, inventory and returns for multi-box items?

Go to [_WFS multi-box: Manage inventory_](https://marketplacelearn.walmart.com/guides/manage-wfs-multi-box-inventory) to learn more.
