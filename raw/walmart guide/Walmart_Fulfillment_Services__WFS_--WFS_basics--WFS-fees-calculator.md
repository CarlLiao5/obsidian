Overview

The [Walmart Fulfillment Services (WFS) Cost Estimator](https://marketplace.walmart.com/walmart-fulfillment-services-pricing/) is an easy-to-use tool to estimate your fees for a Walmart-fulfilled item. This includes the fulfillment fee, Walmart referral fee, Prep Services and storage fees. It’s a great way to preview costs for your current Walmart-fulfilled items, as well as any items you’re thinking of selling on Walmart.com. In this guide, we’ll break down examples of how the fees are calculated.

## How does it work?

You can choose between metric units (centimeters and kilograms) and imperial units (inches and pounds). To estimate costs for a single item, just enter your item’s weight, dimensions and any additional details that apply. Make sure your item measurements include the manufacturer packaging _before_ it’s packaged for shipping.

If you have multiple items, switch to the _Bulk estimate_ tab to quickly calculate costs for your catalog. You’ll need to download the template, then fill in details for each item. Columns marked with asterisks are required. When you’re done, upload the template and then download a new file with the estimated costs in a new tab. It will look like this:

![](https://azure-na-images.contentstack.com/v3/assets/blta7903c6b840b702d/blt13b791a59e427b82/68c4684b5c76c07c7eee4cf0/bulk_calculator_2.png?branch=us_mplearn)

## Examples

Here are examples of fulfillment and storage fees with the WFS Cost Estimator.\* They also show a breakdown of how those fees are calculated. For more detail on how fees work, go to [_WFS fees_](https://marketplacelearn.walmart.com/guides/Shipping%20&%20fulfillment/Walmart%20Fulfillment%20Services%20(WFS)/WFS-fees?locale=en-US).

### Example 1: Standard WFS item

John sells bike baskets. A single basket unit is 12" x 10" x 10" and weighs 6 lb. John begins storing the inventory in March. He uses the WFS Cost Estimator to calculate his fees for each basket unit:

![](https://azure-na-images.contentstack.com/v3/assets/blta7903c6b840b702d/blt3f55fa9613d82d83/68c4682dcd276b511abc5d77/calculator_example_1.png?branch=us_mplearn)

#### Fulfillment fee

|     |     |
| --- | --- |
| **Unit weight** | 6 lb. |
| **Dimensional weight** | (12 x 10 x 10) / 139 = 8.63 lb. |
| **Shipping weight** | The dimensional weight is greater than the unit weight, so John uses the dimensional weight to determine the shipping weight:<br>- 8.63 weight + 0.25 packing = 8.88 lb.<br>- Rounded up = **9 lb.** |
| **Fulfillment fee** | John's item is in the 4–20 lb. shipping weight category, so he uses this formula:<br>- $5.75 + $2.00 \[$0.40 per additional lb. over 4 lb.\] = **$7.75** |

#### Storage fee

|     |     |
| --- | --- |
| **Unit cubic feet** | (12 x 10 x 10) / 1728 = 0.694 |
| **Monthly storage fee per unit** | 0.694 x $0.75 = **$0.52** |

### Example 2: Standard WFS item with oversize

Anita sells shelves. Each shelf unit is 10" x 40" x 40" and weighs 10 lb. She begins storing the inventory in June. She uses the WFS Cost Estimator to calculate her fees for each shelf unit:

![](https://azure-na-images.contentstack.com/v3/assets/blta7903c6b840b702d/blteeeaf3500f460019/68c4683b8724bfc2f7ca72a1/calculator_example_2.png?branch=us_mplearn)

#### Fulfillment fee

|     |     |
| --- | --- |
| **Actual unit weight** | 10 lb. |
| **Minimum oversize-item unit weight\*** | 90 lb. |
| **Dimensional weight** | (10 x 40 x 40)/139 = 115.11 lb. |
| **Shipping weight** | The dimensional weight is greater than both unit weights, so Anita uses the dimensional weight to determine the shipping weight:<br>- 115.11 weight + 0.25 packing = 115.36 lb.<br>- Rounded up = **116 lb.** |
| **Base fulfillment fee** | Anita's item has a shipping weight greater than 51 lb., so she uses this formula:<br>- $17.55 + $26.00 \[$0.40 per additional lb. over 51 lb.\] = $43.55 |
| **Oversize fee\*\*** | Anita’s item has a longest side + girth is between 130" and 165," so she adds this fee:<br>- $20 |
| **Total fulfillment fee** | $43.55+ $28.00 = **$63.55** |

\*\*This applies because the item’s longest side + girth is between 130" and 165"

#### Storage fee

|     |     |
| --- | --- |
| **Unit cubic feet** | (10.0 x 40.0 x 40.0) / 1728 = 9.259 |
| **Monthly storage fee per unit** | 9.259 x $0.75 = **$6.94** |

### Example 3: Big and bulky item

Sam sells kayaks. Each kayak is 120" x 28" x 14" and weighs 36 lb. She begins storing the inventory in March. She uses the WFS Cost Estimator to calculate her fees for each kayak:

![](https://azure-na-images.contentstack.com/v3/assets/blta7903c6b840b702d/bltaaecc82168526a47/68c468440fa6f138726d090a/calculator_example_3.png?branch=us_mplearn)

#### Fulfillment fee

Though the kayak is lightweight, it’s considered a [big and bulky item](https://marketplacelearn.walmart.com/guides/Walmart%20Fulfillment%20Services%20(WFS)/Shipping%20to%20WFS/WFS-big-and-bulky-items) because its longest side is over 108 inches and its length + girth is over 165 inches.

|     |     |
| --- | --- |
| **Longest side** | 120" |
| **Length + girth** | 120" + 2\*(28" + 14") = 204" |
| **Shipping weight** | 36 lb. |
| **Fulfillment fee** | Sam's item is in the Big and Bulky tier, so she uses this formula:<br>- $155 + $0 \[$0.80 per additional lb. over 90 lb.\] = **$155** |

#### Storage fee

|     |     |
| --- | --- |
| **Unit cubic feet** | (120 x 28 x 14) / 1728 = 27.22 |
| **Monthly storage fee per unit** | 27.22 x $0.75 = **$20.42** |

\*The examples provided in this document are for illustrative and educational purposes only. They are not intended as legal, financial, or professional advice. Situations may vary, and users should exercise their own judgment and discretion when applying any concepts described herein.

## Frequently asked questions

* * *

Do the fees shown in the estimator accurately represent what WFS will charge me?

The fees calculated by the WFS Cost Estimator are accurate based on the information that you entered at the time. It is an estimate only and not a guarantee of actual costs. The estimate also does not include shipping fees, return processing fees and other costs.

* * *

Why are there differences between the estimator and the Settlement Report?

Item dimensions and weight are remeasured at our facilities, so that may cause minor differences.  If you notice large differences between the estimator and the Settlement Report, please contact Support by selecting the _Help_ button in the Seller Center menu bar.

* * *

Does the estimator support metric units of weight?

Yes, the estimator supports metric units (centimeters and kilograms) and imperial units (inches and pounds). For a single item, select the units in the _Units of measurement_ dropdown. For the bulk template, look for the _Units of measure_ column and select from the dropdown.

* * *

Is there a multi-unit discount on the fulfillment fees in WFS?

Fulfillment fees are charged per unit fulfilled. Some exceptions may apply if your item is a [virtual pack](https://marketplacelearn.walmart.com/guides/Item%20setup/Item%20setup%20methods/Create-a-virtual-pack:-full-setup-template) or fulfilled with [Multichannel Solutions](https://marketplacelearn.walmart.com/guides/Walmart%20Fulfillment%20Services%20(WFS)/Walmart%20Multichannel%20Solutions/walmart-multichannel-solutions?locale=en-US).

* * *

Is there a restocking fee for items that are returned but can be resold?

There is no customer-related restocking fee. Sellers are responsible for paying the [return processing fee](https://marketplacelearn.walmart.com/guides/Shipping%20&%20fulfillment/Walmart%20Fulfillment%20Services%20(WFS)/WFS-fees#Return_processing_fees_exceptions) for any returns that are not identified as _Walmart at fault_.

* * *

Does the WFS fee structure include shipping costs?

Yes, fulfillment fees cover receiving, stowing, picking, packing and shipping. Additionally, the fee structure covers customer support and non-seller fault returns.

* * *

What are the referral or commission fees if I sell with WFS?

The Marketplace referral fee is a percentage of the gross sales of each item sold. We only deduct this commission once you complete a sale. The fee varies [by category](https://marketplace.walmart.com/pricing/#1714691464282-e856902c-8ac3). Enter your item’s retail list price and item category in the WFS Cost Estimator to calculate the referral fee.

* * *

Am I responsible for remitting sales tax?

No additional taxes apply to WFS. Refer to [_Sales tax collection_](https://marketplacelearn.walmart.com/guides/Taxes%20&%20payments/Tax%20information/Sales-tax-collection) for more information about remitting sales tax.
