Create item-level return rules

Last updated on Dec 22, 2025

Reading time: 2 min

* * *

In this guide:

- [Overview](https://marketplacelearn.walmart.com/guides/Order%20management/Returns%20&%20refunds/Create-item-level-return-rules#overview)

- [Create return rules](https://marketplacelearn.walmart.com/guides/Order%20management/Returns%20&%20refunds/Create-item-level-return-rules#create)

- [Frequently asked questions](https://marketplacelearn.walmart.com/guides/Order%20management/Returns%20&%20refunds/Create-item-level-return-rules#frequently-asked-questions)


* * *

Overview

By uploading a Return rules update template in Seller Center, you can customize your return settings at the item level and prioritize them over category-level return rules. In this guide, you’ll learn how to set up your return rules. If you need to set up return exemptions for hazmat, freight or custom items, you’ll need to [request approval](https://marketplacelearn.walmart.com/guides/Order%20management/Returns%20&%20refunds/Request-a-return-policy-exemption) from Walmart first.

## Create return rules

### Step 1 – Get started

Navigate to your [**Catalog**](https://seller.walmart.com/catalog/list-items) in Seller Center. Select the _Update items_ button and choose _Update with file_ from the dropdown menu.

### Step 2 – Download and complete the template

Select the _Change return rules_ link and save the template to your computer. Then open the template and make your updates.

![2511_Create-item-level-return-rules_gif-1.gif](https://azure-na-images.contentstack.com/v3/assets/blta7903c6b840b702d/blt79958a9b3d5536ad/6930acedf6621c49d738731b/2511_Create-item-level-return-rules_gif-1.gif?branch=us_mplearn)

### Step 3 – Upload the template

Navigate back to your [**Catalog**](https://seller.walmart.com/catalog/list-items) and select the _Update items_ button and choose _Update with file_. Then use the file browser window to upload the template. When you’re ready, select _Submit_.

![2511_Create-item-level-return-rules_gif-2.gif](https://azure-na-images.contentstack.com/v3/assets/blta7903c6b840b702d/blt1ef1e236fe550cf5/6930acf620c57c3eebd6b76f/2511_Create-item-level-return-rules_gif-2.gif?branch=us_mplearn)

Notes:

You’re responsible for ensuring your return rules are accurate. You’ll be responsible for all costs generated as a result of any return setting applied in error.

## Frequently asked questions

* * *

What is the return timeline?

You must support a minimum return window of 30 days, which begins seven days after an item’s ship date, with some exceptions. For more information, check out the [_Returns Policy._](https://marketplacelearn.walmart.com/guides/Order%20management/Returns%20&%20refunds/returns-policy)

* * *

How do I download a report of my return rules?

Navigate to the [**Reports**](https://seller.walmart.com/analytics/reports) dashboard in Seller Center, locate the **On Request Reports** section and select the _Report Type_ dropdown menu. Choose _Return Overrides Report_ and select the _Request report_ button. To download your report, navigate to the **Report activity** section, locate your report and select the download icon at the end of the row.
