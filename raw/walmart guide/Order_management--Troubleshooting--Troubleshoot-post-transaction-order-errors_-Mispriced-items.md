Troubleshoot post-transaction order errors: Mispriced items

Last updated on Dec 8, 2024

Reading time: 2 min

* * *

In this guide:

- [Overview](https://marketplacelearn.walmart.com/guides/Order%20management/Troubleshooting/Troubleshoot-post-transaction-order-errors:-Mispriced-items#overview)
- [Troubleshoot mispriced items](https://marketplacelearn.walmart.com/guides/Order%20management/Troubleshooting/Troubleshoot-post-transaction-order-errors:-Mispriced-items#troubleshoot-mispriced-items)

Overview

Sometimes, post-transaction order errors occur. For example, on rare occasions, mispriced items are sent to a customer. When this happens, you need to know how to troubleshoot the problem so you can correct the error. This guide will show you how.

## Troubleshoot mispriced items

### 1\. The order has not shipped

**What should I do?**

If you see a discrepancy in an item price before you ship it to the customer and you’ve verified the list price details are correct, cancel the order and select the _Help_ button in the [Seller Center](https://seller.walmart.com/home) menu bar to contact Support.

### 2\. The order has shipped but not delivered.

**What should I do?**

Initiate a Stop Shipment request with the carrier. Once you request a Stop Shipment for the order, the carrier will track the package. You’ll receive the item as if it were returned and the customer will be refunded. While the order issue is being resolved, correct the price in Seller Center if you have not done so already.

### 3\. The order has been delivered

**What should I do?**

If the order has already been delivered, the customer can initiate a return via [Walmart.com](https://walmart.com/) or Walmart Customer Care.

### 4\. The price is correct

**What should I do?**

If you’ve verified that your item is correctly priced **,** select the _Help_ button in the [Seller Center](https://seller.walmart.com/home) menu bar to contact Support _._ Provide the following information for a quick resolution:

- A complete description of the issue and overview of steps you've taken to investigate and resolve the issue.

- Order number

- Item ID

- Customer information

- Order status (e.g., Acknowledged, Shipped, Canceled)


Notes:

Sellers that participate in price matching programs in Seller Center may see differing prices because the system has automatically priced items competitively. For more information, visit [_Repricer: Overview_](https://marketplacelearn.walmart.com/guides/Catalog%20management/Price%20management/Repricer:-overview).

## Frequently asked questions

* * *

What if I’m unable to complete the order?

Consider offering an alternative to the customer, as outlined in [_Issue a standard refund in Seller Center_](https://marketplacelearn.walmart.com/guides/Order%20management/Returns%20&%20refunds/Issue-a-standard-refund-in-Seller-Center). If the customer requests a return or refund, you’re expected to complete the request.
