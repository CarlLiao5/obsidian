Overview

Returns are easy through Walmart’s omnichannel [Free & Easy Returns](https://www.walmart.com/cp/returns/1231920) program. For Walmart Fulfillment Services (WFS), we’ll handle the returns process for you. In this guide, learn what items are eligible for returns and what the returns process is like.

## How does it work?

Customers can return most items by printing shipping labels from their Walmart.com accounts or by bringing shipment-ready packages to a Walmart store. However, there are exceptions to this standardized return process. In some cases, the customer will contact the Walmart Customer Care team directly. Items designated as not returnable are outlined in the [Walmart Return Policy](https://corporate.walmart.com/policies#return-policy).

With the exception of the extended holiday returns window, customers have 90 days to initiate returns for most items, besides those that are listed in the Walmart Return Policy provided to customers. For example, items such as most wireless phones and major appliances have a shorter return window.

## Returns in stores

Walmart will accept eligible WFS returns from customers in any of our nearly 5000 Walmart stores across the US. If a customer chooses to return an item in store, the customer will put the item in a shipment-ready package and bring it to a Walmart store.

A Walmart associate will receive the item and check return eligibility. If the item is eligible for return, they’ll route the item to our WFS return centers for processing.

In this case, you won't see detailed tracking until the return is processed at the WFS return center. The status will be _In Transit_ and won't update until the item is processed. When you download the [Customer Returns Report](https://seller.walmart.com/wfsLite/reports), look for the _Received Scan Date_ for confirmation that it was received at the return center. Some items (e.g., luxury goods, hazmat items, perishables) may not be returnable to a Walmart store per our policies, but they may be returnable directly to our WFS return centers.

## Return processing fee

No shipping fees will be charged to the customer. Walmart will cover the cost of return shipping for any returns identified as _Walmart at fault_ (such as damaged in shipment, arrived late, lost in transit, and wrong item received).

Sellers are responsible for paying the return processing for any returns that are not identified as _Walmart at fault_. Go to [_WFS fees_](https://marketplacelearn.walmart.com/guides/Shipping%20&%20fulfillment/Walmart%20Fulfillment%20Services%20(WFS)/WFS-fees#Return_processing_fees_exceptions) for return processing fees.

## Setting return rules in Seller Center

You can tell us how to handle returns that arrive at a WFS return center. Your options are restock, return to me and dispose. Review [_WFS settings: Customize return rules_](https://marketplacelearn.walmart.com/guides/Getting%20started/Walmart%20Fulfillment%20Services%20(WFS)/WFS-settings-return-rules) to learn how to set these preferences. You can also update your seller-fulfilled returns on this page.

## Return centers

WFS items returned by customers will be sent to a WFS returns center to be processed.

- If the item is in a sellable condition, we’ll put it back into inventory. No restocking fees will be applied.
- If the item is unsellable, we’ll send the item back to you or dispose of it based on your [return rules](https://marketplacelearn.walmart.com/guides/Getting%20started/Walmart%20Fulfillment%20Services%20(WFS)/WFS-settings-return-rules). For disposal and removal fees, go to [_WFS fees_](https://marketplacelearn.walmart.com/guides/Shipping%20&%20fulfillment/Walmart%20Fulfillment%20Services%20(WFS)/WFS-fees#disposal_removal). Walmart will keep any inventory for returns identified as _Walmart at fault_.

## Refund timing

For returns initiated in store, eligible customers will get a refund at the store. For returns initiated via Walmart.com or Walmart Customer Care, we issue refunds based on milestones between return initiation and item processing at the WFS returns center. Learn more about our policy at [_WFS customer returns, reimbursement and dispute processes_](https://marketplacelearn.walmart.com/guides/Walmart%20Fulfillment%20Services%20(WFS)/Troubleshooting/wfs-customer-returns-reimbursement-and-dispute-processes) _._

## WFS Keep It Rule policy

If you added a Keep It Rule for seller-fulfilled returns, these preferences will also be applied to Walmart-fulfilled returns. Your preferences may not be honored under some circumstances, including:

- When a customer brings the item to a Walmart store without initiating the return online

- When Walmart suspects potential customer fraud, we may override your Keep it Rules and fully process the return to ensure you're protected from customer fraud.


For more information, go to [_Add Keep It Rules_](https://marketplacelearn.walmart.com/guides/Order%20management/Returns%20&%20refunds/Add-a-Keep-It-Rule).

Notes:

The above does not apply to _Walmart at fault_ as you have retained the original sales price. Walmart will also not reimburse you for unsellable, damaged returns that were not deemed to be _Walmart at fault_. Additionally, you can’t dispute returns that have been initiated, but not returned, by the customer unless you were charged back for the return.

## Frequently asked questions

* * *

Am I responsible for a refund if an item was shipped by WFS, but the customer says that it hasn’t been received?

No, you won’t be responsible for the refund. Find more examples of returns identified as _Walmart at fault_ in [_WFS fees_](https://marketplacelearn.walmart.com/guides/Shipping%20&%20fulfillment/Walmart%20Fulfillment%20Services%20(WFS)/WFS-fees#Return_processing_fees_exceptions).

* * *

If a customer changes their mind and no longer wants an item (buyer’s remorse), will I be reimbursed?

If it’s the fault of the seller, the customer no longer wants the item or an item is not as described, you will be charged. If Walmart is at fault (e.g., an item is damaged in transit to customer), we'll reimburse you. For more on fulfillment and returns exceptions, visit [_WFS fees_](https://marketplacelearn.walmart.com/guides/Shipping%20&%20fulfillment/Walmart%20Fulfillment%20Services%20(WFS)/WFS-fees#Return_processing_fees_exceptions).
