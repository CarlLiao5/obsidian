Overview

Select items may not be approved for Self-Serve Search Engine Marketing (SEM) campaigns because of errors that prevent them from being advertised on Google. These errors can be categorized as either **Resolvable** or **Non-resolvable**. In this guide, you'll learn how to resolve the errors in Seller Center.

## Resolvable errors

| **Error** | **Solution** |
| --- | --- |
| **Additional image too big** | Update the image to a resolution of less than 64 megapixels, file size smaller than 16MB. |
| **Additional image too small** | Update the image to be at least 100x100 pixels for non-apparel products and 250x250 pixels for apparel products. |
| **Extraneous text in title** | Update the item title. |
| **Generic image** | Update the item image. |
| **Image too big** | Update the image to a resolution of less than 64 megapixels, file size smaller than 16MB. |
| **Image too small** | Update the image to be at least 100x100 pixels for non-apparel products and 250x250 pixels for apparel products. |
| **Invalid brand** | Update the brand. |
| **Invalid characters in title** | Update the item title. |
| **Invalid GTIN** | Update the GTIN by editing the Full Item Spec. |
| **Invalid image** | Update the item image. |
| **Missing or invalid description** | Update the item description. |
| **Missing or invalid price** | Update the item price. |
| **Missing or invalid title** | Update the item title. |
| **Overlay on image** | Update the item image. |
| **Shipping not offered** | Add a shipping option. |
| **Shipping price too high** | Update the shipping price. |
| **Single color image** | Update the item image. |

## Non-resolvable errors

Non-resolvable errors are violations against Google policy. If you receive a **Policy Violation** or **Item Not Available** error, that means Google has flagged the item and it cannot be advertised on Google.

## Frequently asked questions

* * *

When will I know when my item is approved?

Once you take action to resolve an error, check back after 48-72 hours to see if the updates have been approved.
