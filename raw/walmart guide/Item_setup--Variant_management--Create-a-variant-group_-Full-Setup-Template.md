Create a variant group: Full setup

Last updated on Aug 20, 2025

Reading time: 6 min

* * *

In this guide:

- [Overview](https://marketplacelearn.walmart.com/guides/Item%20setup/Variant%20management/Create-a-variant-group:-Full-Setup-Template#overview)
- [Full setup](https://marketplacelearn.walmart.com/guides/Item%20setup/Variant%20management/Create-a-variant-group:-Full-Setup-Template#full)
- [Frequently asked questions](https://marketplacelearn.walmart.com/guides/Item%20setup/Variant%20management/Create-a-variant-group:-Full-Setup-Template#frequently-asked-questions)

* * *

Overview

As a Marketplace seller, you can create variant groups in Seller Center or with [Walmart APIs](https://developer.walmart.com/doc/us/mp/us-mp-items/). In this guide, you’ll learn how to create a variant group with 50 or more items using the Full Setup Template in Seller Center.

Notes:

Too many variations can be confusing for customers. We recommend no more than 50 items for a variant group with one attribute (a T-shirt in two colors and three sizes), and no more than 500 items for a variant group with three or fewer attributes (a pair of pants in 20 colors and 25 sizes).

## Full setup

### Step 1 — Get started

Navigate to your [**Catalog**](https://seller.walmart.com/item/list) in Seller Center and select the _Add items_ button. In the **Create items** section, select the _Create items_ button and then select _Multiple items_. On the next page, select the _Build spreadsheet_ button in the **New items** section.

### Step 2 — Choose a fulfillment method and product type

Select your fulfillment method and select _Next_. Then, choose up to 10 product types that best fit the items you want to add _._ If you aren’t sure which classification to choose, review [_Content standards: Overview_](https://marketplacelearn.walmart.com/guides/Item%20setup/Item%20content,%20imagery,%20and%20media/Content-standards:-Overview).

When you’re ready, select the _Build spreadsheet_ button. If you chose both fulfillment method options, two templates will download.

### Step 3 — Add variant attributes

There are five columns you must fill out to successfully set up a variant group. Open the template and locate the variant columns listed in the table below. All items in a variant group should be included as separate items in the template. For example, a T-shirt in two colors and three different sizes would have six rows with each variation defined in a separate row.

| **Column** | **Context** |
| --- | --- |
| **Variant Group ID** | This is a unique alphanumeric identifier, up to 20 characters in length (e.g.,DryFit\_VG5). You’ll use this to track your variant group in the **Catalog** dashboard once it’s been created. |
| **Variant Attribute Names (+)** | Attribute names indicate which attributes vary within the group, such as size or color in the example of a clothing group. If your items vary for multiple attributes (e.g., different colors and sizes), you can add more columns. You can't create custom variant attribute names. You must select an option from the dropdown. |
| **Is Primary Variant** | The primary variant provides suggestions on which variant should be highlighted in search. Based on customer search, Walmart will optimize the variant to display to the customer which may not match the primary variant provided. If you select _Yes_ on more than one item in a single variant group, your submission may fail. Consider selecting the top-selling item or the item with the most inventory as the primary variant.<br>If your item is a merged variant group, the primary variant you select may not be the primary variant that appears on Walmart.com. |
| **Swatch Image URL** | You’ll need to include an image for each variation of the swatch, but not each variation within the group. For example, the same URL could be used for a green T-shirt in three different sizes. Swatch image URLs must follow our [image guidelines](https://marketplacelearn.walmart.com/guides/Item%20setup/Item%20content,%20imagery,%20and%20media/Product-detail-page:-Image-guidelines-&-requirements).<br>If you don't provide a swatch image URL, variants will appear as tiles with the color stated as text on Walmart.com. Walmart may choose a swatch image from your primary or secondary images to optimize variants for the customer experience. |
| **Swatch Variant Attribute** | The swatch image requirement varies by product type. Visual variants can be displayed as swatches on the item page. For example, if a T-shirt comes in three colors, the swatch variant attribute would be set to color. |

### Step 4 — Upload the template

When you’re ready, upload the template. Navigate to your [**Catalog**](https://seller.walmart.com/catalog/list-items), select the _Add items_ button and select the _Upload in bulk_ link located under **Add thousands of items using a spreadsheet**. Then, drag and drop your updated template file into the file browser window.

Updates can appear in as quickly as 15 minutes or can take up to four hours.

## Frequently asked questions

* * *

How do I add or remove items from an existing variant group?

Check out [_Manage a variant group_](https://marketplacelearn.walmart.com/guides/Item%20setup/Variant%20management/Manage-a-variant-group) to learn how to edit your groups.

* * *

What if I create a variant group that contains items sold by another seller?

Your groups will merge to appear on the same page. While you can group, modify and delete your own variant groups, any items in other groups will continue to appear on the item page. Consequently, you may not always be able to define which content will be displayed on a product page if there are multiple sellers providing content for the same item. Walmart will determine the best available content for items unless provided by a brand owner or authorized reseller.

* * *

The variant attribute name that I want to use is not on the Supported Variant Attribute Names list. What should I do?

You can’t create custom variant attributes. Choosing the correct product type is important because variant attribute names are populated based on the product type of the items in the group. Check to see if there’s another way you could set up your item using a supported variant name. For example, if you’re selling vitamins in one-, two- and three-bottle packs, you could set them up to vary by count instead of by bottle. If you’re unable to find an alternative like this, you shouldn't set up your items in a variant group.

* * *

How does Walmart determine the ranking of variant attributes and how they show up on the item page?

Every subcategory or item type has a ranked list of variant attributes that defines how the attributes show up on the item page. For example, clothing items will always list the item’s size first followed by the color.

* * *

How many different variants can I add to an item?

That depends on the variant. For a single variant item, there's a limit of 50-100 variants—for example, a towel that comes in 100 different colors. For a multi-variant item, there’s a limit of 1,000 variants—for example, pants with 25 different sizes and 20 color options.

* * *

How can I change the primary variant if it’s out of stock?

If the primary variant goes out of stock, the grouping remains valid, with the primary variant showing as out of stock on the item page. If you decide to discontinue the primary variant (e.g., never add inventory again), you have the option to set a different variant as the primary by updating the variant group list with a different primary item selected. If your item is in a merged variant group and the other seller's primary variant is out of stock, select the _Help_ button in the [Seller Center](https://seller.walmart.com/home) menu bar to contact Support.

* * *

How can I discontinue the primary variant in a variant group when it matches a variant group set up by Walmart?

The Walmart variant group takes precedence. Your items will be included under Walmart's group if all the items match the items in that group. Matching is done using the item's unique item ID (e.g., UPC, ISBN). If you decide to discontinue the primary variant (by never adding inventory again), you have the option to set a different variant as the primary by updating the variant group ID grouping with a different primary variant selected.
