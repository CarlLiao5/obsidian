Multichannel Solutions: Create customer returns

Reading time: 6 min

Overview

Whatever sales channel you use, we’ll provide a return shipping label for your customer to ship items back to a Walmart facility, then follow your [return rules](https://marketplacelearn.walmart.com/guides/Getting%20started/Walmart%20Fulfillment%20Services%20(WFS)/WFS-settings-return-rules) to restock it, ship it back to you or dispose of it.

![](https://azure-na-images.contentstack.com/v3/assets/blta7903c6b840b702d/blt8c024fb00de88137/66c3877b43115a0307af3549/Multichannel-Returns_Infographic.jpg?branch=us_mplearn)

## Return eligibility

For multichannel sales, **items can only be returned through mail**. In-store returns will not be accepted. Customers can return most items by printing a shipping label and dropping off the package at their local FedEx.

In general, follow the return window on the original ecommerce site to check if the item is eligible for return. Walmart does not manage replacements or exchanges.

**Walmart** **will not accept returns for certain items**, such as hygienic medical equipment and prescription medications. View all restrictions in [Marketplace Non-Returnable Product Types.](https://azure-na-assets.contentstack.com/v3/assets/blta7903c6b840b702d/blt00619b7571612d8d/661ffb8ae8bd2d000de7b52d/Non-Returnable-Product-Types.pdf)

## Fees & refunds

We charge a [return processing fee](https://marketplacelearn.walmart.com/guides/Shipping%20&%20fulfillment/Walmart%20Fulfillment%20Services%20(WFS)/WFS-fees) based on shipping weight. This fee covers the shipping cost and handling at the fulfillment center. We’ll charge the [billing method](https://seller.walmart.com/finance-and-tax/billingServices) on file, and you can view fees in the [**Payments**](https://seller.walmart.com/partner-analytics/payments/statements/period) page of Seller Center.

Since payment happened outside of Walmart, you’re responsible for any customer refunds or store credit. Manage this from the ecommerce site where the customer originally placed the order.

## Set return rules

Customize your [return rules](https://seller.walmart.com/seller-profile/return-policy?) to choose how we process sellable and unsellable items. You can have items restocked, shipped back to you or disposed of. ( [Disposal or removal fees](https://marketplacelearn.walmart.com/guides/Shipping%20&%20fulfillment/Walmart%20Fulfillment%20Services%20(WFS)/WFS-fees) may apply.) Please note that in-store returns are not permitted. You should also have your own return address on file with us. We'll only use a Walmart return center address for orders that'll be restocked or disposed of.

If you don’t update the settings, we’ll **default to restocking sellable inventory and disposing unsellable inventory**. For more details, view [_WFS settings: Return rules_](https://marketplacelearn.walmart.com/guides/Getting%20started/Walmart%20Fulfillment%20Services%20(WFS)/WFS-settings-return-rules).

## Create a customer return

Notes:

You can only create a return if items have already been delivered.

When a customer contacts you to return items, follow these steps:

1. Log into Seller Center and go to the [**Multichannel orders**](https://seller.walmart.com/wfs/multi-channel-solutions/orders) page.
2. Search for the delivered customer order.
3. Select the 3 dots and select **Create return**.
4. Choose the quantity and return reason.
5. Download the return shipping labels and forward them to the customer. (Some returns may be split into multiple shipments.)
6. Ask the customer to print the labels and drop off packages at FedEx.

Once you’ve created a return, you cannot cancel it. However, you’ll only be charged the return processing fee once the carrier scans the shipping label.

![](https://azure-na-images.contentstack.com/v3/assets/blta7903c6b840b702d/blt74266516cdee7c3a/68911a292e90500243cfb055/Create_return.png?branch=us_mplearn)

Notes:

You can also create a return by selecting Create return from the dropdown under Manage fulfillment on the [Multichannel orders](https://seller.walmart.com/wfs/multi-channel-solutions/orders) page, and following the steps provided.

Pro tip

To learn more about how to automatically manage returns through API, view [Multichannel Solutions: API integration](https://marketplacelearn.walmart.com/guides/Walmart%20Fulfillment%20Services%20(WFS)/Walmart%20Multichannel%20Solutions/multichannel-api).

## Track a return

Once items arrive at a returns center, associates will check them. If they're in a sellable condition, we’ll put the item back in inventory. If not, we’ll send the items back to you or dispose of them, based on your [return rules](https://marketplacelearn.walmart.com/guides/Getting%20started/Walmart%20Fulfillment%20Services%20(WFS)/WFS-settings-return-rules).

It can take up to **2 weeks** **to process a return**. You can track the status by downloading the [Customer Returns Report,](https://seller.walmart.com/wfsLite/reports) and utilizing the dedicated MCS Returns tab for your multichannel returns. This report includes the order type, date the item was delivered to the returns center, when it was scanned for receiving and the item condition upon return. To find the tracking number, check the return shipping label.

Notes:

If you’ve set up the [webhook](https://marketplacelearn.walmart.com/guides/Walmart%20Fulfillment%20Services%20(WFS)/Walmart%20Multichannel%20Solutions/multichannel-api), you can get real-time notifications on the return status.

## Frequently asked questions

* * *

How will fraudulent returns be handled?

We’ll notify you if we suspect fraud during the returns process. Please contact the customer directly to resolve. We’ll dispose the item, regardless of your return rules.

* * *

Where can I find the return processing fee?

View fees in the [**Payments**](https://seller.walmart.com/partner-analytics/payments/statements/period) page of Seller Center, or download the [Settlement Report](https://seller.walmart.com/wfsLite/reports).

* * *

Who’s responsible for customer refunds?

Since payment happened outside of Walmart, you’re responsible for any customer refunds or store credit. Manage this from the eCommerce site where the order was originally placed.

* * *

What items can’t be returned?

Walmart retains final authority to accept or decline any item that is eligible for return. Under no circumstance will Walmart accept the return of an ineligible item. Walmart reserves the right to refuse any return that poses risk to our customers, associates or facilities. Review a full list of non-return able product types [here](https://azure-na-assets.contentstack.com/v3/assets/blta7903c6b840b702d/blt00619b7571612d8d/661ffb8ae8bd2d000de7b52d/Non-Returnable-Product-Types.pdf).

* * *

What happens if something goes wrong with the shipment?

Whatever the issue, you must first start a Support case. Select the _Help_ button in the [Seller Center](https://seller.walmart.com/home) menu bar to contact Support. In the description, add these details:

- Customer order number
- Proof of the original order from the eCommerce site, including SKU, quantity, customer name, address and sale price
- Photo of the incorrect/damaged item if applicable, including the GTIN or SKU label
- Description of the issue (damaged, delayed, missing)
- Please note: Seller does not pay a fee for damaged, delayed, or missing shipments. Seller does pay a fee for customer-related issues. (does not fit, ordered the wrong thing etc.) Items that don’t match the original order will be disposed of upon return, regardless of your return rules.
