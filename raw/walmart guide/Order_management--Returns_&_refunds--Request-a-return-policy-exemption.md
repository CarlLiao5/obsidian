Request a return policy exemption

Last updated on Jun 30, 2025

Reading time: 1 min

* * *

Overview

If you sell HAZMAT, freight, custom items or items that can’t be returned in a single box, you may be eligible to request exemptions to the standard Marketplace returns process. This means the customer will contact you directly to initiate the return for these specific items. In this guide, we’ll show you how to request a return policy exemption.

For a full list of items that classify for return exemptions, see the [_Returns Policy_](https://marketplacelearn.walmart.com/guides/Order%20management/Returns%20&%20refunds/returns-policy) _._

Notes:

Return policy exemptions are discretionary. Walmart reserves the right to modify or cancel any return exemptions granted in its sole discretion.

## Request a return policy exemption

### Step 1 — Fill out the request template

Download a [Return Exemption Request Template](https://gecrm.my.salesforce.com/sfc/p/#61000000ZKTc/a/4M0000000Nf2/TxFIf0pW_vZHa761h06yJ2XfF4bN1l6d0YRJvGujXZA) excel file and fill out the cells for the SKUs you want to make exempt from the standard returns process.

### Step 2 — Contact support

To submit your request, select the _Help_ button in the [Seller Center](https://seller.walmart.com/home) menu bar to contact Support. Follow the instructions to upload your exemption request. Once you submit the completed template, Walmart reviews the request to verify that it adheres to our exemption rules. Walmart reserves the right to reject exemptions as unreasonable.

## Frequently asked questions

* * *

How do I know if a specific item qualifies for a return policy exemption?

You can Review the Return Exemption section of the [_Returns policy_](https://marketplacelearn.walmart.com/guides/Order%20management/Returns%20&%20refunds/returns-policy) for specifics on which items are eligible for exemption.

* * *

What should I do after my request is approved?

You will need to customize your return settings and create return rules for the items with return exemptions. For more information, see [_Create item-level return rules_](https://marketplacelearn.walmart.com/guides/Order%20management/Returns%20&%20refunds/Create-item-level-return-rules).
