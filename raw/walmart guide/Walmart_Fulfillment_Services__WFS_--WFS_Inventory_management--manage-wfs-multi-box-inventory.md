Overview

[_Multi-box_](https://marketplacelearn.walmart.com/guides/Walmart%20Fulfillment%20Services%20(WFS)/WFS%20programs%20&%20services/wfs-multi-box-overview) means a single item that’s shipped in multiple boxes. In this guide, we'll show you how to track sales, inventory and returns for these items.

## Customer orders and returns

During checkout, the customer will see that their item arrives in multiple boxes and get a shipping estimate.

Customers must return the entire sellable set. We do not accept partial returns. At this time, we also cannot send replacements for specific parts. If any box in a sellable set is damaged or lost, we’ll ask the customer to send back the entire sellable set for a refund. They can then buy a new set from Walmart.com.

## Track sales and inventory

The [**Inventory**](https://seller.walmart.com/wfsLite/manage-inventory) page shows the daily sales and stock for all Walmart-fulfilled items. For multi-box items, you’ll see sellable sets on the page. Here's how to drill down:

1. To view just multi-box items, filter by **Item type**.

2. Select the item name to open basic information about each box GTIN.


![](https://azure-na-images.contentstack.com/v3/assets/blta7903c6b840b702d/bltb54a5c5b34cbf69c/6719749364d8aca2b0806acf/Multi-box_filter.gif?branch=us_mplearn)

You can also track inventory movement for each box GTIN. View fulfillment records from the inventory log:

1. Find the sellable set, then select the 3 dots and choose **View inventory log**.

2. Choose a box GTIN to start with.

3. View events for that box GTIN.

4. To filter by a different box, select the box GTIN next to the search bar. This will show you the other boxes that are in the sellable set.


![](https://azure-na-images.contentstack.com/v3/assets/blta7903c6b840b702d/bltb33e5bb33a8b5ee6/671974267acb6f2c4bb20931/Multi-box_inventory_log.gif?branch=us_mplearn)

## Data tracking

On the [**Inventory**](https://seller.walmart.com/wfsLite/manage-inventory) page, you’ll see data at different levels for multi-box items. In general, you can track sales data by the sellable set. For data related to physical inventory, use box GTINs. Here’s a snapshot of how data is broken up:

| **Sellable set** | **Boxes** |
| --- | --- |
| - Daily sales<br>- Daily units sold<br>- Max available to sell<br>- Days of supply<br>- Out-of-stock date | - Available to sell<br>- Reservations<br>- Inbound units<br>- Suggested units to send |

Some columns in the **Inventory** page will have additional data for multi-box items. For example, we show how many units of each box GTIN you have available to sell, but also a _max available to sell_ that counts how many units of complete, sellable sets can be sold.

![](https://azure-na-images.contentstack.com/v3/assets/blta7903c6b840b702d/blt74a9033880d6b191/671973e5f94705369fb710af/Multi-box_reservations_flyout.gif?branch=us_mplearn)

## Reports

For in-depth sales and inventory tracking, download these [reports](https://seller.walmart.com/wfsLite/reports):

- View just multi-box information in **Multi-Box Items**.
- Track sales for all WFS items, including multi-box, through **Orders**. Each box has its own line in the report. Use the Bundle Item ID column to check which sellable set it belongs to.
- Find in-stock rate and restock recommendations in **Inventory Health**.
- View net inventory movement in **Inventory Reconciliation**.
