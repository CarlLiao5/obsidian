## Overview

There are two ways to get access to your organization in Walmart Connect Ad Center. Your super admin, org admin or account admin can provide it or you can request it yourself. If you are an admin looking to manage access, visit [Account access as an admin user](https://advertisinghelp.walmart.com/s/guides?channel=Sponsored%20Products&article=000012444&language=en_US).

## Request and manage access a new user

Request management allows you to request new or updated access in Walmart Connect Ad Center. You can view request management by selecting the person icon () labeled Account at the top right corner and selecting Manage profile and account.

![](https://advertisinghelp.walmart.com/servlet/rtaImage?eid=ka0KW000000oPo7&feoid=00N6100000Hxa5T&refid=0EMKW000008vZHI)

### Requesting access to a new account

To request access to a new account:

- Navigate to the Manage your account access section on the **Your home** page and search for either Walmart supplier, Marketplace seller or the organization that the account belongs to in the dropdown.
- Use the search bar to search by name.

![](https://advertisinghelp.walmart.com/servlet/rtaImage?eid=ka0KW000000oPo7&feoid=00N6100000Hxa5T&refid=0EMKW00000AGSJY)

- Select the account from the dropdown under Accounts and select **Save**.

![](https://advertisinghelp.walmart.com/servlet/rtaImage?eid=ka0KW000000oPo7&feoid=00N6100000Hxa5T&refid=0EMKW00000AGSJd)

- Select **Request access** under Actions. This will redirect you to the Request access page.

![](https://advertisinghelp.walmart.com/servlet/rtaImage?eid=ka0KW000000oPo7&feoid=00N6100000Hxa5T&refid=0EMKW00000AGS5v)

**Tip:** Make sure you are searching within the right organization. You can toggle between organizations (if you have access to multiple) using the dropdown.

- Use the text box to type in the reason for your request.

![](https://advertisinghelp.walmart.com/servlet/rtaImage?eid=ka0KW000000oPo7&feoid=00N6100000Hxa5T&refid=0EMKW00000AGS6A)

- Select the level of access: *Standard* or *Admin*. If you select *Standard*, you will need to set access levels for each component.
- Select **Request access**. Your request will automatically be sent to the account admin, org admin or super admin for review. You’ll receive an email notification when the request is approved, modified or denied.

![](https://advertisinghelp.walmart.com/servlet/rtaImage?eid=ka0KW000000oPo7&feoid=00N6100000Hxa5T&refid=0EMKW00000AGS6e)

### Requesting upgraded access to your account

To upgrade your access:

- Navigate to the Manage users section on the Access page.
- Search for your name and select the **pencil** icon under Actions.

![](https://advertisinghelp.walmart.com/servlet/rtaImage?eid=ka0KW000000oPo7&feoid=00N6100000Hxa5T&refid=0EMKW00000AGS9J)

- Select the updated access by changing the role from **Account user** to **Organization user** under the Role dropdown (if applicable).

![](https://advertisinghelp.walmart.com/servlet/rtaImage?eid=ka0KW000000oPo7&feoid=00N6100000Hxa5T&refid=0EMKW00000AGS9Y)

- Select the buttons next to **Edit**, **View** or **None** for each component depending on the access you are looking for.
- Select **Update access**. Your request will automatically be sent to the org admin or super admin for review. You’ll receive an email notification when the request is approved, modified or denied.

![](https://advertisinghelp.walmart.com/servlet/rtaImage?eid=ka0KW000000oPo7&feoid=00N6100000Hxa5T&refid=0EMKW00000AGS9i)

- - You can track your request under Requests by filtering to *Requests I made*.

**Admin-only note:** Admins must assign a new admin before removing their own access from an account. You can follow the same steps to downgrade or remove your access.

### Checking the status of your access request

You can view the status of your request by selecting **Existing Organizations** under Requests. Select **Requests I made** to view the status of your request.

![](https://advertisinghelp.walmart.com/servlet/rtaImage?eid=ka0KW000000oPo7&feoid=00N6100000Hxa5T&refid=0EMKW00000AGSAv)

You can also view or cancel the request by selecting the **three dots** in the Actions column.

![](https://advertisinghelp.walmart.com/servlet/rtaImage?eid=ka0KW000000oPo7&feoid=00N6100000Hxa5T&refid=0EMKW00000AGSBA)

### Following up on pending access requests

If your request hasn’t been approved or denied in about a week:

- Create a support ticket by selecting **Existing Organizations** under Requests.
- Select **Requests I made** and then select the **three dots** under the Actions column to select **Create support ticket**.

### Approving or declining access requests (admin-only)

Account admins, org admins or super admins can review, approve or deny requests from the Manage requests dashboard. For more information, visit [Account access as an admin user](https://advertisinghelp.walmart.com/s/guides?channel=Sponsored%20Products&article=000012444&language=en_US).
