## How to open a dispute

You can open a dispute on Walmart’s accounts payable portal, [High Radius](https://walmart.highradius.com/). See the [High Radius EIPP guide](https://gecrm.my.salesforce.com/sfc/p/61000000ZKTc/a/KW00000035Wu/fipXq0.DxaGUTQDmAng.WljVOu1YlcUWPixRkXt5PZM) for more information about this platform.  
  
To open a dispute, navigate to **EIPP** and select either the **OPEN BILLS** (unpaid claims) or **CLOSED BILLS** (paid claims) tab. This will show the current month's invoices. If you want to review previous invoices, please use the advance search functionality and mention the 10-digit customer number in the *Customer N* *umber* column.

- Select the checkbox for the invoice(s) to dispute
- Select **Dispute** (this will bring up the **Create Dispute** window)

**Important: If the invoice you are looking for isn’t showing up, use the advance search functionality to search for all invoices under the customer number.**

![](https://advertisinghelp.walmart.com/servlet/rtaImage?eid=ka0KW000000oOje&feoid=00N6100000Hxa5T&refid=0EMKW000009Filo)
- Select the **dropdown menu** just below the *Dispute Reason* column in the blue field; then select a **Dispute Reason** from the dropdown menu.
	- **Example:** For backup requests, select **Additional support documentation needed.**
- Enter the amount to be disputed by clicking inside the blue field below the column header.

**Note: If the amount is in the thousands, enter without a comma.**

- In the comments field, be detailed and give facts supporting your reason for the dispute. If you have an email supporting your dispute, we recommend putting the email in a word document and then uploading it as an attachment.
- Attach supporting documentation by selecting **Upload** and then browsing for the upload file. Select **OK**.
- Select **Save**.

For more information on managing disputes, please review the *Dispute Management* section in the [High Radius EIPP guide](https://gecrm.my.salesforce.com/sfc/p/61000000ZKTc/a/KW00000035Wu/fipXq0.DxaGUTQDmAng.WljVOu1YlcUWPixRkXt5PZM).
