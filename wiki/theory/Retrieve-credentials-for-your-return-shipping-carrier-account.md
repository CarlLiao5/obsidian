# Retrieve credentials for your return shipping carrier account

> 获取退货承运人账户的 API 凭证，以便在 Seller Center 中创建退货运输标签

## 概述

在 Seller Center 中自定义退货设置时，您可以使用 Walmart 的默认承运人服务或您自己的承运人账户来生成客户发起的退货运输标签。本指南将介绍如何获取创建运输标签所需的 API 凭证以及如何上传这些凭证。

## 工作原理

如果您想使用现有的承运人账户，需要在 Seller Center 中输入完整的运输承运人 API 凭证，因为这些凭证包含生成标签所需的关键信息。

您可以通过以下方式获取 API 凭证：

- **FedEx API 凭证**：登录您的 FedEx 账户并访问 [FedEx 开发者资源中心](https://www.fedex.com/en-us/developer.html)
- **UPS API 凭证**：登录您的 UPS 账户并访问 [UPS 开发者工具包](https://www.ups.com/upsdeveloperkit)

获取 API 凭证后，导航到 Seller Center 然后选择 **Account settings**。在 **Shipping Profile** 下选择 **Returns**。滚动到页面中部的 **Label Printing (Required)** 部分，选择 **Add Carrier Account** 按钮。按照提示添加您的凭证。
