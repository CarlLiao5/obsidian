# Update seller-fulfilled return label settings

> 添加 FedEx 或 UPS 账户用于退货配送

## 概述

除非您另有指定，否则 Walmart 的退货配送服务（RSS）将用作自配送退货运输标签的默认选项。如果您选择使用自己的 FedEx 或 UPS 账户进行退货配送，需要提供您的 API 账户凭证。本指南将展示如何添加用于退货配送的新 FedEx 或 UPS 账户。

## 退货标签设置

### 步骤 1 — 开始

导航到 **Shipping Profile** 中的 [**Returns**](https://seller.walmart.com/settings/shipping-profile/returns) 页面。

**注意**：如果对退货运输承运人账户进行了任何更改（包括更改密码），需要在 Seller Center 中移除已连接的账户并重新输入您的信息。如果在密码更改后未在 Seller Center 中重新输入账户详情，客户在尝试打印退货标签时会收到错误。

### 步骤 2 — 添加账户信息

在 **Returns** 页面上，导航到 **Label Printing (Required)** 部分并选择 **Add Carrier Account**。要添加 FedEx 或 UPS 账户信息，从 _Carrier Name_ 下拉菜单中选择 **FEDEX** 或 **UPS**。要了解如何添加 API 账户凭证，选择 **How to obtain or manage API credentials**。

## 常见问题

**承运人凭证验证需要多长时间？**

FedEx 账户可能需要最多 72 小时才能验证并准备好处理退货。如果 72 小时后仍未添加账户，请尝试重新提交凭证。

UPS 账户验证是即时的。提交凭证后，等待几分钟并刷新页面。如果仍未添加 UPS 账户，您的凭证可能无效。请尝试重新提交凭证。

**我没有承运人 API 凭证，我该怎么办？**

如果您没有承运人 API 凭证，请参阅[_检索退货运输承运人账户的凭证_](https://marketplacelearn.walmart.com/guides/Shipping%20&%20fulfillment/Shipping%20methods/Retrieve-credentials-for-your-return-shipping-carrier-account)了解更多信息。
