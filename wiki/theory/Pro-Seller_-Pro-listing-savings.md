# Pro Seller: Pro Listing Savings

> 符合条件的 Pro Seller 卖家可获得高达 10% Referral Fee 折扣

## 概述

Pro Listing Savings 计划旨在帮助正在获得或已获得 [Pro Seller](pro-seller-overview) 身份的卖家。该计划旨在帮助卖家成长、改善产品可见性，并在符合条件的 Pro Listing 报价上获得高达 10% 的 Referral Fee 折扣。

## 工作原理

查看符合条件的 Pro Listing 报价，导航到 Seller Center 中的 [**Pro Seller**](https://seller.walmart.com/growth/seller-tiering) 仪表板，选择 **Pro Listings** 标签。过去 90 天的 Pro Listing Savings 显示在列表顶部。

商品要符合 Pro Listing Savings 条件，必须：
1. 拥有 **Advanced Seller** 或 **Pro Seller** 状态
2. 您的报价必须有**有竞争力的价格**达到或击败市场
3. 您的报价必须提供**三天或更短的快速免费配送**

满足这些要求后，这些 Pro Listings 上的未来订单将为您带来 Advanced Seller 5% 或 Pro Seller 10% 的 Referral Fee 折扣。

## 增加符合条件机会的最佳实践

| 最佳实践 | 详情 |
| --- | --- |
| **有竞争力的价格** | 启用 [Repricer](Repricer_-overview) 并为商品分配竞争性价格或基于规则设置的外部价格策略 |
| **外部价格匹配** | 提交 [外部价格匹配](Submit-an-external-price-match) 以帮助客户评估价格竞争力并定义某些 Marketplace 定价计划中的商品资格 |
| **快速免费配送** | 注册 [TwoDay 配送](Shipping-methods_-expedited-delivery-programs)、[Simplified Shipping Settings](simplified-shipping-settings-overview)，或使用 [Walmart Fulfillment Services (WFS)](WFS_basics) 履行订单 |

**注意：** 必须拥有类似商品的 Amazon Standard Identification Number (ASIN) 才能提交外部价格匹配。

## 常见问题

**如何查看可以价格调整的报价？**
导航到 [**Pro Seller**](https://seller.walmart.com/growth/seller-tiering/overview) 仪表板，选择 **Pro Listings** 标签。在 _Listings missing competitive price_ 下，选择 _View items_。

**Pro Listings 上的 Referral Fee 折扣持续多长时间？**
符合条件的 Pro Listing 报价将在符合条件的订单上获得最多 90 天的 Referral Fee 折扣。

**为什么我收不到 Pro Listing 订单的任何节省？**
可能的原因：
- 订单日期时您的活跃等级状态不是 Advanced Seller 或 Pro Seller
- 您的报价已收到 90 天的 Pro Listing Savings，不会收到额外节省
- 订单时您的报价价格没有竞争力
- 订单的 **Ordered** 日期和 **Deliver by** 日期之间超过两天
