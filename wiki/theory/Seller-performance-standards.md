# 卖家绩效标准 (Seller Performance Standards)

> Walmart Marketplace 对卖家的最低绩效要求，包括配送、取消率、退货率、负面反馈等关键指标。

## 概述

Walmart 会持续监控所有卖家的绩效表现。低于最低标准的卖家将收到预警通知，多次或持续未达标可能导致**账户暂停或终止**。

## 核心绩效指标

| 指标 | 说明 | 最低要求 |
|------|------|----------|
| **有效追踪率** | 订单在 SLA 内提供有效追踪信息的比例 | ≥97% |
| **取消率** | 未经批准主动取消的订单比例 | ≤2% |
| **退货率** | 客户发起退货的订单比例 | 监控中 |
| **负面反馈率** | 收到的负面评价占总评价的比例 | 监控中 |
| **客户响应时间** | 平均回复客户消息的时间 | 越快越好 |
| **按时交货率** | 订单在承诺时间内送达的比例 | 监控中 |

## 绩效预警与行动

| 级别 | 说明 | 可能后果 |
|------|------|----------|
| **Alarm（预警）** | 指标接近阈值 | 收到通知，需改善 |
| **Critical（严重）** | 指标低于要求 | 可能暂停新订单 |
| **Termination（终止）** | 持续不达标 | 账户暂停/终止 |

## 提升绩效的建议

1. **有效追踪率**：确保发货后及时上传有效追踪号；使用可靠承运商
2. **取消率**：保持充足库存；不在无法履约时接受订单
3. **退货率**：确保商品描述准确；提高包装质量；减少运输损坏
4. **负面反馈**：及时响应客户问题；主动解决潜在问题
