# 商品详情页政策（Product Detail Page Policy）

> Walmart 商品页面的内容标准规范，涵盖标题、描述、关键特性、图片等所有展示内容。

## 概述

卖家须为所售商品提供**准确、真实、不误导**的详细信息。违规可能导致商品被下架、账户暂停或终止。

## AI 生成内容要求

**重要更新（2026 年）：** AI 生成的内容须经过卖家核实后方可发布。禁止：
- 虚假或误导性内容
- 未经验证的安全声明
- 伪造的评论或背书

## 标题规范

| 允许 | 禁止 |
|------|------|
| 不超过 150 个字符的简洁描述性标题 | 全大写、特殊字符（~ ! * $ 等）|
| 正确大小写 | 任何促销语（免费配送、Walmart fulfilled、热销等）|
| 包含关键属性值（如 5 in、4 count、12 oz）| 使用其他零售商名称（除非获得许可）|
| 清晰的品牌/商品类型描述 | 无关信息（即将到货、缺货、停售等）|
|  | 任何格式的 URL |
|  | 非英语（除非是品牌名称的一部分）|
|  | 包含年份（除非类目标准要求）|

## 描述规范

| 允许 | 禁止 |
|------|------|
| 描述商品功能与优势（至少 150 词）| 声称竞争对手独家销售权 |
| 包含顾客可能搜索的关键词 | 使用其他零售商名称（除非获得许可）|
| 一个段落，自然流畅 | 任何促销语 |
|  | 真实性声明（须经 Walmart 批准，食品类除外）|
|  | 外部 URL |
|  | Emoji 或特殊字符 |
|  | 项目符号或编号列表 |
|  | 非英语（除非是品牌名称的一部分）|
|  | 大量重复关键词 |

## 关键特性规范

- 列出 3-10 个最重要的功能和优势
- 使用简短短语，避免重复关键词
- 每条特性不超过 80 字符（含空格）
- 禁止促销语、零售商名称、emoji、URL

## 图片要求

- 高质量白色背景主图（推荐 800x800 像素以上）
- 多角度展示
- 不得包含促销文字、水印
- 须与实际发货商品一致
