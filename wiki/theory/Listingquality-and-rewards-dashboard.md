# Listing Quality and Rewards Dashboard

> 了解您的评分如何计算

## 概述

您的 [Listing Quality](https://seller.walmart.com/growth/listing-quality) 分数提供有关商品绩效的洞察。优质列表推动更高的转化率并提高产品可见性，让您有机会获得 Pro Seller 徽章等增长机会。

## 评分构成

Listing Quality 是一个衡量商品质量的分数。您的 Listing Quality 分数由五个组成部分构成。这些组成部分涵盖 Walmart.com 上产品详情页面的属性。

| 组成部分 | 描述 |
| --- | --- |
| **Content quality** | 衡量您的列表的[商品名称、商品描述、关键特性和图片](Content-standards_-Overview)的有效性 |
| **Price competitiveness** | 将您的价格与外部市场的类似产品进行比较 |
| **Shipping** | 衡量跨邮政编码的承诺配送速度 |
| **Published & in stock** | 显示商品对客户的可用频率，基于过去七天 |
| **Ratings & reviews** | 根据客户评论数量和平均评分衡量商品 |

**注意：** 您将有一个目录级别的 Listing Quality 分数和每个单独列表的商品级别 Listing Quality 分数。目录级别分数是商品级别分数的平均值。

## 如何提高分数

| 功能 | 开始使用 |
| --- | --- |
| **Apply Listing Quality suggestions to individual listing components** | 访问 [Listing Quality: Fix issues](listingquality-fix-issues) |
| **Apply content updates to multiple items at once** | 访问 [Listing Quality: Bulk Attribute Editor](Bulk-attribute-editor) |

## 常见问题

**为什么我的某些 Listing Quality 组成部分显示 N/A？**
如果在目录或商品级别您的 Listing Quality 组成部分显示 _N/A_，则该组成部分被排除在整体分数计算之外。

**如何报告不正确的产品类型？**
如果您认为您的商品被分配到了不正确的产品类型，请访问 [Report incorrect product type](Report-incorrect-product-type)。

**为什么无法提交商品内容的更改？**
为确保高质量的购物体验，Walmart 为每个商品属性选择最佳内容，优先考虑品牌所有者及其授权转售商直接提供的内容。
