# Add a Keep It Rule

> 添加保留规则 (Keep It Rule)

## 概述
有时让 Walmart.com 客户保留商品并处理退款比支付退货运费更具成本效益。通过 Keep It Rules，你可以向客户提供全额或部分退款并让他们保留商品。本指南将介绍如何在 Seller Center 中设置 Keep It 和 Partial Keep It Rules。

## 核心要点

### 添加 Keep It Rule 步骤

**步骤 1 — 开始**
- 在 Seller Center 中导航到 **Returns** 页面
- 找到 **Keep It Rules** 或 **Partial Keep it rules** 部分

注意：你不能将 Partial Keep It Rules 应用于有退货豁免的商品。

**步骤 2 — 添加规则**
- **Keep It Rules**：选择 **Add Keep It Rule** 按钮，输入你的 **Department**、**Item price limit** 和 **Return Order limit** 信息。选择 **Save**
- **Partial Keep It Rules**：选择 **Add Partial Keep It Rule** 按钮，输入你的 **Department**、**Return Reason**、**% Refund Off** 和 **Max Amount** 信息。选择 **Add Partial Keep It Rule**

## 核心要点

### Keep It Rules 定义
每个 Keep It Rule 由以下定义：
- **Department**：商品类别
- **Item price limit**：客户能够保留商品的美元上限
- **Return Order limit**：客户能够从该类别保留多个商品的订单总金额上限

### Partial Keep It Rules 定义
每个 Partial Keep It Rule 由以下定义：
- **Department**：商品类别
- **Return Reason**：退货原因
- **% Refund Off**：特定类别和退货原因的退款百分比
- **Max Amount**：客户能够保留该商品的金额上限

## 常见问题

**如果客户不想保留商品怎么办？**
对于线上和店内退货，将询问客户是否保留商品。如果客户不想保留商品或 Walmart 确定特定客户交易不符合 Keep It 计划资格，Walmart 将自行决定按照其程序处理商品处置。

**我可以在退货请求时提供替代解决方案吗？**
不可以。除非通过 Walmart 的 Partial Keep It Rules 或其他预先批准的解决方案明确授权，否则禁止与客户谈判替代解决方案。
