# 访问 Walmart Connect Ad Center

> 访问 Walmart Connect 广告平台，包括新广告主入驻和现有广告主登录。

## 入驻沃尔玛广告

**新广告主入驻流程：**

| 角色 | 入驻链接 | 前置条件 |
|------|----------|----------|
| Walmart 供应商 | [供应商入驻表单](https://advertising.walmart.com/onboard/supplier) | 已有 Supplier One 账户并获得 Supplier ID |
| Marketplace 卖家 | [卖家入驻表单](https://seller.walmart.com/signup) | 已有 Seller Center 账户并获得 Seller ID |

**Marketplace 卖家自动入驻：**
所有新的 Marketplace 卖家自动入驻自助广告平台，Seller Center 账户管理员自动成为 Walmart Connect Ad Center 管理员。

**Walmart 供应商入驻：**
供应商 One 管理员自动创建组织（Organization），管理员负责审查、批准或拒绝新账户申请。

## 如何登录广告账户

1. 访问 [Walmart Connect Ad Center](https://advertising.walmart.com/signin)
2. 使用 Seller Center、Supplier Center 或 Walmart.com 现有凭证登录
3. 选择要投放广告的广告主账户
4. 选择媒体解决方案（Sponsored Search / Display / Shop Builder 等）
5. 无法访问的媒体解决方案将显示为灰色

**找不到账户？** 输入广告主账户名称或 ID 的前 3 个字符搜索。

## 查找 Vendor ID（入驻用）

| 账户类型 | 查找位置 |
|----------|----------|
| Marketplace 卖家 | Seller Center 顶部页面 / 齿轮图标 → Company Info |
| Walmart 供应商 | Supplier One → Account → Supplier profile → Supplier number |

## 广告主 ID（广告账户用）

| 平台 | 查找方式 |
|------|----------|
| Sponsored Search | Reports → Advertiser → URL 中的数字 ID（如 `...advertiser/xxxxxx`） |
| Display | URL 中的 `aid=xxxxxx` |
| Brand Shop / Shelf | URL 中的 `aid=xxxxx` |

## 账户整合（现有广告主）

如果已有多个广告账户，可以整合到单一账户管理：

1. 登录 Walmart Connect Ad Center
2. 导航到 **Reports → Advertiser** 获取 Account ID
3. 在目标账户中复制原账户的活动
4. 暂停原账户中的活动

**注意：** 无法合并 Marketplace 卖家和 Walmart 供应商账户。
