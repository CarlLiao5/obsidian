# Performance Dashboard: Return Rate

> 绩效仪表板：退货率

## 概述
退货率是追踪客户退回的已交付商品百分比的卖家绩效标准。

## 核心要点

### 政策说明

退货率是在过去 60 天内因卖家过错被客户退回的商品百分比。保持较低的退货率有助于确保积极的客户体验并保护你的绩效指标。

必须保持退货率 **≤ 6%**。

### 绩效驱动因素表格

| 驱动因素 | 说明 | 建议 |
| --- | --- | --- |
| **Defective** | 发货给客户的缺陷订单率高 | 审查商品确保所有零件按广告所述包含、质量如 Walmart.com 所示、无制造商缺陷、产品照片和描述与客户订购的商品准确匹配 |
| **Damaged** | 发货给客户的损坏订单率高 | 审查你产品的包装，确保考虑到产品类型和重量；审查你选择的承运商，评估是否有特定承运商造成重复性问题 |
| **Incorrect item** | 发送给客户的错误商品率高 | 审查你的报价确保产品照片和描述准确代表客户将收到的商品；审查你的库存确保 UPC 与正确产品准确关联 |
| **Arrived late** | 送达客户延迟的订单率高 | 审查你选择的承运商，评估是否有特定承运商造成重复性问题 |
