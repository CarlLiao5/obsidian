# Promotional Pricing: Set Up via the Promotions Dashboard

> 使用 Promotions 页面创建和管理促销

## 概述

促销定价允许您为客户提供优惠。本指南介绍如何使用 Seller Center 中的 [**Promotions**](https://seller.walmart.com/pricing/promotions) 页面创建和查看 Active、Scheduled、Draft、Expired 或 Canceled 促销定价。

## 核心步骤

### Step 1 — 开始

在 Seller Center 中 **Pricing** 下导航到 [**Promotions**](https://seller.walmart.com/pricing/promotions) 页面，您可以在其中创建新促销或管理现有促销。

**Pro 提示：** 使用 **Set up suggested promotion** 功能轻松选择正确的商品和价格。下载包含高潜力商品及其建议 **Promo price** 的电子表格。

### Step 2 — 创建促销

选择 **Create promotion** 并指定：
- **Promotion name：** 用于该促销组的名称
- **Discount Value：** 促销组的值（例如：5% off、$5 off）
- **Strikethrough promotion rules：**
  - Percent off：对促销组下所有商品应用百分比折扣
  - Amount off：对促销组下所有商品应用特定金额折扣
  - Custom promo price：为促销组下每个单独商品定义特定促销价格
- **Start date – End date：** 促销组的开始到结束日期
- **Start time – End time：** 促销组的开始到结束时间

**注意：** 促销必须在开始前至少四小时设置。开始和结束时间为 UTC 时间。

### Step 3 — 添加商品

创建促销后，导航回 **Promotions** 页面。选择您的促销并选择 **Assign items**。您可以从目录添加商品或使用模板。

**从目录添加：**
搜索目录中的商品并分配到此促销组。

**从模板添加：**
下载模板并填写：
- **SKU：** 要分配给促销组的 SKU
- **Promo Price：** 仅当您在 Step 2 中选择 custom promo price 时需要定义

## 常见问题

**如何管理促销中的商品？**
通过选择特定促销名称并选择 **Manage items** 来管理促销的已分配商品：
- _Active_ 状态促销：只能删除或移除促销
- _Scheduled_ 或 _Draft_ 状态促销：可以添加或移除促销

**如何删除促销？**
有两种方式：
- 从 **Promo Setting Action** 列的下拉菜单中选择 **Delete**
- 从下拉菜单中选择 **Replace All**，并将模板列留空

**可以更改 Scheduled 阶段的促销规则吗？**
可以。但是不能在促销开始前 30 分钟内编辑促销规则。
