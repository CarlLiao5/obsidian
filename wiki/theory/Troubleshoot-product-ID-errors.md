# Troubleshoot Product ID Errors

> 解决商品设置过程中的产品 ID 错误

## 概述

产品标识符（通常称为产品 ID）是 Walmart.com 上每个销售商品的通用唯一代码。Walmart Marketplace 要求所有商品都必须有产品 ID。如果在商品设置过程中遇到添加产品 ID 的错误，请使用以下信息排查问题。

## 四种常见产品 ID 错误

### 1. 在设置过程中遇到产品 ID 错误

**该怎么办？**
GTIN 或 UPC 错误可能由输入错误、复制粘贴错误或不存在的产品 ID 号码引起。检查电子表格上的产品 ID 条目并重新提交。

### 2. 商品没有产品 ID

**该怎么办？**
Walmart Marketplace 不允许在 Walmart.com 上使用不正确或缺失产品 ID 的商品。如果您想销售没有产品 ID 的商品，请购买 GS1 认证的产品 ID。如果您认为商品应该有产品 ID 但没有，可以[申请 GTIN 豁免](request-a-gtin-exemption)。

### 3. 产品被错误合并

**该怎么办？**
当存在产品 ID 或内容不匹配时，错误的信息（如产品名称、图片、描述和其他产品规格）可能会被合并到商品页面。

例如，如果您销售的是 12 块香皂，但使用的是 24 块香皂的产品 ID，您的商品可能会与其他提供 24 块香皂的报价合并到同一商品页面。

有关如何将商品设置为变体（相同商品但不同包装数量），请访问 [Product variant groups: Overview](Product-variant-groups_-Overview)。

### 4. 想防止产品 ID 错误

**该怎么办？**
请记住，不正确的产品 ID 会导致列表错误，并可能导致低销量、高退货率和负面客户体验。

防止产品 ID 错误的提示：
- 通过查看商品包装上的条形码或制造商网站上的产品来验证条形码
- 仔细检查商品产品 ID 没有输入错误和复制错误
- 确保您使用的产品 ID 适用于正确的数量和版本
- 如果商品没有产品 ID，切勿创建虚假的

## 常见问题

**如果持续收到商品的产品 ID 错误怎么办？**
如果我们发现上传到目录中的产品 ID 导致内容错误，我们会自动取消发布您的商品。您可以在 Seller Center 中查看未发布原因。Walmart 保留将不匹配频率计入卖家绩效指标的权利。
