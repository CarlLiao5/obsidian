# Sponsored Products 展示位置详解

## 概述

Sponsored Products 广告在沃尔玛平台上拥有多个展示位置，每个位置都有独特的用户触达方式和广告效果。了解各展示位置的特点，有助于广告主制定更精准的投放策略。

---

## 展示位置概览

| 展示位置 | 页面位置 | 主要用途 | 适用活动类型 |
|----------|----------|----------|--------------|
| **Search In-grid** | 搜索结果页面 | 关键词搜索流量 | 手动/自动 |
| **My Items In-grid** | 商品详情页 | 竞品抢夺流量 | 手动/自动 |
| **Carousels** | 首页/类目页轮播 | 品牌曝光推广 | 品牌活动 |
| **Buy Box** | 商品详情页购买区 | 促进直接转化 | 手动/自动 |
| **Stock Up** | 大促/批量采购页 | 批量购买引导 | 促销活动 |

---

## Search In-grid（搜索结果网格位）

### 位置描述

Search In-grid 展示位置出现在消费者搜索关键词后的搜索结果页面中，位于自然搜索结果之内或旁边。

### 广告位特点

| 特点 | 说明 |
|------|------|
| **高意向用户** | 用户主动搜索，购物意图明确 |
| **竞争激烈** | 多个广告主竞价同一关键词 |
| **关键词驱动** | 依赖关键词匹配和出价 |
| **转化率高** | 相比其他位置，转化效果通常最佳 |

### 适用场景

- 新品推广（使用产品核心关键词）
- 促销商品推广（配合降价活动）
- 爆款商品维护（保持搜索排名）
- 竞品关键词截流

### 优化建议

- 使用精准的关键词匹配
- 关注 Quality Score 和相关性
- 设置合理的出价以保持竞争力
- 利用否定关键词排除无效流量

---

## My Items In-grid（我的商品网格位）

### 位置描述

My Items 展示位置出现在商品详情页的"经常一起购买"或"看了又看"区域，位于消费者浏览特定商品时的推荐位。

### 广告位特点

| 特点 | 说明 |
|------|------|
| **品类关联** | 展示互补或替代商品 |
| **决策影响** | 消费者处于购买决策阶段 |
| **浏览路径** | 消费者正在研究比较产品 |
| **曝光机会** | 基于用户当前浏览行为 |

### 适用场景

- 配件和补充产品推广
- 替代产品推荐
- 捆绑销售引导
- 竞品页面流量拦截

### 优化建议

- 选择与目标商品高度相关的商品投放
- 使用 Expanded Targeting 自动发现相关商品
- 关注曝光到点击的转化率
- 测试不同关联商品的组合效果

---

## Carousels（轮播广告位）

### 位置描述

Carousels 以横向滚动的方式展示多个商品，通常出现在首页推荐、类目页顶部或搜索结果页显著位置。

### 广告位特点

| 特点 | 说明 |
|------|------|
| **品牌展示** | 支持多商品集中展示 |
| **视觉冲击** | 更大的视觉空间吸引注意力 |
| **库存充足** | 适合有多个 SKU 的品牌 |
| **品牌认知** | 提升品牌整体曝光度 |

### 适用场景

- 品牌全系列推广
- 新品上市集中曝光
- 季节性促销活动
- 清理库存（多 SKU 同时推广）

### 优化建议

- 选择视觉一致性强的商品图片
- 使用统一的促销活动主题
- 合理安排商品展示顺序
- 监控各 SKU 的点击和转化数据

---

## Buy Box（购买框广告位）

### 位置描述

Buy Box 出现在商品详情页右侧或顶部的购买区域，与沃尔玛自营商品直接竞争购买机会。

### 广告位特点

| 特点 | 说明 |
|------|------|
| **黄金位置** | 最接近"加入购物车"按钮 |
| **高转化** | 消费者已有明确购买意向 |
| **价格敏感** | 此位置用户对价格较为敏感 |
| **激烈竞争** | 与沃尔玛自营直接竞争 |

### 适用场景

- 价格有竞争力的商品
- 急于提升销量的商品
- 替代沃尔玛自营的商品
- 限时促销活动

### 优化建议

- 确保价格具有竞争力
- 优先选择高评分商品
- 关注 Buy Box 占有率指标
- 结合促销和优惠券提升转化

---

## Stock Up（批量采购位）

### 位置描述

Stock Up 展示位置专门针对批量采购场景，通常出现在大促活动页面、批发页面或会员专享区域。

### 广告位特点

| 特点 | 说明 |
|------|------|
| **批量场景** | 针对大批量购买需求 |
| **高客单价** | 单次订单金额较高 |
| **会员优先** | Walmart+ 会员可见性更高 |
| **活动驱动** | 与平台大促活动紧密关联 |

### 适用场景

- 家庭装、量贩装商品
- 办公用品和商业采购
- 节日促销套装
- 会员专享活动

### 优化建议

- 设置阶梯定价或会员专享价
- 准备充足的库存
- 配合平台大促节奏
- 关注 Stock Up 专属报告数据

---

## 展示位置选择策略

### 预算分配建议

| 预算规模 | 推荐策略 |
|----------|----------|
| **小型（<$1000/月）** | 专注 Search In-grid，追求高转化 |
| **中型（$1000-$5000/月）** | Search + My Items，覆盖搜索和浏览 |
| **大型（>$5000/月）** | 全位置覆盖，Carousels 品牌曝光 |

### 活动类型匹配

| 活动目标 | 推荐展示位置 |
|----------|--------------|
| 新品破零 | Search In-grid |
| 爆款维护 | Search In-grid + Buy Box |
| 关联销售 | My Items In-grid |
| 品牌推广 | Carousels |
| 促销清仓 | Stock Up + Carousels |
