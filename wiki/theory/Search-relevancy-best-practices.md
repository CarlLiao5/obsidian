# 搜索相关性最佳实践

> 提高 Sponsored Products 广告相关性的策略和技巧。

## 相关性重要性

广告相关性直接影响：
- 广告排名（出价 + 相关性决定排名）
- 点击率（CTR）
- 转化率和 ROAS
- 广告成本效率

## 提高相关性的方法

### 1. 优化商品内容

- 标题中包含核心搜索词
- 描述中自然融入关键词
- 选择正确的商品类别和子类别
- 使用准确的 Product Type 和 Taxonomy

### 2. 关键词策略

- 使用自动活动让系统学习最佳关键词
- 定期审查关键词表现报告
- 使用 Keywords Planner 发现新关键词
- 将高表现关键词添加到手动活动

### 3. 活动结构优化

- 按相关性分组广告组
- 避免在同一广告组中混合不相关商品
- 为每个广告组设置明确的关键词集合

### 4. 持续监控和优化

- 使用 Item Keyword Performance 识别最佳关键词
- 移除低相关性关键词
- 根据表现数据调整出价

## 相关性诊断清单

- [ ] 商品标题包含目标关键词
- [ ] 商品描述准确且包含相关关键词
- [ ] 商品分类正确
- [ ] Taxonomy 和 Product Type 映射准确
- [ ] 使用 Keywords Planner 验证关键词关联
- [ ] 定期检查关键词表现报告
