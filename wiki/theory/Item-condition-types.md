# Item Condition Types

> 了解 Walmart 支持的不同商品状况类型，以及如何为商品选择正确的状况。

## 概述

在 Walmart Marketplace 上设置商品时，需要为每个商品指定一个状况类型。状况类型直接影响消费者的购买决策和退货率，选择不准确的状况可能导致客户投诉或退货。

## 商品状况类型

Walmart 支持以下商品状况类型：

| 状况 | 说明 |
|------|------|
| **New** | 全新、未使用、未开封的商品，带有原始包装 |
| **Open Box** | 打开但未使用过的商品，可能带有原始包装或无包装 |
| **Certified Refurbished** | 经过认证翻新的商品，经过测试和认证达到如新的功能状态 |
| **Used - Like New** | 使用过但状况如新的商品 |
| **Used - Very Good** | 有轻微使用痕迹但状况非常好的商品 |
| **Used - Good** | 有使用痕迹但状况良好的商品 |
| **Used - Acceptable** | 有明显使用痕迹或磨损但仍可正常使用的商品 |
| **Reconditioned** | 经过翻新处理的商品，恢复到正常工作状态 |

## 如何选择状况

在为商品选择状况时，请考虑：

- 商品的实际物理状况
- 商品的原始包装是否完整
- 商品是否经过测试或认证
- 商品的功能是否完全正常

## 重要说明

- 确保为商品选择的状况**准确反映其实际状况**
- 不准确的状况可能导致客户投诉或退货，影响卖家绩效指标
- 某些商品类别可能仅支持部分状况类型（如食品只允许 New）
