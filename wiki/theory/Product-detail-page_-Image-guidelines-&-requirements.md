# Product Detail Page: Image Guidelines & Requirements

> Walmart Marketplace 商品图片的标准和要求

## 概述

作为 Marketplace 卖家，您必须遵守 Walmart Marketplace 的图片指南和要求。高质量图片有助于为客户提供准确的内容，让他们在 Walmart.com 获得积极的购物体验。

## 内容要求

图片不得违反 [Prohibited Products Policy](Prohibited-products-policy_-overview) 或任何其他 Walmart 政策，包括但不限于：
- 宣传、促进仇恨、暴力或歧视的内容
- 显示明确裸体或粗俗语言的内容
- 包含淫秽、性暗示或色情材料的内容
- 被认为在文化上不适当或敏感的内容

**AI 生成内容：** AI 生成的内容必须准确、真实、符合版权规定，并符合 Marketplace 政策。

## 文件要求

| 要求 | 规格 |
| --- | --- |
| 文件格式 | JPEG, JPG, PNG, BMP（GIF 不允许） |
| 颜色格式 | RGB |
| 文件大小 | 5MB 或更小 |
| 位深度 | 每像素 8 位 |
| 像素尺寸 | 2200px x 2200px |
| 最小缩放像素尺寸 | 1500px x 1500px |
| 色卡像素尺寸 | 100px x 100px |
| 宽高比 | 1:1（正方形） |
| 背景 | 无缝白色 (255/255/255 RGB) |

## 图片 URL 要求

Walmart 提供图片托管服务。使用 Walmart 托管服务时，URL 有以下限制：

| 支持的 URL 格式 | 不支持的 URL 格式 |
| --- | --- |
| URL 必须以图片文件类型结尾 (.jpg, .png 等) | 无法公开访问的 URL |
| 通信端口必须是 8080、80、443 或 8443 | HTML 页面 |
| 大多数 Amazon 图片 URL 如果符合指南可以接受 | 包含查询的 URL |
| | 通信端口不是 8080、80、443 或 8443 的 URL |
| | 包含未编码特殊字符的 URL 路径 |
| | Dropbox URL |

## 常见问题

**每个产品类别有特定的图片要求吗？**
是的，某些类别有额外要求。更多信息请查看 [Content Standards: Overview](Content-standards_-Overview)。

**图片不符合要求会怎样？**
不合规可能导致商品被取消发布。

**可以使用库存照片吗？**
不允许。必须使用准确反映在 Walmart Marketplace 上销售的商品的真实产品图片。
