# Shipping templates: Overview

> 创建和管理配送模板，定义每个区域的配送规则

## 概述

在开始向客户配送商品之前，您需要设置配送规则并为每个配送区域定义配送规则。您可以在 Seller Center 中创建配送模板或使用 Walmart APIs。本指南将介绍您可以创建的不同类型的模板。

## 工作原理

在配送模板配置中，您将选择首选的配送方式、区域和配送成本。

### 手动配送设置（国内）

如果您不使用批准的运输承运人，则需要手动添加配送设置。您可以添加几种类型的模板：

| 模板类型 | 说明 |
| --- | --- |
| **Default** | 每个 Seller Center 账户都有一个默认配送模板，无法删除，但可以自定义以满足您的配送需求。除非您已创建并分配了其他自定义配送模板的 SKU，否则默认配置将适用于您的整个目录。建议在有多个自定义配送模板时，将最长的预计运输时间应用于默认模板，以避免默认模板覆盖您的自定义模板 |
| **Custom** | 您可以为自配送商品创建最多 60 个不同的自定义配送模板。如果您使用第三方配送提供商（如 Deliverr），可以为其配送的商品额外创建 30 个模板。如果 SKU 未分配到自定义配送模板，商品将反映默认配送模板中的设置，包括运输时间、配送预计和配送费用规则 |
| **Paid Standard** | 为不符合免费标准配送条件的商品使用付费标准配送模板。如果需要禁用 Value 配送，也使用此模板。设置此模板后，需要分配将链接到新创建的付费标准模板的 SKU |
| **Freight** | 用于运输需要小于卡车装载（LTL）的大型和重型商品。如果商品属于指定的货运类别或其运输重量超过 150 磅，则有资格使用货运模板。设置模板后，选择 6 至 10 天的运输时间 |

### 国际配送设置

- **International**：用于从中国、香港或印度发货的商品。国际配送模板允许您配置最长 13 天的运输时间

## 功能概览

| 功能 | 开始使用 |
| --- | --- |
| **添加配送设置** | 了解如何为国内或国际配送设置创建配送模板 |
| **分配 SKU 到模板** | 创建配送模板后，您需要分配 SKU 到模板 |
| **更新配送设置** | 修改您的配送模板（如需要更新或更改配送设置） |

## 常见问题

**创建配送模板后我该怎么办？**

如果您是手动设置模板，则需要将 SKU 分配到配送中心和一个配送模板。创建新模板后，必须等待至少四个小时才能分配 SKU。如果不等待所需时间，您的配送设置将恢复到默认配送模板，创建的所需模板将不会被激活。

**如何从 Value 配送中移除 SKU？**

要移除 Value Shipping 中的 SKU，必须创建付费标准配送模板。创建模板后，必须等待四个小时，然后将 SKU 添加到付费标准配送模板。这将从 Value 配送模板中移除 SKU 并将其添加到付费标准模板。

**为什么我的客户面向配送预计与分配的配送模板设置不匹配？**

在许多情况下，Walmart 会根据历史运输承运人绩效、卖家绩效和外部因素调整客户面向的配送预计。

**如何确认我的 SKU 分配到哪个模板？**

如果您想确认每个 SKU 分配到哪个模板，可以从 Seller Center 中的 [**Reports**](https://seller.walmart.com/analytics/reports) 中心拉取配送和配置报告。

**为什么我无法编辑模板的配送费用？**

每个模板每小时限制一次更改。每天最多可以进行 10 次编辑。如果超过此限制，您的配送模板将被锁定 24 小时。
