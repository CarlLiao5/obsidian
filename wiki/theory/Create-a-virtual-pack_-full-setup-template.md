# Create a Virtual Pack: Full Setup Template

> 使用虚拟包装模板批量设置虚拟包装商品

## 概述

本指南介绍如何使用虚拟包装完整设置模板在 Walmart Marketplace 上设置虚拟包装商品。

**注意：** 使用此功能需要 Seller Center 的管理员级别访问权限。

## 核心要点

### 虚拟包装设置流程

**步骤 1 — 准备工作**
- 导航至 Seller Center 中的 **Catalog**
- 点击 _Add items_
- 选择 **Virtual Pack** 部分

**步骤 2 — 选择商品**
- 选择要包含在虚拟包装中的各个商品
- 指定每个商品的单位数量

**步骤 3 — 完成设置**
- 填写所有必需的属性
- 提交虚拟包装

### 虚拟包装示例

虚拟包装可用于捆绑销售或批量销售场景，例如：
- 将多个单品组合成一个 SKU 销售
- 创建批发包装规格
