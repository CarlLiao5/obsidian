# 无效流量过滤方法

> Sponsored Search 广告中无效流量（GIVT）过滤的指标定义和说明。

## 无效流量概述

General Invalid Traffic（GIVT）是通过以下方式识别的流量：
- 应用行业黑名单列表
- 其他标准化参数检查

## 无效流量指标

| 指标 | 说明 |
|------|------|
| **Invalid Clicks** | 识别为 GIVT 的无效点击数 |
| **Invalid Click Rate** | 无效点击占点击总数的百分比（Invalid Clicks ÷ Clicks） |
| **Invalid Impressions** | 识别为 GIVT 的无效展示数 |
| **Invalid Impression Rate** | 无效展示占展示总数的百分比（Invalid Impressions ÷ Impressions） |
| **Invalid Item Clicks** | 商品级别的无效点击数 |
| **Invalid Item Click Rate** | 无效商品点击百分比（Invalid Item Clicks ÷ Total Clicks） |
| **Invalid Item Impressions** | 商品级别的无效展示数 |
| **Invalid Item Impression Rate** | 无效商品展示百分比（Invalid Item Impressions ÷ Total Impressions） |
| **Item Clicks** | 特定商品广告的净点击数（过滤 GIVT 后） |
| **Item CTR** | 商品点击数 ÷ 商品展示数 |
| **Item Impressions** | 特定商品广告的净展示数（过滤 GIVT 后） |
| **Total Clicks** | 过滤前广告总点击数 |
| **Total Impressions** | 过滤前广告总展示数 |

## 报告中的标注

- `*` 标注的指标：经过 GIVT 过滤后的净数据
- `**` 标注的指标：基于广告点击的转化指标
