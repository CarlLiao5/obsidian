# 广告创意与编辑标准

> Walmart Connect 广告创意内容的合规要求，包括广告文案、图片、视频及促销活动的完整规范。

## 广告声明与内容版权要求

### 核心原则

- 所有广告声明必须**准确、真实、有依据**
- 广告主须确保已获得所有第三方知识产权的使用授权
- 所有声明须由广告主自行创建和确认，Walmart 不创建或验证产品声明
- Walmart **不为广告主**创建口号、标语或商标
- 须经 Walmart 提前审批的声明类型：**6 周**审批周期

### 需要提前审批的声明

| 声明类型 | 审批要求 | 周期 |
|----------|----------|------|
| **"Made in USA"（美国制造）** | 须提前审批（含使用美国国旗等符号的暗示性声明） | 6 周 |
| **"Green" / 可持续发展/环保声明** | 须提前审批（含环保认证、碳抵消、无毒、不含XX、可回收等） | 6 周 |
| 竹制品声明 | **禁止** | - |
| 性能声明 | 广告主自行确认准确性 | - |
| 代言/证言 | 须确认所有陈述真实反映代言人实际体验 | - |
| 名人/网红/音乐/高风险 IP | 广告主须确保已获得所有必要授权 | - |

### 广告创意的通用要求

- 禁止拼写/语法错误、过度大写、特殊字符
- 禁止亵渎语言（含缩写、符号、星号、emoji 的暗示性脏话）
- 禁止个性化定向语言（如"我们注意到您搜索了XXX"）
- 禁止对比竞品品牌或使用竞品 logo
- 如需订阅服务，须在广告中明确说明
- 须在广告中明确标注免费/折扣的使用限制条件

### 文案风格

- **语言规范**：使用标准英语，避免俚语、缩写或网络用语
- **专业语气**：保持专业、友好的语气，避免攻击性表述
- **品牌一致性**：保持与品牌调性一致的风格
- **清晰简洁**：信息传达清晰，避免歧义

### 定价声明规则

| 场景 | 允许使用具体定价/节省声明 |
|------|--------------------------|
| Rollbacks（价格回调） | ✅ Display 广告可使用 Dynamic Rollback 工具 |
| Wireless（无线产品） | ✅ 预付费计划、激活配套计划的手机可使用 |
| 其他场景 | ❌ 不允许 |

## 图片规范

### 禁止内容

- 成人或不当图片（含裸体、暗示裸体、聚焦身体部位）
- 性行为（含暗示）
- 暴力/性暗示内容
- 粗俗内容

### 特殊类别要求

**医疗健康类**：
- 须正确展示产品使用方法
- 补充剂仅展示完整药瓶或预期剂量
- 仅展示适合该产品的年龄段人群

**减肥类产品**：
- 须展示真实的前后对比照片
- 禁止使用经修改的照片

**酒精/烟草**：
- 站内广告：禁止展示酒精品牌或消费图像
- 站外广告：可展示酒精图像但不得包含 Walmart logo
- 禁止展示未成年人或吸引未成年人的形象
- 禁止展示大麻/CBD/电子烟产品（外用乳液和面霜除外）

**武器图像**：
- 禁止展示人持致命武器（含玩具枪、气枪、刀等）
- 允许展示厨刀、多功能工具中的单件刀

### 图片技术规格

| 参数 | 要求 |
|------|------|
| **分辨率** | 最小 300×300 像素 |
| **格式** | JPG、PNG 或 GIF |
| **文件大小** | 不超过 5MB |
| **文字比例** | 图片中文字不超过 20% |

## 视频规范

- 内容必须为 **G 级（适合全年龄）**
- 禁止：消音、紧急警报音效、闪光效果、像素化遮盖、长时间停顿
- 禁止包含 Walmart 竞争对手
- **时长**：建议 6-15 秒，移动端优先
- **格式**：MP4 或 MOV
- **字幕**：建议添加字幕以提高可访问性

## 优惠券政策

Walmart **官方不推广优惠券**（Manufacturer coupons 除外）。

Display 广告优惠券规则：
- 不得直接链接到优惠券页面或提及优惠券
- 必须是 Manufacturer coupon（非 Walmart 专属），不得联合品牌
- 首选 CTA：`Save $X` 或 `Save XX¢`
- 通过 Quotient 画廊页面的数字优惠券**可支持**

---

## 抽奖/赠品/竞赛政策

所有涉及抽奖、赠品、竞赛须**提前获得 Walmart Legal 和 Ad Policy 团队审批**。活动规则必须完整透明，中奖概率须合理披露，法律声明须符合当地法规。

---

## 落地页要求

| 场景 | 落地页要求 |
|------|-----------|
| 站内广告 | 须留在站内（除非抽奖/赠品/竞赛从 Brand Page 跳转） |
| 站外广告（DSP） | 可直接链接到站外页面 |
| Walmart App | 仅购买后广告可跳转站外 |

站外落地页要求：
- 须获得 Walmart Ad Policy 团队批准
- 不得直接链接到竞品网站
- 须包含隐私政策（若收集个人信息）
- 必须适合移动端
- 涉及支付的页面必须使用 HTTPS

---

## Walmart Logo 使用规范

- **使用场景**：仅在与沃尔玛直接合作关系时使用
- **Logo 版本**：必须使用官方提供的标准 Logo
- **尺寸要求**：Logo 周围必须保留足够空白区域
- **禁止修改**：不得拉伸、变形或更改 Logo 颜色
- **品牌调性**：使用场景必须符合沃尔玛品牌形象
