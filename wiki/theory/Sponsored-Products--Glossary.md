# Sponsored Products 词汇表 (Glossary)

> Sponsored Products 广告核心术语定义与性能指标说明。

## 一般术语

| 术语 | 定义 |
|------|------|
| **Ad Group（广告组）** | 活动内的容器，用于管理商品和关键词（若适用） |
| **Ad Group ID** | 广告组的唯一数字标识符 |
| **Ad Spend（广告花费）** | 广告主花费总额，按点击成本累加计算 |
| **Advertised Item ID** | 所推广商品的 Walmart 在线商品 ID |
| **Advertiser ID** | 广告账户的唯一数字标识符（通常 5-10 位） |
| **Attribution Type（归因类型）** | 广告归因的商品销售是同一商品还是同品牌其他商品 |
| **Automatic Campaign（自动活动）** | Walmart 根据商品标题、描述、类目自动匹配关键词的活动类型 |
| **Bid（出价）** | 顾客点击广告时愿意支付的最高金额 |
| **Bidded Keyword（投放关键词）** | 广告主选择投放的标准化词或词组 |
| **Buy Box（购买框）** | 商品详情页上顾客加入购物车的核心位置 |
| **Campaign（活动）** | 管理广告的框架，含名称、日期、预算 |
| **Campaign ID** | 活动的唯一数字标识符 |
| **Carousel（轮播）** | 商品详情页底部、搜索结果等位置的广告展示格式 |
| **Daily Budget（日预算）** | 每日最高花费限额 |
| **Digital Purchase（数字购买）** | Walmart 网站/App 上的购买（配送/自提/送货） |
| **Keyword（关键词）** | 顾客搜索词或广告主用于定向的词 |
| **Manual Campaign（手动活动）** | 广告主自选关键词的活动类型 |
| **Marketplace Seller（市场卖家）** | 在 Walmart 网站展示库存的第三方卖家 |
| **Normalized Query（标准化查询）** | 将关键词变体（如 book/books）归并为同一词 |
| **Placement（展示位置）** | 广告在 Walmart 页面上的具体位置（Browse/Search In-grid、Buy Box、Carousel 等） |
| **Primary Variant（主变体）** | 变体组中在搜索结果和商品页显示的默认商品（原 Base ID） |
| **Revenue（收入）** | 广告带来的购买金额 |
| **Search In-grid** | 搜索结果网格内的优质广告位置 |
| **Total Budget（总预算）** | 活动生命周期内的最高总花费 |

## 活动性能指标

| 指标 | 定义 |
|------|------|
| **Advertised Add to Cart** | 点击广告后 14 天内加入购物车的所推广商品次数 |
| **Advertised Item Sales** | 所推广商品及其变体组产生的归因销售总额 |
| **Advertised Item Units** | 所推广商品及其变体的归因购买件数 |
| **Average CPC** | 每次点击的平均成本 |
| **Clicks** | 广告点击次数（已过滤无效流量） |
| **Click-through Rate (CTR)** | 点击率（点击数/展示数） |
| **Conversion Rate** | 转化率（订单数/点击数） |
| **Cost-per-click (CPC)** | 每次点击成本 |
| **Impressions** | 广告展示次数（已过滤无效流量） |
| **New-to-brand Orders** | 一年窗口期内对同品牌商品的首次订单数 |
| **New-to-brand Sales Revenue** | 新品牌购买的总归因销售额 |
| **New-to-brand Units** | 新品牌订单的购买件数 |
| **Orders** | 点击广告后 14 天内产生的订单数（含推广及其他商品） |
| **Other Add to Cart** | 同品牌其他商品加入购物车次数 |
| **Other Item Sales** | 同品牌其他商品的归因销售额 |
| **ROAS（广告支出回报率）** | 归因总收入/广告花费 |
| **Total Attributed Sales Revenue** | 广告带来的总归因销售额（含推广及其他商品） |
| **Units Sold** | 同品牌购买件数（含推广及变体） |

## 展示份额指标（Impression Share）

| 指标 | 定义 |
|------|------|
| **Search Term Impression Share** | 获得展示的搜索词数 / 符合展示条件总搜索词数 |
| **Search Term Impression Rank** | 同一搜索词下广告主的展示份额排名 |
| **Top of Search Impression Share** | 顶部搜索位置展示份额 |
| **Top of Search Impression Rank** | 顶部搜索位置的展示排名 |

## 无效流量指标（Invalid Traffic）

| 指标 | 定义 |
|------|------|
| **Invalid Clicks** | 识别为一般无效流量（GIVT）的无效点击数 |
| **Invalid Click Rate** | 无效点击率（无效点击/总点击） |
| **Invalid Impressions** | 识别为 GIVT 的无效展示数 |
| **Invalid Impression Rate** | 无效展示率（无效展示/总展示） |
| **Item Clicks** | 商品广告点击数（已过滤 GIVT） |
| **Item CTR** | 商品点击率（商品点击/商品展示） |
| **Item Impressions** | 商品广告展示数（已过滤 GIVT） |

## 归因类型（Attribution Type）

| 归因类型 | 定义 |
|----------|------|
| **Advertised Item Sale** | 购买商品与推广商品相同或在同变体组内 |
| **Other Item Sale** | 购买商品与推广商品不同，但属于同一品牌和卖家账户 |

## 补充术语

| 术语 | 定义 |
|------|------|
| **Variant Group（变体组）** | Walmart 为优化购物体验，将同一品牌/款式的相似商品聚合在同一商品页。变体组商品最多可差异化 3 个属性（如颜色、尺寸、图案）。每个变体组可包含第一方商品（Walmart 自营）和第三方商品（卖家）。每个变体商品保留唯一 Item ID。**注意：Sponsored Brands 和 Sponsored Videos 活动仅主变体（Primary Variant）可作为推广商品。** |
| **Primary Variant（主变体）** | 变体组中在搜索结果和商品页显示的默认商品（原 Base ID） |
| **Omni Purchase（全渠道购买）** | 跨所有 Walmart 销售渠道的购买（含数字购买和门店购买） |
| **Buy In-Store Purchase（门店购买）** | 顾客在 Walmart 实体店收银台完成的购买 |
| **Page Type（页面类型）** | 广告展示的页面分类 |
| **Seller Center** | Marketplace 卖家管理商品的管理门户（[Seller Center](https://seller.walmart.com)） |
| **Supplier One** | 供应商管理商品的管理门户（[Supplier One](https://one.walmart.com)） |
| **Supplier（供应商）** | 向 Walmart 供货并由 Walmart 销售配送的商品来源（第一方库存） |

## 新品牌指标（New-to-Brand Metrics）

> 基础定义见上方「活动性能指标」表格，此处补充百分比指标和详细说明。

| 指标 | 定义 |
|------|------|
| **% of Orders New-to-Brand** | 新品牌订单占总订单的百分比（新品牌订单数/总订单数） |
| **% of Sales New-to-Brand** | 新品牌销售额占总归因销售额的百分比（新品牌销售额/总归因销售额） |
| **% of Units New-to-Brand** | 新品牌购买件数占总购买件数的百分比（新品牌件数/总件数） |
| **Order Rate（订单率）** | 新品牌订单与点击数之比（新品牌订单数/点击数） |
| **In-Store Orders（beta）** | 归因到 Walmart 实体门店购买的订单数（含推广商品和其他商品） |
| **In-Store Units Sold（beta）** | 归因到门店购买的件数（含推广商品和其他商品） |
| **In-Store Advertised Sales（beta）** | 门店购买中匹配推广商品或其变体组的归因销售额 |
| **In-Store Attributed Sales（beta）** | 归因到 Walmart 门店购买的总销售额 |
| **In-Store Other Sales（beta）** | 门店购买中属于同品牌但非推广商品或变体的归因销售额 |

## 无效流量报告指标（Invalid Traffic Report）

| 指标 | 定义 |
|------|------|
| **Item Clicks** | 商品广告点击数（已过滤 GIVT）= Sponsored Products + Sponsored Brands + Sponsored Videos 商品广告点击 |
| **Item CTR** | 商品点击率（商品点击数/商品展示数） |
| **Item Impressions** | 商品广告展示数（已过滤 GIVT） |
| **Total Clicks** | 过滤前总点击数（含 GIVT） |
| **Total Impressions** | 过滤前总展示数（含 GIVT） |
| **Invalid Item Clicks** | 识别为 GIVT 的商品无效点击数 |
| **Invalid Item Click Rate** | 无效商品点击率（无效商品点击/商品总点击） |
| **Invalid Item Impressions** | 识别为 GIVT 的商品无效展示数 |
| **Invalid Item Impression Rate** | 无效商品展示率（无效商品展示/商品总展示） |
| **Invalid Video Clicks** | 视频广告无效点击数 |
| **Invalid Video Impressions** | 视频广告无效展示数 |
| **Invalid Video Click Rate** | 无效视频点击率 |
| **Invalid Video Impression Rate** | 无效视频展示率 |

## MRC 认证指标（Media Rating Council）

> Walmart Sponsored Search 的点击、展示和 CTR 指标已获 MRC 认证，涵盖桌面端、移动网页和 App 端。所有认证指标均经过一般无效流量（GIVT）过滤。

| 认证指标 | 所在报告 |
|----------|----------|
| Total Impressions | Invalid Traffic Report |
| Item Impressions / Video Impressions | Invalid Traffic Report |
| Invalid Impressions / Invalid Impression Rate | Invalid Traffic Report |
| Total Clicks | Invalid Traffic Report |
| Item Clicks / Video Clicks | Invalid Traffic Report |
| Invalid Clicks / Invalid Click Rate | Invalid Traffic Report |
| Impressions / Clicks / CTR | Dashboard, Advertiser Reports, On-demand Reports, Custom Reports |
| Item CTR / Video CTR | Invalid Traffic Report, Custom Reports |

详细路径：Walmart Connect Ad Center → Sponsored Search → Reports → Advertiser → Campaign Performance → Invalid Traffic Report
