# WFS 账户联系人（WFS Account Manager Contacts）

> 在 Seller Center 中管理 WFS 账户联系人信息，确保沃尔玛能及时与你沟通。

## 概述

使用 WFS 必须在文件中指定一名账户经理联系人。保持联系人信息更新，可确保：支持问题得到解答、产品更新及时传达等。

**权限要求：** 须拥有 Admin 或 Read and Write 访问权限。

## 联系人角色

| 角色 | 职责 | 通知场景 |
|------|------|---------|
| **WFS Account Manager Contact** | 主要联系人和支持案件对接人 | 所有账户、订单问题 |
| **WFS Content Manager Contact** | 商品内容问题对接人 | 商品内容相关问题 |
| **WFS Warehouse Manager Contact** | 库存管理问题对接人 | 库存管理及退回商品处理 |

多联系人的优势：若首位联系人未回复，沃尔玛将依次联系其他联系人。

**注意：** 分发列表（Distribution lists）不能设置为账户经理联系人。

## 设置步骤

1. 登录 Seller Center → 点击齿轮图标进入账户设置
2. 进入 **Partner Profile** → **Manage Contacts**
3. 添加 WFS Account Manager：姓名、邮箱、电话
4. 可选：添加 WFS Content Manager 和 WFS Warehouse Manager
