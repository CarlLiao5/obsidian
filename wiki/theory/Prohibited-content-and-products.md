# 广告政策 — 禁止内容与商品

> 所有通过 Walmart Connect 购买的广告必须遵守的禁止内容和商品规定。

## 禁止内容

以下内容在任何广告中均**不得出现**：

**对象相关：**
- 针对儿童或收集儿童信息的广告
- 将儿童置于成人或危险场景中（武器、不系安全带、无成人监护等）
- 宗教鼓吹或歧视
- 神秘/撒旦符号（666、五芒星、异教符号等）
- 政治内容（含争议社会问题、选举、政党、政治 agendas）
- 针对受保护群体的歧视性内容（种族、肤色、宗教、残障、性别、性取向、年龄等）

**内容质量：**
- 欺骗性、虚假或误导性内容
- 促进可能导致人身或精神伤害的行为
- 鼓励或美化过度饮酒/吸毒
- 非法药物、相关产品或用具
- 剥削性或情感骚扰内容（自然灾害、虐待、暴力内容等）
- 猥亵内容（含模糊字母、暗示性对话、手势等）
- 暴力内容（强烈、图形化、过度或逼真的暴力）
- 暴露性内容（含内衣展示，特殊规定除外）
- 色情或明确性内容
- 侵犯知识产权、隐私权或_publicity 权的内容

**其他禁止：**
- 不得与其他竞争对手品牌进行比较
- 不得提及包装外的 GLP-1s
- 不得使用 PFAS/PFOA/PFOS/PTFE Free 声明
- 禁止 Emergency Alert System 音调
- 禁止 1-900 电话号码
- 禁止垃圾信息

## 禁止商品与服务

以下商品和服务**不得推广**：

| 类别 | 禁止项 |
|------|--------|
| **烟草相关** | 香烟、电子烟、雾化器、尼古丁袋 |
| **非法药物** | 非法/娱乐性药物及周边产品 |
| **武器/爆炸物** | 枪械、弹药、火药、BB枪、Airsoft、弹簧刀、狩猎刀、爆竹 |
| **成人用品** | 性增强产品、成人出版物/杂志/网站 |
| **金融诈骗** | 现金垫付、发薪日贷款（利率>36%）、金字塔计划 |
| **加密货币** | 虚拟货币 |
| **武器许可** | 武器许可证、认证、安全课程 |
| **假冒商品** | 伪造商品 |
| **违规测试** | 毒品检测套件、性健康检测套件 |
| **其他禁止** | 活体动物（宠物救助例外）、骨灰盒、危险化学品、召回商品 |
