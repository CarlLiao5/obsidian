# 超预算功能

> 识别和解决每日/总预算耗尽（Out-of-budget）和预算风险（Budget-at-risk）问题。

## 功能概述

**Out-of-budget** — 活动已用完每日和/或总预算，广告停止展示。需增加预算使活动恢复投放。

**Budget-at-risk** — 活动在过去 7 天内至少一次用完日预算，当前以次优预算运行。广告仍在展示但面临耗尽风险。

## 功能组件

| 功能 | 说明 | 位置 | 更新频率 |
|------|------|------|----------|
| **警报** | 高可见性通知，标识预算问题 | All Campaigns、Campaign 编辑页、邮件通知 | 近实时 |
| **推荐** | 个性化、可操作的建议，提供最优预算 | All Campaigns、Campaign 编辑页、Dashboard | 每日 |
| **关键指标和洞察** | 估算漏报印象/点击、平均封顶时间等 | All Campaigns、Campaign 编辑页、Dashboard | 每日 |

## 关键指标说明

| 指标 | 说明 |
|------|------|
| **Sugg. Daily Budget** | 建议日预算，防止预算耗尽 |
| **Sugg. Total Budget** | 建议总预算，防止预算耗尽 |
| **Est. Missed Impressions** | 估算漏报印象范围（基于历史数据，非保证） |
| **Est. Missed Clicks** | 估算漏报点击范围 |
| **Avg. Cap-out Time** | 过去时间窗口内活动用完预算的平均时间 |

## 使用位置

- **Dashboard 和推荐页面 Widgets**：显示超预算活动和建议的每日/总预算，一键应用全部推荐
- **All Campaigns 页面**：列超预算活动，应用预算推荐
- **Campaign 编辑页**：顶部警报标签，查看和调整预算
- **邮件警报**：在 All Campaigns 页面顶部邮件图标设置
