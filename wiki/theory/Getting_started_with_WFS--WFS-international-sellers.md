# WFS 国际卖家（International Sellers）

> 非美国境内卖家使用 WFS 的特殊要求与常见问题解答。

## 概述

WFS 接受来自美国境外的库存，但所有 WFS 路由和包装指南（如打包、贴标）同样适用。

## 核心要求

| 要求 | 说明 |
|------|------|
| 清关 | 卖家自行负责清除美国海关（使用 Walmart Cross Border 可选其指定报关行） |
| 商品要求 | 须满足 [WFS 产品要求](https://marketplacelearn.walmart.com/guides/Getting%20started/Walmart%20Fulfillment%20Services%20(WFS)/Select-items-for-WFS) |
| 包装准备 | 按 [WFS Routing and Packaging Guide](https://azure-na-assets.contentstack.com/v3/assets/blta7903c6b840b702d/blt57da3349569cc081/WFS_Routing_Guide.pdf) 执行 |
| 退货地址 | 必须提供美国境内退货地址 |

## 关键规则

| 问题 | 答案 |
|------|------|
| 谁负责关税？ | 卖家自行承担，Walmart 非进口记录方 |
| Walmart 是否为最终收货人？ | 否；美国实体须以 EIN 作为收货人；无 EIN 者需指定其他企业 |
| WPC 首选承运商是否处理国际提货？ | 否；WPC 仅处理美国仓库提货 |
| EAN 是否可用于标签？ | **不可以**；必须使用 GTIN 或 UPC |
| 短少/丢失库存如何索赔？ | 需提供 3 份装运确认文件（含英文明细），按[_争议索赔流程](https://marketplacelearn.walmart.com/guides/Walmart%20Fulfillment%20Services%20(WFS)/Troubleshooting/Troubleshoot-WFS-receiving-issues:-open-a-dispute-claim)操作 |

## WFS 跨境进口 vs 自找清关

- **Walmart Cross Border Imports**：沃尔玛提供跨境运输（含报关选项），推荐使用
- **自找清关**：需自行委托货运代理将货物运至美国 WFS 仓库

## WFS 加拿大/墨西哥

加拿大和墨西哥 WFS 仅对部分卖家开放，可申请加入等待名单。
