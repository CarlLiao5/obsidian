# 卖家论坛使用指南（Seller Forum: Overview）

> Walmart Marketplace 卖家论坛——协作、提问、分享最佳实践的社区空间。

## 概述

卖家论坛是 Walmart 为 Marketplace 卖家提供的交流社区，登录 [Marketplace Learn](https://marketplacelearn.walmart.com/forum?forcedLogin=true) 后可匿名使用。你可发帖讨论、评论点赞、收藏有用帖子、获得徽章。

沃尔玛版主负责维护论坛规范、验证回复准确性、传达最新政策和程序变更。

## 核心规则

| 规则 | 说明 |
|------|------|
| 发帖前先搜索 | 避免重复帖子，论坛已有话题请直接回复 |
| 一个档案对应一个账户 | 不能用一个档案管理多个卖家账户 |
| 须遵守论坛政策 | 违反 [[theory/seller-forum-policy]] 可能导致发帖权限被移除 |

**匿名性：** 用户名匿名，但 Walmart 仍可查看账户信息。

## 功能

- 发帖讨论（Post new discussions）
- 评论和点赞
- 收藏有用帖子（Bookmark）
- 获得徽章（Badges）
- 浏览讨论（无需登录）

## 论坛 vs 客服

| 场景 | 使用 |
|------|------|
| 通用问题、经验分享 | 卖家论坛 |
| 账户特定问题 | Seller Center → Help 联系支持 |

**响应时间：** 其他卖家回复取决于活跃度；Walmart 版主回复通常 2 个工作日内。
