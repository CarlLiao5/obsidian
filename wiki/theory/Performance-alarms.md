# Performance Notifications

> 绩效通知

## 概述
如果违反了一项或多项卖家绩效标准，你可能会收到以下绩效通知。触发绩效警报将影响你的绩效指标，并可能导致受影响商品自动取消发布。

## 核心要点

### 通知类型表格

| 通知 | 说明 |
| --- | --- |
| **At Risk of Auto-Cancellation** | 订单已超过预期发货日期。如果未在自动取消窗口内更新订单状态，Walmart 将取消订单并退款给客户 |
| **Auto-Cancelled Orders** | 因未在预期发货日期后第四个日历日内提供有效跟踪，订单已被 Walmart 取消 |
| **Unshipped by Expected Ship Date** | 订单已超过预期发货日期且没有发货确认 |
| **Late Origin Scan** | 已发货订单在发货确认发送给 Walmart 后 48 小时内未被承运商扫描 |
| **Orders with an Expected Ship Date Today** | 订单应在收到通知当天发货 |
| **Sales Velocity** | 商品因价格差异被取消发布 |
| **Cancellations Velocity** | 取消订单数量超过预定义阈值 |
| **Daily ship** | 通知你在预期发货日期前未发货、缺少有效跟踪号或承运商扫描等问题 |
| **Performance warnings** | 通过电子邮件或 Seller Center 通知告诉你绩效低于关键指标 |
| **Weekly performance reports** | 总结你前一周的表现 |

### 最佳实践

- 在预期发货日期前发货
- 提供有效跟踪号
- 在 48 小时内响应客户消息
- 保持准确的库存水平
- 监控你的绩效指标
