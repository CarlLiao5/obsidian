# Dispute Standards

> 争议标准

## 概述
如果符合条件，你可以对客户退货、Walmart Customer Care 退款、无法配送并退回给发件人的包裹或错误收费的推荐费提出争议。

## 核心要点

### 通用标准

- 你必须在 Walmart 规定的特定时间范围内提交争议请求
- 每个商品只能提出一次争议
- 如果你发起了退货或退款，则不能争议费用
- 如果商品设置了 Keep It Rules，则不能争议费用
- 如果商品通过 Extended Returns 计划退货，则不能争议费用（因为你不会被收取这些退货）
- 如果商品退回到 Walmart 商店地点且在店内 12 天内未发货，则视为 "Lost-in-store"，你不能争议费用
- 你负责列出有效/正确的退货中心地址。如果地址无效，包裹可能会被重新路由到 Walmart 设施，你将无法获得包裹，且不能争议费用
- 除非符合退货豁免条件的商品，否则你不能向客户收取退货运费

### 争议类型表格

| 争议原因 | 说明 |
| --- | --- |
| 客户退货或退款 | 通过 **Returns** 仪表板提出 |
| 配送失败和退回发件人 | 通过 **Payments** 仪表板提出 |
| 推荐费调整 | 联系 Support |

### 争议资格

| 争议情况 | 可争议 |
| --- | --- |
| 你发起了退货或退款 | 否 |
| 商品有 Keep It Rule | 否 |
| 通过 Extended Returns 退货 | 否 |
| 退货超过 45 天 | 否 |
| 地址无效/无法送达 | 否 |

## 常见问题

**争议获批后会怎样？**
你将在 1-2 个付款周期内（最多 30 天）获得 Walmart 报销。

**我可以上诉争议决定吗？**
可以。如果对争议决定不满意，可以提交上诉。只能上诉一次，必须在收到拒绝或决定后 30 天内提交。
