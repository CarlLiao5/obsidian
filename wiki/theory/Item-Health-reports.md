# 商品健康报告 (Item Health Reports)

> 监控推广商品健康状态，优化广告表现的工具。

## 概述

Item Health 报告展示所有 Sponsored Search 推广商品的关键健康信号，帮助卖家识别需优化的商品并采取纠正措施。

**访问路径：** Reports → Item Health

**数据范围：** 仅显示已启用且上线至少 3 天的商品。

## 报告列定义

| 列 | 说明 |
|----|------|
| **Primary variant** | 是否为主变体（原 Base ID） |
| **Item status** | 商品状态（已发布/未发布） |
| **Product detail page views** | 今日或近 7 天商品页访问量 |
| **Item availability** | 商品可用性百分比（近 7 天） |
| **Win rate** | 赢得 Buy Box 的百分比（近 7 天） |
| **Listing quality** | Listing 质量分（仅 Marketplace 卖家） |

## Listing Quality Score（质量分）

### 评分维度

| 维度 | 说明 |
|------|------|
| **Content & discoverability** | 内容准确性、关键词优化 |
| **Offer** | 价格竞争力、配送速度、库存状态 |
| **Ratings & reviews** | 评分数量和评价数量 |

**目标：** 保持 Listing Quality Score > 60%

### 查看详细 breakdown

鼠标悬停在 Listing Quality 列的百分比上，可看到各维度得分。点击 "strengthen your Listing Quality score" 直达 Seller Center 优化页面。

## 建议行动指南

| 问题现象 | 可能原因 | 建议行动 |
|---------|---------|---------|
| 高浏览 / 低 Buy Box 胜率 | 竞品价格更低 | 降价/优化配送 或 换品 或 持续投放 |
| 高浏览 / 高 Buy Box 胜率 | 商品详情页不完整 或 竞争力不足 | 优化标题/描述/图片 或 降价/提升配送 |
| 低浏览 / 高 Buy Box 胜率 | 曝光不足 | 逐步提高 CPC 出价 或 扩大关键词匹配 |
| 低浏览 / 低 Buy Box 胜率 | 非理想广告商品 | 降价 或 换品 或 持续等待赢得 Buy Box |
| Item available ≤ 2% | 商品缺货 | 补货 |
| Item status = unpublished | 商品未发布 | 联系 Seller/Supplier Support |

## 注意事项

- Seller Center 数据实时更新，Item Health 报告每日更新
- 修改商品后约 24 小时反映在报告中
- 需有 Seller Center 访问权限才能查看 Listing Quality 详情页
