# Add Items in Bulk: Set Up Items by Match

> 通过匹配 Walmart 目录中的已有商品批量添加商品

## 概述

本指南介绍如何在 Seller Center 中通过匹配 Walmart.com 上已有商品来批量添加商品。如果要添加 Walmart.com 上尚未有售的商品，请参考 [Add multiple items: Full setup](Add-items-in-bulk_-full-setup)。

## 核心步骤

### Step 1 — 开始操作

在 Seller Center 中导航到您的 [**Catalog**](https://seller.walmart.com/catalog/list-items)，选择 **Add items** > **Upload in bulk** > **Match items** 部分，然后选择 **Build spreadsheet** 按钮。

### Step 2 — 下载并填写模板

下载模板后，模板按要求级别分组：

| 级别 | 描述 |
| --- | --- |
| **SKU** | 必填项 |
| **Required** | 必须完全填写，缺失会导致数据错误 |
| **Conditional/Required** | 以下条件需要主图 URL：Remanufactured、Pre-Owned: Like New、Pre-Owned: Good、Pre-Owned: Fair、New with defects |
| **Optional** | 库存、其他图片 URL、销售限制 |

### Step 3 — 上传模板

返回 **Upload in bulk** 页面，拖放或浏览选择您的文件。上传后可在 [**Activity Feed**](https://seller.walmart.com/catalog/activity-feed) 中追踪状态。

## 常见问题

**如果找不到匹配项怎么办？**
如果没找到匹配，建议使用 **Full Setup** 模板添加新商品。

**如果只需要匹配一个商品怎么办？**
可以使用 **Express setup** 功能。

**如何添加国际市场？**
访问 [Global onboarding: Set up an additional market](global-onboarding-set-up-an-additional-market)。

**Walmart 配送与卖家配送有何不同？**
Walmart 配送需要额外信息：原产国、尺寸、合规和贸易商品层级。
