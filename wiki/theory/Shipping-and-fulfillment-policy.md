# Shipping and fulfillment policy

> 卖家必须遵守的最低配送政策要求

## 概述

为客户提供一致的配送体验，您需要满足一套最低配送政策要求。本指南将介绍配送和履行政策的要求以及相关指南。

## 政策是什么？

下表概述了配送政策的最低要求和限制：

| 要求 | 禁止 | 允许 |
| --- | --- | --- |
| 必须支持至少一种配送方式向客户发货订单 | 不得从另一家零售商购买产品并让该零售商直接向客户发货（也称为零售套利） | 可以使用多渠道配送，只要您使用中性包装并使用不带任何其他零售商 LOGO、商标或品牌标识的无品牌配送车辆 |
| 必须选择适当的滞后时间、运输方式、承运人和承运人服务级别，以在预期交货日期或之前将订单交付给客户 | 不得使用竞争对手的盒子或包装发货 | |
| 必须满足为商品设置的预期交货日期。Walmart 保留更新或延长运输日期的权利，以期为 Walmart.com 客户提供一致准确的配送体验 | 不得运输包含任何非制造商或卖方的名称或其他标识符的包裹（运输单据、材料、发票或外部包装上），除非已获得 Walmart 豁免 | |
| 必须为 Seller Center 中的每个活跃配送中心配置运营计划，以定义在给定日期停止接收订单的时间范围 | 不得运输包含任何卖家或第三方促销或营销材料的包裹 | |
| 必须每周至少五天支持上午 11:00（当地时间）的发货 | 不得在未实际发货的情况下将订单标记为已发货 | |
| 必须谨慎使用额外休息日，仅用于特殊情况。每个配送中心每年最多可使用 30 天休息日 | | |
| 必须提供准确有效的跟踪号码，仅在包裹已交给承运人运输后才将其告知 Walmart | | |

## 附加指南

### 预期交货日期

预期交货日期是商品预计交付给客户的日期。它根据您为商品指定的订单截止时间、履行滞后时间、运营天数和运输时间计算。此预期交货日期可能与向 Walmart.com 客户显示的预计交货日期相同或不同。

向 Walmart.com 客户显示的预期交货日期根据您的历史准时配送率、承运人配送输入和 Walmart 的物流洞察计算。此日期也决定了您产品的搜索排名。

**注意**：您必须保持 90% 或更高的准时配送率以满足卖家绩效标准。

当商品超过预期交货日期 5 天且承运人在过去 3 天内未报告商品有任何移动时，Walmart 认为该商品处于终端状态。在这种情况下，Walmart 保留取消在预期交货日期后合理时间内仍未交付的订单的权利。您将承担任何相关费用。未能满足预期交货日期可能导致配送特权限制。

### 履行滞后时间

履行滞后时间是决定您准备发货商品所需天数的商品级别属性。Walmart Marketplace 要求卖家在收到订单后两个工作日内发货订单（除非您已提交滞后时间豁免请求并获得 Walmart 批准）。运营日是您在配送中心设置中指定的活跃日，默认为周一至周五，允许调整。

### 预期发货日期

预期发货日期是您必须使用有效跟踪信息将订单标记为已发货的日期。将订单标记为已发货并上传跟踪信息后，Walmart 将向客户发送发货通知。如果在预期发货日期或之前收到发货确认，订单被视为准时发货。

Walmart 将在预期发货日期后 4 个日历日仍未标记为已发货且未上传跟踪信息的订单自动取消。

| 履行滞后时间 | 必须发货的日期 |
| --- | --- |
| 0 天 | 当天 |
| 1 天 | 第二天 |
| 2 天 | 后天 |

**注意**：如果在订单截止时间后下订单，将额外增加一天到履行滞后时间。

### 运输时间

订单运输时间决定承运人在发货后将订单交付给客户所需的工作日数。运输时间要求特定于您特定的运输方式和支持的区域。

## 常见问题

**订单截止时间是什么？**

您每天停止接收订单的时间。

**我可以修改订单截止时间吗？**

是的。除非另有指定，您的配送中心将默认截止时间为当地时间下午 2:00，并将自动注册支持所有运营日（周一至周五）。未能配置订单截止时间可能会负面影响您的准时发货率。

**Walmart 支持哪些运输承运人？**

货运承运人列在这里，包裹承运人列在这里。

**Walmart 可以使用 Amazon Multi Channel Fulfillment (MCF) 吗？**

可以。使用 Amazon MCF 时，必须用中性盒子运输商品，并阻止 Amazon Logistics 作为承运人以确保订单不使用 Amazon 品牌车辆配送，并遵守 Walmart 的配送和履行政策。
