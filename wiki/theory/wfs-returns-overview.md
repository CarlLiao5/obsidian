# WFS 退货（WFS Returns Overview）

> WFS 订单退货的处理流程、费用与退款规则。

## 退货处理

当客户发起 WFS 订单退货时：
- Walmart 处理所有客服和退货物流
- 损坏/过期商品由 Walmart 处置
- 卖家收到退款通知

## 退货处理费

| 重量 | 费用 |
|------|------|
| ≤ 1 lb | $4.70 |
| 2 lb | $6.20 |
| 4–20 lb | $7.00 + $0.40/lb（超 4 lb）|
| 大件（可达 500 lb）| $155 + $0.80/lb（超 90 lb）|

## WFS vs 卖家自配送退货

| 维度 | WFS | 卖家自配送 |
|------|------|-----------|
| 退货处理 | Walmart 全程处理 | 卖家负责 |
| 运费 | Walmart 承担（RSS 政策）| 卖家决定 |
| 客户 | Walmart 客服 | 卖家客服 |
| 损坏品 | Walmart 处置 | 卖家接收 |
