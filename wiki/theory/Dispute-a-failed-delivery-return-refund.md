# Dispute a failed delivery return refund

> 争议配送失败退货运费

## 概述
如果配送不成功且客户声称未收到商品，你可能有资格争议该退货运费。如果符合条件，必须在退货运费发出后 **45 天内** 提出争议。在此时间范围外提出的争议将被拒绝。

## 核心要点

### 资格表格

| 配送失败/退回原因 | 争议资格 |
| --- | --- |
| 地址信息不正确或缺失（如单元号、邮编等） | 可争议 |
| 退款对你的 Seller Performance Standards 产生负面影响 | 可争议 |
| 客户因运输途中包装损坏而拒绝配送 | 不可争议 |
| 包裹送达错误地址 | 不可争议 |
| 包裹在运输中丢失 – 无延迟扫描 | 不可争议 |
| 包裹在运输中丢失 – 已确认延迟扫描 | 可争议 |
| 交付后丢失 (LAD) | 可争议 |
| 使用自己的承运商退回 | 不可争议 |
| 通过 Marketplace Return Shipping (RSS) 服务退回，金额超过 $100 | 可争议 |
| 通过 Marketplace Return Shipping (RSS) 服务退回，金额低于 $100 | 不可争议 |

### 提出争议步骤

**步骤 1 – 开始**
- 在 Seller Center 中导航到 **Transactions** 仪表板
- 可以按采购订单 (PO) 号或客户订单 (CO) 号搜索

**步骤 2 – 查看详情**
- 找到交易后，选择最右侧的 **三个点** 并选择 **View details**

**步骤 3 – 提出争议**
- 在 **Refund details** 部分选择 **Dispute this Refund**
- 在 **Tell us what went wrong** 部分选择原因
- 提供详细信息和文档，如：
  - 承运商的签收确认
  - 承运商的照片配送确认
  - 客户关于收到订单的电子邮件确认

## 常见问题

**提交争议后可以追踪状态吗？**
是的，你可以通过 Seller Center 中的 **Disputes** 仪表板追踪状态。
