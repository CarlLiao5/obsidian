# WFS 服务概述 (Walmart Fulfillment Services Overview)

> WFS 是沃尔玛提供的端到端仓储、配送、退货管理服务，类似 Amazon FBA。

## 概述

WFS（Walmart Fulfillment Services）是沃尔玛的自营仓储物流解决方案。卖家将商品发送至 Walmart 仓库后，平台负责存储、分拣、包装、配送、客服及退货处理。

**适用场景：**
- 希望提升配送速度和客户体验
- 需要专业仓储管理减少自运营负担
- 利用 Walmart 品牌背书提升转化

## WFS 核心优势

| 优势 | 说明 |
|------|------|
| **配送速度** | 次日达/两日达，提升客户满意度 |
| **Buy Box 优先** | WFS 商品在 Buy Box 竞争中享有优先权 |
| **客户服务** | Walmart 处理客户咨询和退货 |
| **多渠道支持** | 支持 Walmart Marketplace 以外的渠道（如自建站）订单发货 |
| **可信标志** | "Walmart fulfilled" 标志增强消费者信任 |
| **WPC 首选承运商** | 享受运费折扣和优先配送 |

## 费用结构

WFS 费用包含：

| 费用类型 | 说明 |
|----------|------|
| **仓储费** | 按商品体积（立方英尺）计费，按月收取 |
| **配送费** | 按重量和尺寸分段计费 |
| **长期仓储费** | 库存超过 365 天后收取 |
| **其他费用** | 标签费、预处理费、退货处理费等 |
| **费用计算器** | 参考 [[theory/WFS-fees-calculator]] |
