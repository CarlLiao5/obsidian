# Listing Quality: Bulk Attribute Editor

> 批量更新多个商品的属性

## 概述

Bulk Attribute Editor 帮助您提高 Content quality 分数，这是整体 [Listing Quality](Listingquality-and-rewards-dashboard) 分数的关键组成部分之一。本指南介绍如何使用此工具同时更新多个商品属性，如商品名称、描述、关键特性和图片。

## 核心步骤

### Step 1 — 开始

在 Seller Center 中导航到 [**Listing Quality**](https://seller.walmart.com/growth/listing-quality) 仪表板，选择 **Create spreadsheet** 开始设置批量编辑文件。

### Step 2 — 生成电子表格

使用过滤器对商品进行排序：

| 过滤器 | 描述 |
| --- | --- |
| **Customer Favorites** | 您的搜索最多和最畅销的商品 |
| **Walmart Fulfillment Services (WFS) Items** | 您的活跃 WFS 商品 |
| **Attributes with issues** | 需要编辑以提高 Listing Quality 分数的属性 |
| **Refine by item** | 根据 Listing Quality 分数、GMV 或页面浏览量应用额外过滤器 |

选择过滤器后，选择 **Create spreadsheet** 生成文件。

### Step 3 — 下载并提交电子表格

电子表格准备好后，选择 **Go to Activity Feed** 查看。在 [**Activity Feed**](https://seller.walmart.com/catalog/activity-feed) 中的 **Downloads** 标签下找到您的文件，选择 **download icon** 下载。打开下载的电子表格，填写或编辑您要更改的属性值，然后保存更新。返回 Listing Quality 仪表板并选择 **Upload spreadsheet** 提交您的更新文件。

**注意：** 分数可能不会立即更新。新发布的报价最多可能需要一周时间才能在 **Listing Quality** 仪表板中反映。

## 常见问题

**提高分数有什么好处？**
通过提高分数，您将增加获得 [Pro Seller](pro-seller-overview) 身份的机会。
