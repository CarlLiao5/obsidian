## WFS 费用一览

### 履约费（标准物品）

| 重量范围 | 费用 |
|----------|------|
| ≤ 1 lb | $3.45 |
| 2 lb | $4.95 |
| 3 lb | $5.45 |
| 4–20 lb | $5.75 + $0.40/lb（超4lb部分）|
| 21–30 lb | $15.55 + $0.40/lb（超21lb部分）|
| ≥ 31 lb | $17.55 + $0.40/lb（超51lb部分）|

### 附加费

| 类型 | 附加费 |
|------|--------|
| 服装 | +$0.50 |
| 危险品 | +$0.50 |
| 零售价 < $10 | +$1.00 |
| Oversize 1（最长边 >48" 或围度 >105"）| +$3.00 |
| Oversize 2（最长边 >96" 或围度 >130"）| +$20.00 |

### 仓储费

| 期间 | 费用 |
|------|------|
| 1月–9月 | $0.75/立方英尺/月 |
| 10月–12月（前30天）| $0.75/立方英尺/月 |
| 10月–12月（30天后）| $2.25/立方英尺/月 |
| 超365天 | $2.25/立方英尺/月（长期仓储费）|

## 关键规则

- WFS 费用**比竞争对手平均低 15%**
- 无注册费、月订阅费、库存最低/最高限制
- 超大件商品使用单独费率表（单品 >150lb）
- 体积重量 = 长×宽×高/139，与实际重量取较大值
