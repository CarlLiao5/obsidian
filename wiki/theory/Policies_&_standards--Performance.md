# 绩效标准（Performance）

> 卖家绩效仪表盘：退货率、未收到率、负面反馈率、承运商表现、区域表现。

## 概述

Walmart 要求所有 Marketplace 卖家达到绩效标准。本分类涵盖各项绩效指标的跟踪、监控和改进指南。

## 绩效指标

|| 指标 | 阈值 | 说明 |
||------|------|------|
| **Return Rate** | ≤ 6% | 退货率（60天内） |
| **Item Not Received Rate** | ≤ 2% | 未收到率 |
| **Negative Feedback Rate** | ≤ 2% | 负面反馈率 |
| **On-time Delivery** | ≥ 90% | 准时配送率 |
| **Valid Tracking Rate** | ≥ 99% | 有效追踪率 |
