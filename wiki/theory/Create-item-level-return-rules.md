# Create item-level return rules

> 在商品级别自定义退货设置

## 概述

通过在 Seller Center 中上传退货规则更新模板，您可以自定义商品级别的退货设置，并优先于类别级别的退货规则。本指南将介绍如何设置退货规则。如果您需要为危险品、货运或定制商品设置退货豁免，需要先向 Walmart 请求批准。

## 创建退货规则

### 步骤 1 — 开始

导航到 Seller Center 中的 [**Catalog**](https://seller.walmart.com/catalog/list-items)。选择 **Update items** 按钮，从下拉菜单中选择 **Update with file**。

### 步骤 2 — 下载并填写模板

选择 **Change return rules** 链接，将模板保存到您的计算机。然后打开模板并更新。

### 步骤 3 — 上传模板

导航回您的 [**Catalog**](https://seller.walmart.com/catalog/list-items)，选择 **Update items** 按钮并选择 **Update with file**。然后使用文件浏览器窗口上传模板。准备好后，选择 **Submit**。

**注意**：您有责任确保退货规则准确无误。因任何错误应用退货设置而产生的所有费用由您承担。

## 常见问题

**退货时间线是什么？**

您必须支持至少 30 天的退货窗口，从商品发货日期后 7 天开始，有例外情况。

**如何下载退货规则报告？**

导航到 Seller Center 中的 [**Reports**](https://seller.walmart.com/analytics/reports) 仪表板，找到 **On Request Reports** 部分，选择 **Report Type** 下拉菜单。选择 **Return Overrides Report** 并选择 **Request report** 按钮。要下载报告，导航到 **Report activity** 部分，找到您的报告并选择行末的下载图标。
