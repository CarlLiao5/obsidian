# Performance Dashboard: Negative Feedback Rate

> 绩效仪表板：负面反馈率

## 概述
负面反馈率是追踪你在 Walmart.com 上收到的一星或二星评级的订单百分比的卖家绩效标准。

## 核心要点

### 政策说明

负面反馈率是在过去 60 天内收到一星或二星评级的订单百分比。如果订单同时收到负面商品评级和负面卖家评级，Walmart 将取两者中较低的评级。

所有 Marketplace 卖家必须保持负面反馈率 **≤ 2%**。

### 绩效驱动因素表格

| 驱动因素 | 定义 | 建议 |
| --- | --- | --- |
| Assembly or instructions | 投诉关于不清楚、缺失或令人困惑的组装说明 | 提供清晰、分步骤的打印和在线说明 |
| Cancellation | 未经客户同意取消订单或客户取消订单困难 | 保持库存水平更新，及时沟通 |
| Carrier | 关于承运商的投诉 | 使用可靠承运商，确保包裹正确扫描 |
| Damaged | 商品到达时破损、泄漏或明显损坏 | 使用坚固、保护性包装并测试 |
| Defective | 商品功能不正常 | 定期检查库存，与制造商合作改进 |
| Incorrect item | 交付了错误的产品、尺寸或变体 | 验证产品列表准确性 |
| Invalid tracking | 跟踪号缺失、无效或未更新 | 始终提供有效跟踪 ID |
| Item not as described | 产品描述、图片与客户收到的不匹配 | 保持准确、详细的列表 |
| Late | 商品在承诺的配送窗口外送达 | 监控承运商绩效，设置合理的配送窗口 |
| Lost | 包裹从未送达 | 提供准确的跟踪信息，快速解决包裹索赔 |
| Missing item | 订单缺少零件、配件或产品 | 进行履约审计，使用检查清单确认订单完整 |

## 常见问题

**在哪里查看负面反馈率？**
在 Seller Center 中，导航到 **Performance** 仪表板，在 **Performance Standards** 部分找到你的比率。

**如果我不认同收到的客户反馈怎么办？**
如果你认为客户反馈不准确或不公平，可以联系 Support 并附上订单详情和支持文档。
