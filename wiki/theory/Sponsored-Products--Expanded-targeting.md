# 扩展定向 (Expanded Targeting)

> 通过品牌词定向（Brand Term Targeting）和互补品定向（Complementary Targeting）扩大广告覆盖范围。

## Brand Term Targeting（品牌词定向）

帮助品牌提高在沃尔玛平台内的可见度，支持：
- **品牌防御**：对自家品牌词出价（自动活动默认开启）
- **品牌进攻**：对竞品品牌词出价（需主动启用）

### 适用活动类型和位置

| 广告类型 | 品牌词定向 | 关键词匹配 |
|---------|----------|-----------|
| Sponsored Products | ✅ 自动 + 手动（仅精确匹配） | Exact（精确匹配）|
| Sponsored Brands | ✅ | Exact |
| Sponsored Videos | ✅ | Exact |

| 展示位置 | 支持品牌词定向 |
|---------|-------------|
| Search In-grid | ✅ |
| 商品详情页轮播 | ❌ |
| 其他位置 | 部分支持 |

### 启用方式

| 活动类型 | 操作 |
|---------|------|
| **现有自动活动** | 勾选复选框，Walmart 自动决定品牌词定向 |
| **新自动活动** | 默认自动开启 |
| **手动精确匹配活动** | 直接对品牌词出价即可，无需复选框 |

### 沃尔玛私有品牌限制

沃尔玛自有品牌（Private Labels）目前不参与品牌词定向：
- 不能对其他广告主的品牌词出价
- 其他广告主不能对沃尔玛私有品牌词出价

## Complementary Targeting（互补品定向）

允许 Walmart Connect 向正在搜索**相关商品**的顾客展示广告。商品以轮播形式展示在相关商品详情页。

**示例：** 顾客搜索咖啡过滤器，互补品（如咖啡粉）广告可出现在该商品页面的轮播位。

### 适用条件

- 仅限**自动活动**
- 新活动默认开启
- 现有活动可在活动编辑页面勾选启用

### 配套定向功能

互补品定向也支持 **Brand Term Targeting**，形成完整扩展定向方案。

## 两种扩展定向对比

| | 品牌词定向 | 互补品定向 |
|---|---|---|
| 适用活动类型 | 自动 + 手动（精确匹配） | 仅自动 |
| 展示位置 | Search In-grid（主要） | 商品详情页轮播 |
| 用途 | 品牌防御/进攻 | 扩大相关品类覆盖 |
| API 支持 | ✅ | ✅ |
