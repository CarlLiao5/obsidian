# Content Quality Guide 内容质量指南

> 高质量商品 listing 是广告效率的基础。通过优化商品内容质量提升可发现度、增加流量并获得回头客。

## 内容质量的核心价值

- **改善广告效率**：高质量 listing 提升广告点击率和转化率
- **提升可发现度**：精准的分类和关键词使商品更容易被搜索到
- **增加自然流量**：减少对付费广告的依赖
- **获取回头客**：完善的详情页建立信任，提高复购率

## 商品详情页应包含的要素

| 要素 | 说明 |
|------|------|
| **详细产品描述** | 完整、准确、用要点突出核心卖点 |
| **高质量图片** | 多角度展示，含颜色变体和尺寸参考 |
| **评分与评论** | 积累正面评价，建立社会证明 |
| **有竞争力的定价** | 定期监控竞争对手价格，使用 [[theory/Catalog_management--Price_management|定价工具]] |
| **盈利性产品组合** | 确保利润空间，参考 补货决策方法论 |
| **可用的变体选项** | 提供颜色、尺寸等变体选择 |

## 提升相关性的操作

1. **定期优化标题和描述**：持续更新关键词和描述内容
2. **在描述中嵌入关键词**：自然融入搜索词，提升搜索匹配度
3. **确认商品分类正确**：使商品在合适的类目中展示
4. **使用关键词报告优化**：定期查看 [[theory/All-Keywords|关键词报告]]，识别高转化关键词并补充到商品页面

## 广告与 listing 的协同

广告将顾客直接引导至商品详情页。如果页面信息不足以促成购买，广告花费就会浪费。

**核心逻辑**：
- 广告负责引流，listing 负责转化
- Listing 质量差 → 广告 CTR 高但转化率低 → ACOS 居高不下
- Listing 质量好 + 广告精准 → 低 ACOS + 高 ROAS

**建议**：每月至少审查一次主要推广商品的 listing 质量，结合广告数据识别需要优先优化的商品。
