# 非管理员用户访问管理

> 非管理员用户如何申请或升级对组织和账户的访问权限，查看请求状态。

## 申请新账户访问权限

1. 点击右上角 **Account** 图标 → **Manage profile and account**
2. 在 *Manage your account access* 中搜索供应商/卖家/组织名称
3. 从下拉列表中选择账户，点击 **Save**
4. 点击 **Request access**
5. 输入申请原因
6. 选择访问级别（Standard 或 Admin），如选 Standard 需设置各组件权限
7. 点击 **Request access**

请求自动发送给账户管理员、Org Admin 或 Super Admin 审核。批准/修改/拒绝后你会收到邮件通知。

## 申请升级访问权限

1. 导航到 Access 页面的 **Manage users** 部分
2. 搜索自己的名字，点击 **铅笔** 图标
3. 将角色从 **Account user** 改为 **Organization user**
4. 为各组件设置 Edit / View / None
5. 点击 **Update access**

请求自动发送给 Org Admin 或 Super Admin 审核。

## 申请状态

| 状态 | 含义 |
|------|------|
| **Pending** | 等待管理员/Super Admin 审核 |
| **Approved** | 已按请求授予访问权限 |
| **Approved with modifications** | 已授予但有部分修改 |
| **Denied** | 访问被拒绝（附原因）或被直接覆盖 |
| **Voided** | 请求被取消或被新请求覆盖 |
| **Auto-approved** | 访问降级或移除请求自动完成 |

## 跟进待处理请求

如果请求一周内未获审核：
1. 在 **Existing Organizations** → **Requests** 中选择
2. 点击 **Requests I made**
3. 在 Actions 列点击 **三个点** → **Create support ticket**
