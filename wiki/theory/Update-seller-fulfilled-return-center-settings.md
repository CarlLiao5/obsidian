# Update seller-fulfilled return center settings

> 添加和管理退货中心

## 概述

您的账户必须提供至少一个退货中心。添加或更新退货中心设置之前，请查看退货政策以确保您的首选设置符合 Walmart 标准。按照以下步骤更新您的退货中心设置。

## 添加新的退货中心

### 步骤 1 — 开始

导航到 **Shipping Profile** 中的 [**Returns**](https://seller.walmart.com/settings/shipping-profile/returns) 页面。

**注意**：您指定的退货设置将默认适用于您目录中的所有商品。如果您有不应包含的商品（例如已批准退货豁免的危险品或货运商品），必须创建商品级别退货规则。

### 步骤 2 — 添加新的退货中心

在 [**Returns**](https://seller.walmart.com/settings/shipping-profile/returns) 页面上，导航到 **Return Centers (Required)** 并选择 **Add Return Center**。输入所需详情并选择 **Add Return Center**。

**注意**：Walmart 保留审核 Marketplace 卖家提供的退货中心地址的权利，以进行数据质量审核，并将卖家提交的退货地址标准化为适当的承运人格式。未能提供有效的美国退货地址可能导致 Walmart Marketplace 暂停或终止。对于因卖家在 Seller Center 中提供的不完整或无效地址而导致物品被重新路由到 Walmart 退货中心的情况，Walmart 保留向 Marketplace 卖家收取失败配送费用的权利。

## 常见问题

**如何为商品选择退货中心？**

退货中心根据客户位置选择。您可以通过更新商品级别退货规则来指定退货地点。

**我可以编辑现有的退货中心吗？**

是的。在 [**Returns**](https://seller.walmart.com/settings/shipping-profile/returns) 页面上，导航到 **Return Centers (Required)** 部分并找到要编辑的退货中心。选择 _Actions_ 列中的铅笔图标进行编辑。

**我需要提供美国的退货地址吗？**

是的。您的退货中心地址必须位于美国，不能位于夏威夷、阿拉斯加或美国领土（包括美属萨摩亚、关岛、北马里亚纳群岛、波多黎各和美属维尔京群岛）。

**我的 PO Box 可以作为退货地址吗？**

不可以。
