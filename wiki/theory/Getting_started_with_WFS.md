# WFS 入门指南（Getting Started with WFS）

> 从零开始使用 Walmart Fulfillment Services 的完整指南，包括设置、配置和运营要点。

## 概述

Getting Started with WFS 模块为新卖家提供从申请到日常运营的完整指引，帮助你快速上手 WFS 仓储配送服务。

### 入门步骤

|| 步骤 | 说明 |
||-----|------|
| **1. 账户设置** | 在 Seller Center 中完成 WFS 申请和配置 |
| **2. 联系人信息** | 设置 WFS 相关的联系人 |
| **3. 账单信息** | 配置 WFS 费用支付方式 |
| **4. 退货规则** | 设置 WFS 商品的退货政策 |
| **5. 选择商品** | 选择要使用 WFS 的 SKU |
| **6. 创建入库计划** | 准备配送入库到 WFS 仓库 |

### 核心配置项

| 配置项 | 说明 | 重要性 |
|-------|------|-------|
| **联系人信息** | WFS 相关通知联系人 | 必须 |
| **账单信息** | WFS 仓储和履约费用支付 | 必须 |
| **退货规则** | 客户退货处理方式 | 必须 |
| **商品选择** | 哪些 SKU 使用 WFS | 必须 |
| **首选承运商** | 发货到 WFS 的折扣运费 | 推荐 |

### 重要注意事项

- **WFS 库存与自配送库存独立**：两种履约方式的库存分开管理
- **WFS 优先原则**：当商品同时有 WFS 和自配送库存时，优先使用 WFS 库存
- **绩效要求**：使用 WFS 不影响卖家绩效，但如果切换到自配送需确保绩效达标
- **API 支持**：可使用 Walmart APIs 自动化 WFS 商品管理和入库流程
