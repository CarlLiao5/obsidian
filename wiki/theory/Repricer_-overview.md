# 智能调价（Repricer）概述

> Walmart 提供的自动化价格调整工具，帮助卖家在竞争中保持价格优势。

## 概述

Repricer 是 Seller Center 内置的自动调价工具，可根据市场环境（竞争对手价格、外部价格）自动调整商品价格，以维持 Buy Box 赢率或达成其他策略目标。

## 核心功能

- **基于 Buy Box 的调价**：当竞争对手赢得 Buy Box 时自动调价夺回
- **基于外部价格的调价**：参照其他电商平台的同类商品价格调整
- **价格下限保护**：防止价格调得过低保护利润率
- **规则灵活**：可为不同商品组设置不同策略

## 创建调价策略

1. 选择目标商品或商品组
2. 设置调价触发条件（Buy Box 丢失、外部价格变化）
3. 设置调价幅度（固定金额或百分比）
4. 设置价格下限（最低价格）
5. 激活策略

## 使用场景

- **夺回 Buy Box**：竞争对手价格更低导致失去 Buy Box 时自动降价
- **保护利润率**：市场低价低于成本时停止调价
- **主动竞争**：持续监控外部价格，保持价格竞争力
