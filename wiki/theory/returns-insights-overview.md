# Returns insights: Overview

> 查看退货数据洞察，包括退货原因和退款金额

## 概述

[**Returns insights**](https://seller.walmart.com/orders/returns) 仪表板提供退货的历史数据，包括退货原因和退款金额。本指南将介绍这些洞察以及如何访问它们。

## 工作原理

仪表板包括所有退货商品的 SKU 级别详情以及其他相关数据列字段。要访问洞察，在 Seller Center 中选择 **Orders** 下的 **Returns**，然后选择 **View returns insights** 按钮。

仪表板还允许您查看退货原因或识别缺陷商品。选择 **Returns reasons** 列中的带下划线的数字以查看客户退货的原因。以下是可以跟踪的其他指标：

| 绩效指标 | 定义 |
| --- | --- |
| **Total returns (orders)** | 客户退货订单总数 |
| **Refund rate** | 因卖家原因（如商品损坏或收到错误商品）退款的订单百分比。必须保持 < 6% 的比率 |
| **Refund gross merchandise value (GMV)** | 退款的 GMV 总计 |
| **Returns processing fee** | 处理退货支付的总费用 |
| **Units** | 日期范围内的退货单位数 |

**注意**：[**Returns insights**](https://seller.walmart.com/orders/returns) 仪表板上显示的退款率可能与用于计算您的卖家绩效指标的总体退款率不同。

## 常见问题

**我的指标多久更新一次？**

数据每周汇总并在周六展示。

**为什么我的退款率与绩效仪表板上显示的退款率不同？**

根据您选择的日期范围，**Returns insight** 仪表板上显示的退款率可能与用于计算您的卖家绩效指标的总体退款率不同。

**如果我处理 WFS 退货会发生什么？**

如果您使用 WFS，可能会收取退货处理费。

**我应该多久处理一次退货？**

退货应在 48 小时内处理。

**如果我想 dispute 退货该怎么办？**

如果您符合条件，可以 dispute 退货。
