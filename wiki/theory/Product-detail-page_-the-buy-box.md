# Buy Box（购物车）机制

> Walmart 商品页面上赢得优先展示位置的核心机制，是 SEM 和 Sponsored Search 广告资格的前置条件。

## 概述

Walmart Marketplace 上同一商品可能有多个卖家报价。Buy Box 是商品详情页上"添加到购物车"按钮的位置——赢得 Buy Box 的卖家的商品将获得最高可见性和转化率。

**赢得 Buy Box 的意义：**
- 商品获得最高展示位置
- 显著提升销售转化率
- 是 [[theory/Advertise-with-Walmart-Connect-sponsored-search|Sponsored Products]] 广告的必要前提
- 不影响 [[theory/Advertising--SEM|SEM]] 广告（在 Google 上独立展示）

## Buy Box 归属规则

归属基于综合绩效考量，包括：
- 商品价格（竞争力）
- 库存可用性
- 配送速度（Prime/WFS 优先）
- 卖家绩效评分
- 退货率
- 客户服务历史

## Buy Box 与广告的关系

| 广告类型 | Buy Box 要求 |
|----------|--------------|
| **Sponsored Products** | 必须赢得 Buy Box |
| **Sponsored Brands/Videos** | 必须赢得 Buy Box |
| **SEM（Google）** | 无需 Buy Box |
| **Onsite Display** | 无需 Buy Box |

## 常见问题

**Q：失去 Buy Box 后，广告会自动暂停吗？**  
A：SEM 广告不会暂停，会继续在 Google 上展示。Sponsored Products 会停止展示。

**Q：如何提升 Buy Box 赢率？**  
A：优化价格、确保库存充足、提供快速配送（建议 WFS）、维护良好的绩效评分。

**Q：所有商品都有 Buy Box 吗？**  
A：是的，同一商品有多个报价时，最具竞争力的报价赢得 Buy Box。
