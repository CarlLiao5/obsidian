# Tax classifications and documentation

> 了解您的税务分类和所需文件

## 概述

如果您打算在美国 Marketplace 上销售，您需要提供营业执照号或营业许可证号，并根据您的公司注册国提供支持文件以确认您的公司凭证。

## 税务分类

### W-9

如果您是在美国注册的卖家，拥有雇主识别号 (EIN) 且公司注册国在美国，则您的税务分类为 W-9。

### W-8ECI

如果您拥有 EIN 且公司注册国不在美国，则您的税务分类为 W-8ECI。

### W-8BEN-E

如果您没有 EIN 且公司注册国不在美国，则您的税务分类为 W-8BEN-E。

### W-8BEN

仅当您是墨西哥或印度的独资经营者时，我们才支持此税务分类。

## 各国家所需文件

| 注册国 | 所需文件 |
| --- | --- |

| 智利 | 智利政府颁发的 RUT 证书、公司银行对账单 |
| 中国 | 商业执照、公司公章、授权书、法人代表身份证复印件 |
| 德国 | 商业注册证书、实用账单或银行对账单 |
| 香港 | 公司注册证书、商业登记条例、企业年度申报表 (NAR1/NC1) |
| 印度 | 地址证明、GST 注册证书 |
| 日本 | 所有事项证书、实用账单或银行对账单 |
| 墨西哥 | 税务识别卡 (Cédula de Identificación Fiscal)、公司章程 |
| 新加坡 | 注册证书或成立证书、实用账单或银行对账单 |
| 韩国 | 国家税务局颁发的商业注册证书 |
| 台湾 | 公司注册证书、实用账单或银行对账单、法人代表身份证复印件 |
| 泰国 | 泰国纳税人识别号 (TH-TIN)、实用账单或银行对账单 |
| 土耳其 | 贸易登记证书、税务登记证书 |
| 英国 | 成立证书、实用账单或银行对账单 |
| 越南 | 企业注册证书、实用账单或银行对账单 |

## 常见问题

**业务验证需要多长时间？**

业务验证可能需要几分钟到几个工作日不等。

**我的 W-9 有效期多久？**

W-9 不会过期。W-8ECI 和 W-8BEN-E 文件需要每三年更新一次。
