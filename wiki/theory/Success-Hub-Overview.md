# 成功中心（Success Hub）概述

> 每日更新的个性化增长建议平台，涵盖调价、补货、AI 文案、促销等各类卖家运营优化场景。

## 概述

Success Hub 是 Seller Center 内置的智能增长建议引擎，每日刷新，为卖家提供个性化、可操作的业务优化建议。路径：Seller Center → Growth → Success Hub。

**建议类型覆盖：**

| 建议类型 | 说明 |
|----------|------|
| Match Buy Box price | 自动匹配当前 Walmart 最低价 |
| Match external price | 匹配其他电商平台价格 |
| Automate pricing with Repricer | 用 Repricer 自动化调价 |
| Simplified Shipping Settings | 简化配送设置，加速履约 |
| Fulfill with Walmart | 将自配送转为 WFS |
| Edit listings with Gen AI | AI 生成文案优化（标题/描述/关键特性） |
| Add customer favorites | 发现并添加流行商品到目录 |
| Optimize Customer Favorites | 优化客户最爱商品 |
| Restock seller-fulfilled inventory | 自配送缺货补货提醒 |
| Restock WFS items | WFS 库存补货建议 |
| Get more reviews with incentives | 评论激励计划 |
| Reach customers with SEM | 推荐加入 SEM 活动的商品 |
| Enroll items in Flash Deals | 推荐加入周度 Flash Deals 的商品 |

## 关键提示

> **优先实施 Content Quality 建议（Top items to optimize 区域），这对提升曝光和转化效果最显著。**

## 项目激励

使用 Success Hub 的卖家有机会获得项目激励，如广告积分或账单信用。符合资格时会收到邮件通知。

## 使用条款

- Success Hub 所有建议均为参考性质，卖家无义务采纳
- 建议基于预测和历史购买数据，不构成 Walmart 的约束性承诺
- 部分服务（如 WFS、SEM）涉及额外费用，接受建议即表示同意付费
