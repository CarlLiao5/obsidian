# Resold Program: Item Setup

> 为 Resold 计划设置卖家配送商品

## 概述

获批准在 Resold 计划中列出商品并同意相关条款和条件后，下一步是在 Marketplace 目录中设置这些商品。本指南介绍为 Resold 计划设置卖家配送商品的流程。

**参与资格：** Resold 计划目前仅限受邀卖家。如果想通过 Resold 计划销售二手商品，请访问 Seller Center 的 [**Exclusives programs**](https://seller.walmart.com/growth/exclusive-programs) 页面申请。

## 设置要求

### 商品属性要求

| 属性 | 要求 |
| --- | --- |
| **Product ID** | Restored：所有符合条件的恢复商品的产品 ID 必须与原始产品 ID 不同。Product ID 和 Restored ID 在恢复商品设置期间都是必填的且应该相同。<br><br>Pre-Owned：不同状况的商品的产品 ID 可以与原始产品 ID 相同 |
| **SKU** | SKU 应在不同的商品状况中唯一。不能在不同的商品状况中使用相同的 SKU |
| **Model Number** | 如果产品有型号，请为恢复的商品使用与新商品不同的型号 |
| **Product Title** | Walmart 会在标题开头添加商品状况。如果更改这些添加，您的商品将被取消发布 |
| **Condition** | 根据磨损程度、之前使用（如有）、外观和整体质量选择[状况类型](Item-condition-types) |
| **Image Requirements** | 以下状况需要展示商品状况的产品图片：Remanufactured、Pre-Owned: Like New、Pre-Owned: Good、Pre-Owned: Fair、New with defects |
| **Warranty** | Restored Premium：必须在 **Warranty Information** 字段中包含规定的文本<br><br>Pre-Owned：除非法律另有要求，否则不需要保修 |

### 设置方法

可以使用任何[商品设置方法](Item_setup--Overview)设置 Resold 商品。两种主要方法：

| 方法 | 描述 |
| --- | --- |
| **Express setup** | 搜索结果仅显示新状况版本<br><br>Pre-Owned：使用原始产品 ID 并选择状况值进行设置<br><br>Restored：搜索正确状况的恢复商品或使用现有恢复商品中的 GTIN |
| **Full setup** | 模板按要求级别分组<br><br>Pre-Owned：可使用新等效 GTIN 作为产品 ID<br><br>Restored：Product ID 和 Restored ID 必须相同 |

## 常见问题

**如何找到新等效或原始产品标识符？**
在 Walmart.com 上查看商品，复制 URL 末尾斜杠后的 9 位数字。

**Walmart 配送 (WFS) 商品符合 Resold 计划条件吗？**
符合。您将使用原始 GTIN 设置商品，但我们会生成自定义配送 GTIN 以在 Walmart 网络中跟踪您的 Resold 商品。

**哪些产品类型符合 Resold 计划的列出条件？**
如果您是批准卖家，您将有资格设置批准产品类型。如果商品不符合条件，您会收到错误。

**Resold 商品符合 Sponsored Search 广告活动条件吗？**
是的，但以下产品类型除外：调制解调器、杂志、汽车雨刷、VCR、 rotary hammers、对讲机、pole saws、印刷机、CB 收音机和扫描仪、wet/dry 吸尘器、加湿器、汽车扬声器、普拉提机器、灶台、手持草坪修剪机和逆变器发电机。
