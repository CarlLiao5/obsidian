# Duplicate Listings Policy

> 防止和纠正重复列表的政策

## 概述

如果 Marketplace 商品由多个卖家提供且共享相同的[产品标识符](Choose-a-product-identifier)（或产品 ID），商品将被合并为一个列表，以帮助为客户创建更好的导航体验。如果您为 Walmart.com 上已有商品提交了不正确的产品 ID，可能导致重复列表。

## 政策内容

创建重复列表违反 Walmart 政策，重复列表将从 Walmart.com 取消发布。处于未发布状态超过 90 天的商品将从 Walmart 目录中停用。停用后，商品将无法在 Walmart 目录中搜索或使用。

不遵守 Marketplace 政策可能导致列表移除、账户暂停或账户终止。

## 常见问题

**如何找到我的重复列表？**
在 Seller Center 中导航到您的 [**Catalog**](https://seller.walmart.com/catalog/list-items) 的 **Unpublished** 标签。配置表格以通过选择 **Filters** 按钮包含重复列表。接下来，选择 **Status change reason** 部分并勾选 **Potential duplicate item**，然后选择 **Apply**。

**如何纠正重复列表？**
可以通过在 Seller Center 中[更新商品的产品 ID](Update-a-product-ID) 并重新提交更新的商品数据来纠正重复列表。

**为什么不允许重复列表？**
重复列表会使 Walmart.com 上的搜索和浏览结果变得混乱，使客户更难找到他们想要的商品。此外，搜索引擎优化 (SEO) 依赖于准确的产品 ID。如果商品没有使用正确的产品 ID 设置，它在 Google 搜索中的表现可能不佳。

**如何避免创建重复列表？**
在分配产品标识符给商品时注意以下几点。某些具有不同变体属性（如数量、尺寸、颜色或口味）的商品应有不同的产品 ID（例如，一包六盒纸巾的 产品 ID 应与一包四盒的不同）。如果您确认您的特定变体没有产品 ID，可以购买一个新的。不要使用不同的产品 ID 为同一商品生成单独的列表来满足您的供应链或其他内部需求。

**在商品包装上找不到条形码该怎么办？**
在制造商网站或其他在线来源搜索产品 ID 验证。如果仍然找不到，直接联系制造商确认正确的产品 ID。

**应该为商品购买产品 ID 吗？**
是的。如果您是制造商或品牌所有者，应该始终为商品购买产品 ID。购买产品 ID 前，检查商品包装以验证您的商品是否已有产品 ID。

**认为我的商品列表被错误移除该怎么办？**
如果您认为商品被错误取消发布或有疑问，请在 Seller Center 菜单栏中选择 **Help** 按钮联系 Support。

**如果我的重复列表没有采取行动会怎样？**
保持未发布状态至少 90 天的列表将被 Walmart 停用。
