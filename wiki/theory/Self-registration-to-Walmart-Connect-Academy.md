# Walmart Connect Academy 自助注册

## 注册入口

Walmart Connect Academy 提供免费的自助注册渠道，任何人都可以注册学习广告认证课程。

### 注册链接

**官方注册页面**：
```
https://connectacademy.walmartlearning.com/student/authentication/register
```

该链接可永久使用，用于新用户注册和现有用户登录。

---

## 注册步骤

### 第 1 步：访问注册页面

打开浏览器，访问上方链接地址。

### 第 2 步：填写注册信息

在注册页面，需要填写以下信息：

| 字段 | 说明 | 要求 |
|------|------|------|
| **姓名** | First Name（名）+ Last Name（姓） | 与工作证照一致 |
| **邮箱** | 用于登录和接收通知 | 建议使用公司邮箱 |
| **密码** | 账户密码 | 至少 8 位，包含字母和数字 |
| **确认密码** | 再次输入密码 | 与密码字段一致 |
| **国家/地区** | 选择所在国家 | 通常选择美国 |
| **公司名称** | 雇主名称（可选） | 可填 Walmart 合作伙伴或卖家公司 |
| **工作角色** | 职业角色选择 | 可选 Seller、Agency、Media Buyer 等 |

### 第 3 步：同意条款

勾选以下协议：

- [x] I agree to the Terms of Use（同意使用条款）
- [x] I agree to the Privacy Policy（同意隐私政策）
- [x] I consent to receive emails（同意接收邮件通知）

### 第 4 步：提交注册

点击 **"Create Account"（创建账户）** 或 **"Register"（注册）** 按钮。

### 第 5 步：验证邮箱

1. 检查注册邮箱的收件箱
2. 找到来自 Walmart Connect Academy 的验证邮件
3. 点击邮件中的验证链接
4. 邮箱验证完成

## 常见问题

### Q1：注册失败怎么办？

| 错误原因 | 解决方法 |
|----------|----------|
| 邮箱已被注册 | 使用其他邮箱，或点击"Forgot Password"找回密码 |
| 密码不符合要求 | 使用 8 位以上，包含大小写字母和数字 |
| 网络问题 | 更换网络环境后重试 |

### Q2：收不到验证邮件？

| 可能原因 | 建议操作 |
|----------|----------|
| 进入垃圾邮件 | 检查垃圾邮件文件夹 |
| 邮箱服务商延迟 | 等待 10-15 分钟再试 |
| 邮箱地址错误 | 检查拼写后重新注册 |

### Q3：用个人邮箱还是公司邮箱？

| 邮箱类型 | 优点 | 缺点 |
|----------|------|------|
| **公司邮箱** | 专业、便于认证、与工作关联 | 离职后可能无法访问 |
| **个人邮箱** | 永久使用、易于管理 | 在 LinkedIn 展示时可能不够正式 |

**建议**：优先使用公司邮箱进行注册，以便与工作身份关联。
