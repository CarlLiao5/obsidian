# 类别特定要求

## 概述

某些产品类别在 Walmart Connect 投放广告时需要遵守额外的规定。这些要求旨在确保特定敏感类别广告的合规性，保护消费者权益，并遵守相关法律法规。

---

## 受监管类别概览

| 类别 | 监管程度 | 主要要求 |
|------|----------|----------|
| 医疗健康 | 高 | 资质认证、健康声明限制 |
| 金融服务 | 高 | 披露要求、合规声明 |
| 酒精饮品 | 高 | 年龄限制、品牌规范 |
| 娱乐媒体 | 中 | 内容分级、版权合规 |
| 社交媒体 | 中 | 透明度、代言规范 |

---

## 医疗健康类

### 适用商品

- 非处方药（OTC）
- 保健品与膳食补充剂
- 医疗器材
- 个人护理产品
- 健康监测设备

### 广告要求

| 要求类型 | 具体规定 |
|----------|----------|
| **声明限制** | 不得声称能诊断、治愈、治疗或预防任何疾病 |
| **功效声称** | 仅可描述产品的一般功能和用途 |
| **安全声明** | 不得暗示产品完全安全（"完全无害"等） |
| **资质要求** | 部分产品需要提供相关资质证明 |
| **免责声明** | 必须包含适当的健康提示 |

### 禁用表述示例

- "治愈糖尿病"
- "彻底治疗失眠"
- "完全替代医生诊断"
- "永久解决..."等绝对性表述

---

## 金融服务类

### 适用商品

- 信用卡与借贷服务
- 投资理财 products
- 保险产品
- 汇款与转账服务
- 信用修复服务

### 广告要求

| 要求类型 | 具体规定 |
|----------|----------|
| **利率披露** | 必须清晰展示年化利率（APR） |
| **费用说明** | 必须披露所有相关费用 |
| **风险提示** | 投资产品必须包含投资风险声明 |
| **合规声明** | 必须包含必要的法律免责声明 |
| **比较限制** | 不得贬低竞争对手 |

### 必须包含的声明

```
[产品名称]由[机构名称]提供。
详细信息请参阅条款和条件。
利率、费用和条款可能因信用评分而异。
[投资产品]：[风险提示声明]
```

## 酒精饮品类

### 广告限制

| 限制类型 | 具体规定 |
|----------|----------|
| **受众定位** | 必须定向至 21 岁及以上用户 |
| **内容限制** | 不得暗示饮酒具有社交或性吸引力 |
| **目标群体** | 不得针对未成年人或酗酒者 |
| **竞品比较** | 不得直接贬低其他酒精品牌 |
| **消费量** | 不得暗示大量饮酒是可接受的 |

### 品牌规范

- 不得暗示酒精能改善社交、职业或性生活
- 不得将饮酒与驾驶、操作机械关联
- 不得在体育赛事赞助广告中强调酒精元素
- 不得使用卡通人物或年轻人喜欢的形象

---

## 娱乐与媒体类

### 内容分级

| 内容类型 | 广告要求 |
|----------|----------|
| **PG-13 内容** | 需要添加适当的年龄提示 |
| **R 级内容** | 必须定向至成熟受众 |
| **赌博相关** | 必须获得事前批准 |
| **成人内容** | 严禁在 Walmart Connect 投放 |

### 版权合规

- 广告素材必须拥有完整版权授权
- 不得使用未经授权的音乐、影视片段
- 不得模仿知名品牌或版权角色的外观
- 引用名人形象需提供授权证明

---

## 社交媒体与影响者

### 代言透明度

根据 FTC 指南要求：

| 要求 | 说明 |
|------|------|
| **明确披露** | 赞助内容必须明确标注"广告"或"赞助" |
| **平台合规** | 必须符合各平台的代言披露政策 |
| **真实性** | 代言内容必须反映真实使用体验 |
| **禁止虚假** | 不得伪造用户评价或推荐 |

### 影响者合作

- 确保影响者了解 Walmart Connect 广告政策
- 提供清晰的广告内容指南
- 保留合作记录以备审计
