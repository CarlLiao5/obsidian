# 最终结算支付政策（Final Settlement Payout Policy）

> 账户关闭或终止后，最终余额的结算政策与扣款规则。

## 概述

最终结算支付（Final Settlement Payout）是指账户关闭或终止后，卖家应收款项的最终结清金额——即扣除退款、拒付和其他应付款项后的净额。

**到账窗口：** 通常在账户关闭后约 **180 天** 内完成。

## Walmart 可采取的行动

| 行动 | 说明 |
|------|------|
| 资金冻结/保留 | 作为风险控制措施，限制交易或发放 |
| 扣除应付款项 | 退款、拒付、客户调整、费用、罚款、税费、WFS 费用等；余额不足仍需补缴差额 |
| **永久扣押** | 欺诈、欺骗性或非法活动、重复违规——永久扣留支付 |
| WFS 库存留置权 | 终止时可要求移除库存（费用自理）或处置库存并用收益抵债 |
| 放弃物品处理 | 无效退货地址产生的物品按放弃处理，并收取相关费用 |
| 税务扣缴 | 按法律规定代扣税款 |

## 关键规则

- 支付仅在账户关闭、所有退货/拒付/调查解决后发放
- 不保证固定发放日期
- 沃尔玛将通过 Seller Center 通知冻结或永久扣押决定
- 可提交信息供审查，但最终决定权归沃尔玛

## 适用范围

本政策适用于：
- 现有账户及未来终止
- 追溯账户（以本政策和综合零售商协议的条款和证据为准）
