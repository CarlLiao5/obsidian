# 清理老化库存（Move Aged Inventory）

> 避免 WFS 长期仓储费（超过 365 天）的策略，以及清理或优化老化库存的方法。

## 概述

**老化库存（Aged Inventory）**：在 WFS 仓库中存储超过 **365 天**的库存。自第 366 天起，将被收取**长期仓储费**。

## 如何识别老化库存

| 位置 | 说明 |
|------|------|
| WFS Overview 页面 | 右上角显示老化库存卡片 |
| Inventory 页面 | 顶部显示，或查看 **Surplus and aged** 标签页的 Aged units 列 |
| Inventory Health 报告 | 下载后查看 **ATS 365+ days** 列 |
| **提前预警** | **ATS 271–365 days** 列显示即将产生费用的库存 |

## 优化建议（Optimize）

从 Inventory 页面的 **Surplus and aged** 标签页，点击 **Optimize** 按钮，可使用以下优化选项：

| 优化方式 | 说明 |
|----------|------|
| **Pricing update** | 查看当前 Walmart 价、竞品价格、Buy Box 价格，直接调价 |
| **Listing quality** | 优化内容、图片、关键词 |
| **SEM** | 通过搜索广告提升曝光 |
| **Multichannel Solutions** | 通过多渠道分销 |

## 移除库存（Remove Inventory）

如无法销售，提交库存移除请求，避免持续产生仓储费。
