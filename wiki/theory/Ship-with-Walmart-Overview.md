# Ship with Walmart: Overview

> 使用 Walmart 配送：概述

## 概述
如果你自己履约 Walmart 订单，使用 Ship with Walmart (SWW) 可以从值得信赖的国内和国际承运商获得折扣运费，同时保护你的账户免受配送相关问题的影响。

## 核心要点

### 核心功能

| 设置 | 说明 |
| --- | --- |
| **Default print settings** | 配置默认打印设置应用于你所有订单的未来标签打印设置 |
| **Custom packages** | 设置默认包裹尺寸并根据你输入的包裹尺寸和重量接收估计运费 |
| **Shipping labels** | 通过 Walmart 购买折扣运费标签 |
| **Seller protections** | Ship with Walmart 为你提供卖家保护，用于超出你控制范围的配送问题 |
| **Signature on delivery** | FedEx 和 UPS 货运要求在交付时签名 |
| **Bring your own account** | 通过 FedEx 设置你自己的账户以购买标签并添加你自己的账户费率 |

### 主要优势

- 折扣运费（与市场价格相比）
- 绩效指标保护
- 美国国内货运自动保护 — 交付途中或交付后丢失的订单最高可获 $100 退款
- 可购买超过 $100 订单的额外保护，最高 $500

### 可用承运商

**美国境内**：USPS、FedEx、UPS

**从中国到美国**：Yanwen 和 SF Express（仅跨境）

**从印度到美国**：FedEx

## 常见问题

**如何注册 Ship with Walmart？**
Ship with Walmart 可供任何履约自己订单的 Walmart Marketplace 卖家使用。

1. 转到 Seller Center 中的你的账户
2. 导航到 **Shipping Profile** 并选择 **Ship with Walmart**
3. 添加或选择履约中心并选择它作为你的 **Ship from** 地址
4. 如果从中国发货，从 **Ship with Walmart Cross Border** 选项卡注册

**我可以注册后使用自己的承运商吗？**
对于美国国内运输，你可以选择使用 Ship with Walmart 或你自己的承运商。但是，一旦注册 Ship with Walmart Cross Border，从中国发货的注册商品必须通过 Ship with Walmart 运输。
