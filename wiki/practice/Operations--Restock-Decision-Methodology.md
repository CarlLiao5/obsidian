# 补货决策方法论

> 运营实践总结：补货不能仅看销量，需综合退货率、广告数据、利润率和市场热度多维度评估，避免被虚假爆款误导。

## 核心原则

**补货决策 = 销量 × 多维风险系数**

单一销量数据具有欺骗性。经验丰富的运营者在补货前，会系统评估以下维度：

1. 退货退款率
2. 广告数据健康度
3. 利润率是否达标
4. 销量增长是否可持续

## 一、退货退款率评估

**基础知识参考**：
![[theory/returns-insights-overview]]

### 判断标准

| 退货率趋势 | 信号 | 补货建议 |
|------------|------|----------|
| 持续上升 | 产品存在质量问题或描述与实物不符 | **减少补货**，优先优化 listing |
| 居高不下（高销量） | 高销量不可持续 | **谨慎补货**，先解决退货根源 |
| 稳定在合理区间 | 产品成熟度较高 | **正常补货** |

### 行动建议

- 定期查看 [[theory/returns-insights-overview|退货洞察报告]]
- 分析退货原因分类（发错货/质量差/描述不符/用户改变主意）
- 退货率突增时，优先排查 [[theory/Troubleshoot-item-setup-errors|商品设置错误]] 或描述问题

---

## 二、广告数据评估

### A. 出单词热度分析

**目的**：判断出单是否依赖短期热点

**方法**：
1. 在 Google Trends 搜索出单词
2. 观察搜索趋势曲线是否陡升陡降
3. 对比去年同期数据

**判断**：
- 搜索量突然爆发且无持续性 → 热点效应 → **减少补货**
- 搜索量稳定或稳步上升 → 真实需求 → **正常或增加补货**

### B. ACOS 健康度

**参考**：![[theory/Sponsored-Products--Campaign-bidding-strategies]]

| ACOS 区间 | 状态 | 补货建议 |
|-----------|------|----------|
| 长期高于目标 ACOS | 广告效率低 | 先优化广告，再考虑补货 |
| 波动大 | 数据不稳定 | 继续观察，暂缓补货 |
| 逐步下降 | 优化有效 | **可增加补货** |

参考：[[theory/Sponsored-Products--Campaign-bidding-strategies|广告出价策略]]

### C. 利润率底线

**核心原则**：利润率未达标的产品，补货需谨慎

**决策流程**：
1. 确认目标利润率（Target Profit Margin）
2. 计算当前利润率 = (售价 - 成本 - 广告花费 - 平台费用) / 售价
3. 利润率 < 目标 → **暂停补货**，先提升利润率
4. 利润率 ≥ 目标 → **正常补货**

## 三、警惕虚假爆款

### 识别特征

| 特征 | 可能原因 |
|------|----------|
| 单日销量突然翻倍 | 竞争对手断货/热点事件/刷单 |
| 评论数量与销量不匹配 | 短期促销或违规操作 |
| 类目排名快速上升又快速下降 | 非真实市场需求 |

### 应对策略

- **历史对比**：对比去年同期及上月数据
- **类目排名追踪**：使用 [[theory/Growth_opportunities--Search_Insights|Search Insights]] 监控
- **数据隔离**：将突发增长与日常销量分开统计

---

## 四、警惕运营赌性

### 常见心理陷阱

> "万一这个品爆了呢？"

**危害**：将资源押注在不确定性高的产品上，影响主力产品的库存充足度。

### 防赌原则

1. **主力产品优先**：断货对主力产品的伤害远大于错过一个爆款
2. **分批补货**：首次补货量不超过预期销量的 50%，验证后再追加
3. **设置止损线**：补货后连续 X 天销量低于预期，触发补货冻结

---

## 五、补货决策检查清单

补货前，逐项确认：

- [ ] 退货退款率是否在可接受范围？
- [ ] 广告出单词是否为真实需求（非热点）？
- [ ] ACOS 是否在目标范围内？
- [ ] 当前利润率是否达标？
- [ ] 销量增长是否可持续（非虚假爆款）？
- [ ] 主力产品库存是否充足？
- [ ] 是否有足够的补货预算？
