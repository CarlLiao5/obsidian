# 新品冷启动广告策略

> 新品期（1-4 周）的广告投放完整指南。重点不在 ACOS，而在积累初始数据、评论和销量。

---

## 核心认知

### 新品 vs 老品的广告差异

![[theory/Sponsored-Products--Overview#新品期]]

**沃尔玛对新品的态度**：
- 不友好。算法依赖历史数据（转化率、销售速度、WFS 库存稳定性）
- 新品没有历史数据 → 相关性评分低 → 展示位置差 → 曝光多但点击少

**你的问题诊断**：
- 曝光 80000 + 点击 143 = CTR 0.18%
- 这对新品是**预期内的**，不是异常
- 真正的问题是"花费不出去"，说明相关性评分太低

---

## 新品期正确的目标（1-4 周）

**不要看这些指标**：
- ❌ ACOS（新品期 40-60% 是正常的）
- ❌ CTR（新品期 0.2-0.5% 是正常的）
- ❌ ROAS（数据不足，无法判断）

**要看这些指标**：
- ✅ 曝光量（Impressions）：50000+ 是好信号
- ✅ 点击量（Clicks）：100+ clicks/周 是达标
- ✅ 转化量（Orders）：5-10 订单/周 是目标
- ✅ 评论数（Reviews）：+2-3 条/周 是进度
- ✅ 评分（Rating）：≥ 4.5 星 是质量

## 新品期广告投放策略

### 策略 1：自动广告 + 手动精准词

**自动广告**：
```
日预算：$30-50（不要太高）
目的：让系统自动测试关键词
重点看：哪些搜索词带来点击和转化
```

**手动广告**：
```
3-5 个核心品类词（Exact Match）
出价：平台建议的中位数
目的：精准捕获高意向用户
```

**预期**：
- 自动广告会带来大量曝光（因为匹配宽泛）
- 手动广告会带来少量但高质量的点击
- 总体 CTR 可能仍然 0.2-0.5%（这是正常的）

### 策略 2：优化商品详情页（最重要）

**新品期，Listing 质量决定一切。**

#### 标题优化

![[theory/Sponsored-Products--Campaign-setup#标题规范]]

**新品标题要求**：
- 核心品类词在前 80 字符内
- 清晰、简洁、无促销语
- 包含主要规格（吨位、材质、功能）

#### 主图优化

- 分辨率 ≥ 1500×1500
- 背景纯白
- 产品占图片 70-80%
- 无水印、无促销文字

**新品用户信任度低，主图必须一眼看清产品。**

#### 价格策略

```
新品期建议：
- 价格比竞品低 10-15%
- 或提供首单优惠
- 目的：快速积累初始订单和评论
```

#### 描述和富媒体

- 前 3 行：核心卖点
- 中间：详细规格和使用场景
- 最后：保修和售后承诺
- 4+ 张多角度图片
- 1 个产品使用视频（如果有）

## 新品期 4 周行动计划

### Week 1（第 1-7 天）

**优先级：Listing 优化**

- [ ] 优化标题（核心词靠前）
- [ ] 优化主图（清晰、专业）
- [ ] 调整价格（比竞品低 10-15%）
- [ ] 完善描述（突出卖点）
- [ ] 开启自动广告 + 3-5 个手动精准词
- [ ] 日预算 $30-50

**目标**：积累 50+ 点击

### Week 2（第 8-14 天）

**优先级：数据积累 + 评论启动**

- [ ] 监控自动广告搜索词报告
- [ ] 识别高转化词，移入手动广告
- [ ] 暂停明显无效的词
- [ ] 主动联系买家，请求评论
- [ ] 考虑首单优惠加速冷启动

**目标**：积累 100+ 点击 + 5-10 订单 + 初始评论

### Week 3（第 15-21 天）

**优先级：评论积累 + Listing 微调**

- [ ] 评估 Listing 优化效果
- [ ] 如果 CTR 仍 < 0.3%，再次优化标题/主图
- [ ] 继续积累评论
- [ ] 使用 Review Accelerator（如果符合条件）

**目标**：积累 150+ 点击 + 10-15 订单 + 5+ 评论

### Week 4（第 22-28 天）

**优先级：阶段评估 + 过渡准备**

- [ ] 评估是否进入"成长期"（有足够数据）
- [ ] 如果是，开始分层出价（高转化词提价，低效词降价）
- [ ] 如果否，继续新品期策略

**目标**：准备进入下一阶段

## 新品期常见误区

| 误区 | 后果 | 正确做法 |
|------|------|---------|
| 花费不出去就加预算 | 浪费钱，系统仍不展示 | 先优化 Listing，提升相关性评分 |
| 一味追求低 ACOS | 降低出价 → 曝光减少 → 数据不足 → 无法优化 | 新品期应该追求曝光和点击，ACOS 其次 |
| 只投自动广告 | 关键词乱投，转化率低 | 自动 + 手动结合，手动做精准词 |
| 忽视 Listing 质量 | 即使有曝光也没人点击 | Listing 优化是新品期最重要的工作 |
| 过早关停低效关键词 | 数据不足，无法判断真实效果 | 新品期给每个关键词至少 2 周的测试期 |

---

## 新品期与成长期的过渡信号

**何时进入成长期**：
- ✅ 有足够点击数据（100+ clicks/周）
- ✅ 有初始转化数据（5+ 订单/周）
- ✅ 有评论积累（5+ 条评论）
- ✅ 评分稳定（≥ 4.5 星）

**过渡时的策略调整**：
- 从"追求曝光"转向"提升排名"
- 开启手动精准关键词（Exact Match）
- 配合手动广泛匹配（Broad Match）扩展流量
- 关注转化率（CR）和加购率（Add to Cart）
