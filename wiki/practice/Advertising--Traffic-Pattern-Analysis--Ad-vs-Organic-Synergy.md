# 广告与自然位协同策略

> 广告位与自然位的关系不是"非此即彼"，而是"协同放大"——避免无效重叠，最大化整体流量质量。

## 核心洞察

**基础知识参考**：
![[theory/Sponsored-Products-placements]]

### 广告位与自然位的本质差异

|| 维度 | 广告位 | 自然位 |
|------|------|--------|--------|
| **成本** | 按点击付费（无论转化与否） | 零额外成本 | |
| **稳定性** | 随时可调整 | 受算法影响，有波动 | |
| **触达用户** | 高意向搜索流量 | 算法推荐的自然流量 | |
| **转化质量** | 取决于展示时段和位置 | 通常较为稳定 | |

### 重叠的代价：无效点击

当广告位与自然位**高度重叠**时，广告会"吃掉"那些本来会自然点击的用户——这等于**花钱买本该免费的流量**，直接拉高 ACOS。

典型场景：
- 某关键词下自然排名第 3 位，广告也顶到首页顶部
- 搜索该词的用户看到同一个产品两次（广告位 + 自然位）
- 第一次点击是广告（付费），第二次自然位点击本可免费获得

## 协同策略框架

### 策略一：分层出价，物理隔离

对不同竞争强度的关键词使用差异化出价，让广告位和自然位分布在不同展示位置：

```
高排名关键词（自然前 5）
  → 广告出价降低，不争首页顶部
  → 广告展示在第 2-3 页或底部
  → 保护免费自然流量的同时，补充长尾流量

中低排名关键词（自然 6-20）
  → 广告正常出价，争取首页展示
  → 主动拉升关键词整体排名
  → 广告预算重点倾斜区域
```

### 策略二：分时出价，时间互补

广告和自然位的流量高峰时段往往不同——广告在高意向时段效果更好，自然位在日常浏览时段持续获量：

```
早间（00:00-06:00 PST）
  → 广告低预算：$10，bid $0.50
  → 自然位贡献稳定基础流量

白天（06:00-18:00 PST）
  → 广告正常预算：$45，bid $0.70
  → 竞品展示窗口期，广告争取主动权

晚间（18:00-24:00 PST）
  → 广告补充预算：+$15，bid $0.65
  → 高转化时段，广告与自然位协同放量
```

参考：[[practice/Advertising--Optimization-Methodology]] 中的"自然位与广告位协同策略"

### 策略三：关键词分级管理

根据自然排名将关键词分为三类，差异化处理：

|| 关键词类型 | 自然排名 | 广告策略 | 目标 |
|------|------|------|--------|------|
| **收割词** | 高排名词 | 前 5 名 | 降低 bid 或暂停 | 守住自然位，减少无效广告花费 |
| **进攻词** | 中排名词 | 6-20 名 | 正常出价 | 逐步提升自然排名 |
| **探索词** | 无排名词 | 20+ 或新词 | 高出价测试 | 验证关键词潜力 |

### 策略四：广告类型分层

**参考**：![[theory/Advertising--Search_Engine_Marketing__SEM_]]

不同广告类型对自然位的影响路径不同：

```
Brand 广告（品牌广告）
  → 保护品牌词，防止竞品抢量
  → 对自然排名无直接帮助

Sponsored Products（商品推广）
  → 直接提升商品转化率
  → 算法认定为"受欢迎产品" → 自然排名提升

SEM 搜索广告
  → 关键词级别精准打击
  → 与自然排名协同：广告词 → 转化 → 自然排名提升
```

## 实践案例

### 案例：基础款千斤顶广告测试（2026-04-17）

**背景：**
- 产品自然搜索位靠前（前 6 名）
- 广告此前顶到顶部位置
- ACOS 偏高，无效点击严重

**测试结果：**
- 降低 bid，让广告位置与自然位拉开距离
- 当日订单量翻倍（从 ~17 单 → 34 单）
- 结论：**自然位靠前时，适当放低广告出价，避免广告位与自然位重叠**

**操作：**
```
将基础款千斤顶 bid 从 $0.60 降至 $0.40
观察 3 天后：
  → 广告位置从首页顶部退至第 2 页
  → 整体转化率上升
  → ACOS 显著改善
```

### 案例：露营风扇分时策略（2026-04-15）

**问题：** 预算在晚 8 点耗尽，晚间高转化时段零展示

**分时策略：**
```
00:00-06:00：固定预算 $10，bid $0.50
06:00-18:00：固定预算 $45，bid $0.70
18:00-24:00：预算 +$15，bid $0.65
```

执行一周后观察：
- 晚间时段广告展示恢复正常
- 整体 ACOS 下降
- 日均订单量上升

## 自然排名下滑时的广告应对

当自然排名下滑时，广告策略需要主动补位：

### 诊断：确认下滑原因

|| 下滑类型 | 特征 | 优先应对 |
|------|------|------|---------|
| **断货导致** | 变体缺货，库存归零 | 优先补货，广告暂停 | |
| **竞品冲击** | 同款低价竞品出现 | 评估价格竞争力，差异化定位 | |
| **算法波动** | 无明显原因，自然波动 | 持续观察 3-5 天 | |
| **Listing 质量下降** | 评分下滑或差评增加 | 修复质量问题优先 | |

### 竞品出现时的应对（2026-04-21 案例）

**场景：** 双牙千斤顶 2T 变体断货，竞品（同款但标 2.5T）在广告位顶替

**分析：**
- 竞品优势：广告顶替断货位置，价格可能更低
- 我方优势：评论积累高，链接权重高，2T 规格更精准
- 劣势：2T 售价高于竞品 2.5T

**建议策略：**

```
第一步：恢复 2T 库存（最高优先）
  → 库存恢复 → 自然位恢复 → 广告重新占位

第二步：评估价格策略
  → 若竞品 2.5T 实际价格低于我 2T
  → 考虑调整 2T 定价至竞品价格区间

第三步：利用评论优势
  → 差评/reviews 差异化：强调"精准吨位，非虚标"
  → 在 Listing 中突出真实吨位 vs 虚标吨位的区别

第四步：广告策略
  → 竞品断货恢复期：广告高位夺回展示位置
  → 不与断货竞品直接竞争（避免高出 bid）
  → 竞品恢复后：评估价格差异化，或转移广告预算到其他词
```

## 常见误区

### ❌ 误区一：广告位置越高越好

广告顶到首页顶部 ≠ 最优结果。如果自然位已经靠前：
- 继续争顶部广告位 = 花广告费抢自己的自然流量
- **正确做法**：降低 bid，让广告退到自然位下方，两者覆盖不同用户

### ❌ 误区二：自然排名下滑必须靠广告补救

广告可以补位，但**不是万能药**：
- 如果下滑原因是断货 → 补货 > 加广告
- 如果下滑原因是差评 → 修复质量 > 加广告
- 如果下滑原因是竞品 → 差异化定价 > 单纯加 bid

### ❌ 误区三：广告和自然位可以完全独立

广告和自然位之间存在**正向反馈循环**（参考 [[practice/Algorithm-transition--Amazon-COSMO-vs-Walmart-second-price-auction]]）：
- 广告带来转化 → 算法判定产品受欢迎 → 自然排名提升
- 自然排名提升 → 降低广告依赖 → ACOS 改善
- **两者协同才能进入良性循环**

## 操作清单

### 新品期：让广告带动自然位

- [ ] 开启自动广告，日预算 $30-50
- [ ] 手动广告：3-5 个核心品类词，中位出价
- [ ] 目标：积累初始转化数据，推动算法识别

### 成长期：广告与自然位协同

- [ ] 每周跟踪 5-10 个核心词自然排名
- [ ] 自然进入前 5 的词：降低广告 bid 10-15%
- [ ] 自然 6-20 的词：维持或适当加 bid

### 稳定期：收割自然流量

- [ ] 对自然前 3 关键词：考虑暂停广告，避免重叠
- [ ] 维持广告：保护品牌词，测试新词
- [ ] 持续优化 Listing：提升自然转化率
