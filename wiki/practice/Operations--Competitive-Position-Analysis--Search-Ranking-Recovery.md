# 竞品分析：搜索排名下滑与自然位恢复策略

> 竞品出现、断货、价格劣势——三重压力叠加时的系统应对框架。

## 案例背景

**产品：** 双牙千斤顶（Car Jack）
**问题现象：**
- 3.22 搜索排名第 6 → 4.12 搜索排名第 14（20 天内下滑 8 位）
- 主要销售来源 2T 变体断货
- 竞品（同款套装，但标注 2.5T）在广告位顶替
- 竞品 2.5T 售价低于我方 2T 售价

**我方优势：**
- 评论积累高
- 链接权重高

**核心矛盾：**
- 吨位标注不匹配：对方标 2.5T vs 我方 1.5T/2T/3T（无 2.5T）
- 价格劣势：我 2T 售价 > 竞品 2.5T 售价

## 搜索排名下滑归因

**基础知识参考**：
![[theory/Growth_opportunities--Search_Insights]]

### 下滑时间线分析

```
时间节点：
  3.22 — 搜索排名 #6（正常）
  
  2T 变体断货（具体日期待确认）
    → 2T 是主要销售来源
    → 断货 → 转化率下降
    → 算法降权
    → 自然排名下滑

  4.12 — 搜索排名跌至 #14
    → 已错过断货黄金恢复期（通常 3-7 天）
    → 竞品趁机抢占位置
    → 广告位和自然位均被竞品占据
```

### 断货对排名的影响机制

断货不只影响"库存"，而是触发算法连锁反应：

```
断货发生
    ↓
可售库存 = 0（或极低）
    ↓
转化率 → 0%（无法成交）
    ↓
算法判定：产品"不受欢迎"或"不可靠"
    ↓
自然排名下降
    ↓
曝光量减少
    ↓
即使恢复库存，排名恢复需要时间（通常 7-14 天）
```

这解释了为什么断货后"补货回来"不等于"排名回来"——**算法需要新的转化数据来重建信任**。

## 竞品分析

### 竞品定位对比

|| 维度 | 我方（双牙千斤顶） | 竞品（同款） |
|------|------|------|---------|
| 吨位规格 | 1.5T / 2T / 3T | 2.5T | 我方规格更细分 |
| 价格 | 2T 售价较高 | 2.5T 售价较低 | 我方 2T 价格劣势 |
| 评论积累 | 高 | 待确认 | 我方优势 |
| 链接权重 | 高 | 待确认 | 我方优势 |
| 广告位置 | 下跌 | 顶替位置 | 竞品优势 |
| 自然排名 | #14（下跌中） | 上升中 | 竞品优势 |

### 关键洞察：规格不匹配

这不是"同款竞争"，而是**错位竞争**：
- 竞品的 2.5T 和我方的 2T 不是完全相同的产品
- 有 2.5T 需求的用户本来就不应该买 2T
- 有 2T 需求的用户如果看到 2.5T 可能会觉得"太大"

**但价格问题是真实的**：如果竞品 2.5T 比我 2T 还便宜，用户会认为：
1. 竞品性价比更高（吨位更大，价格更低）
2. 2T 这个价位可以买到更高规格的产品

## 应对策略

### 策略一：立即恢复 2T 库存（最高优先）

```
操作：
  1. 确认 2T 变体库存数量
  2. 立即发起补货（参考 [[practice/Operations--Restock-Decision-Methodology]]）
  3. 若库存暂时不足：
     → 考虑将 1.5T 或 3T 设为临时主推
     → 在 Listing 中引导用户选择替代规格

优先级：★★★★★（恢复自然位的前提条件）
```

### 策略二：价格竞争力评估

```
判断：竞品 2.5T 价格 < 我 2T 价格？

  → 是（价格劣势）：
    · 评估 2T 成本结构，看是否有降价空间
    · 或者：将 2T 定位为"精准吨位，非虚标"——与"虚标 2.5T"形成差异化
    · 在 Listing 中强调：我 2T = 真实 2T（非虚标）
    
  → 否（价格优势或持平）：
    · 当前价格合理，专注恢复排名
```

### 策略三：利用评论优势做差异化

评论是算法的信任信号，也是用户决策的关键因子：

```
我方评论优势利用：
  1. 在 Listing 突出真实用户评价（强调精准吨位）
  2. 回复差评，展示品牌态度
  3. 鼓励购买过的用户撰写详细评测

竞品评论劣势：
  → 若竞品虚标吨位（标 2.5T 实际承重更低）
  → 在 Q&A 或评论中合理引导对比
```

### 策略四：广告策略调整

**参考**：![[theory/Sponsored-Products--Campaign-bidding-strategies]]

```
断货恢复期间的广告策略：

阶段一：断货中
  → 暂停 2T 相关广告组（避免无效花费）
  → 将广告预算转移到其他有库存的变体

阶段二：恢复上架（前 3 天）
  → 适度加 bid 夺回展示位置
  → 目标：让算法重新识别产品活跃度
  → 建议 ACOS 可接受范围：目标 ACOS × 1.5（恢复期容忍度高）

阶段三：排名恢复后（7-14 天）
  → 逐步降低 bid，恢复正常 ACOS
  → 监控自然排名是否回到 #6 水平
```

### 策略五：搜索排名恢复时间表

```
Day 1-3：库存恢复 + 广告补位
  → 恢复 2T 库存
  → 适度加 bid 恢复展示
  → 预期：广告位置改善，自然位缓慢恢复

Day 4-7：数据积累期
  → 转化数据逐步积累
  → 算法开始重新评估产品
  → 预期：自然排名小幅回升

Day 8-14：稳定期
  → 转化率回到正常水平
  → 算法信任重建
  → 预期：自然排名接近或回到 #6

Day 15+：若排名未恢复
  → 执行 [[practice/Operations--Mature-Listing-Vicious-Cycle-Recovery]] 中的诊断流程
  → 确认是否有其他因素（差评/竞品/价格）
```

## 防止排名再次下滑

### 断货预警机制

```
库存警戒线：
  → 低于 7 天销量时触发补货提醒
  → 参考 [[practice/Operations--Restock-Decision-Methodology]] 中的补货决策检查清单

断货应急响应：
  1. 库存低于 3 天销量：立即启动补货
  2. 断货 3 天内：联系工厂加急
  3. 断货超 7 天：考虑调整广告策略，避免无效花费
```

### 竞品监控

```
每周检查：
  → 核心关键词下竞品数量和价格变化
  → 竞品是否有新的变体或促销
  → 竞品评论数量和质量变化

工具：
  → [[theory/Growth_opportunities--Search_Insights]]（搜索洞察）
  → [[theory/New-to-brand-metrics]]（归因分析，识别新竞争者）
```
