# 老链接恶性循环破解

> 订单少 → 掉排名 → 加广告 → 有订单 → 减广告 → 订单少 → 掉排名……这不是广告问题，是流量入口结构问题。

## 循环本质

### 典型症状

```
阶段1：无订单
  → 自然排名下滑
  → 曝光量减少

阶段2：加广告补救
  → 花费增加
  → ACOS 上升
  → 利润被压缩

阶段3：有订单
  → 自然排名部分恢复
  → 曝光改善

阶段4：减广告控成本
  → 广告位撤退
  → 自然位未能完全承接
  → 回到阶段1
```

**核心问题**：广告和自然位形成"跷跷板"——广告强则自然弱，自然弱则广告必须更强，形成恶性循环。

### 根因诊断：流量入口分析

不是"广告不够"，而是"流量入口单一"——仅依赖搜索排名这一条路。

|| 流量入口 | 当前状态 | 修复优先级 |
|------|------|------|---------|
| 自然搜索排名 | 依赖关键词排名，波动大 | 高 |
| 广告流量 | CPC 高，ACOS 难控 | 中 |
| Browse 页面流量 | 被动等待，不稳定 | 低 |
| 促销/Deal 流量 | 未充分利用 | 中 |
| 多渠道流量 | WFS/站外等 | 低 |

## 破解路径

### 第一步：确认流量入口比例

**基础知识参考**：
![[theory/All-Keywords]]

从广告报告中拆解流量结构：

```
主要诊断指标（从 All-Keywords 报告下载）：
  1. 广告订单占总订单的比例
  2. 自然订单占总订单的比例
  3. Browse 页订单 vs 搜索页订单
  4. 各关键词的广告位 vs 自然位的贡献

判断标准：
  - 广告订单 > 60%：说明自然位严重不足（高度依赖广告）
  - 自然订单 < 20%：说明 Listing 质量或关键词覆盖面有问题
  - Browse 页流量占比 > 40%：说明自然搜索权重偏低
```

### 第二步：识别拖累因素

|| 拖累因素 | 诊断方法 | 应对 |
|------|------|---------|------|
| **断货** | 库存报告显示缺货 | 优先补货 |
| **差评/评分低** | 查看产品评分 | 修复产品问题 |
| **竞品价格低** | 前台检查竞品价格 | 评估定价策略 |
| **Listing 质量差** | 对比类目头部 Listing | 优化内容 |
| **关键词流失** | 搜索词报告减少 | 扩展关键词 |

### 第三步：修复核心问题

#### 3.1 断货优先

断货是排名下滑最直接的原因：
- 算法认为产品"不可靠"（无法完成订单）
- 断货超过 3 天，排名开始显著下滑
- **修复**：优先恢复库存，哪怕少量补货也比完全缺货好

参考：[[practice/Operations--Restock-Decision-Methodology]]

#### 3.2 优化自然位（治本）

自然位是破解循环的关键——**自然位稳定，广告才能降低依赖**：

```
提升自然位的操作：
  1. 确保库存充足（WFS 稳定性影响算法评分）
  2. 优化转化率（高转化 → 算法认定为优质产品）
  3. 扩展关键词覆盖（让更多词有自然排名）
  4. 提升 Listing 质量分（图片/标题/描述）
```

#### 3.3 控制广告 ACOS

广告不是不能做，但要控制比例：

```
ACOS 警戒线：
  - 高利润产品（> 30% 毛利）：ACOS 40% 以内可接受
  - 中等利润（15-30%）：ACOS 25% 以内
  - 低利润（< 15%）：ACOS 15% 以内

超过警戒线的应对：
  1. 检查是否有无效点击（[[practice/Advertising--Bid-reduction-Paradox]]）
  2. 暂停高花费低转化关键词
  3. 评估广告结构是否有优化空间
```

参考：[[practice/Advertising--Bid-reduction-Paradox]]

### 第四步：建立稳定的流量结构

**参考**：![[theory/Walmart_Fulfillment_Services__WFS_]]

单一搜索排名是不稳定的，需要多入口支撑：

```
健康流量结构（目标）：
  自然搜索订单：40-50%
  广告订单：30-40%
  Browse/其他：10-20%

实现路径：
  1. 自然位：稳住 5-10 个核心词排名
  2. 广告：保护高转化词，控制低效词花费
  3. 促销：参与 Flash Deal / Campaign 增加额外流量
  4. WFS：使用 WFS 提升算法评分，间接提升自然排名
```

参考：[[theory/Growth_opportunities--Promotions]]（促销定价）、[[theory/Walmart_Fulfillment_Services__WFS_]]（WFS 对自然排名的影响）

### 第五步：广告与自然位协同

参考：[[practice/Advertising--Traffic-Pattern-Analysis--Ad-vs-Organic-Synergy]]

核心原则：
- 自然排名好的词 → 降低广告 bid
- 自然排名差的词 → 适度加广告 bid
- 两者协同而非互相竞争

## 关键指标监控

### 周度监控表

|| 指标 | 健康值 | 警戒值 |
|------|------|--------|--------|
| 自然订单占比 | > 30% | < 15% |
| 广告 ACOS | < 目标 ACOS | > 目标 ACOS × 1.2 |
| 核心词自然排名 | 前 20 | 跌出前 30 |
| 库存充足率 | 100% | < 80% |
| 广告占总订单比 | < 50% | > 65% |

### 循环解除标志

当出现以下信号，说明循环已被打破：

- [ ] 自然订单占比稳定在 30% 以上
- [ ] 核心关键词自然排名进入前 20 并保持稳定
- [ ] 广告 ACOS 控制在目标范围内
- [ ] 减少广告花费后订单量不再大幅下滑

## 案例：基础款千斤顶的循环

**问题背景（2026-04-16）：**
- 订单量始终无法上升
- 广告调整带不来单量提升
- 前台位置上升有一定效果，但容易掉位置
- 老链接陷入"死循环"

**解决路径：**
1. 分析流量入口：发现广告流量占主导，自然位贡献不足
2. 调整广告策略：降低 bid 让广告位与自然位拉开距离
3. 修复 2T 变体断货问题（补货）
4. 结果：广告位退到第二页，自然位贡献增加，整体订单量翻倍

**结论**：不是"广告不够"，而是"广告和自然位在互相内耗"——让广告做广告的事，让自然位做自然位的事。
